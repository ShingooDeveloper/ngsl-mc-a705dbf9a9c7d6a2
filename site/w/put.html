put

動詞・名詞/pʊt/CEFR A1頻度#138
  1. 【動詞】（物を）置く、配置する（…place an object in a location…）
  2. 【動詞】～に入れる、～に詰め込む（…insert into a container…）
  3. 【動詞】～の状態にする、～に置く（…cause to be in a state or condition…）
  4. 【動詞】～に向かわせる、進める（…direct or move towards…）
  5. 【名詞】（投擲競技の）ショット、押す動作

⛏️ マインクラフト例文

⛏Put the dirt blocks here.

土のブロックをここに置いてください。

/pʊt ðə ˈdɝt blɑks ˈhɪr/
  • Put の /t/ が未開放（後ろが子音）
  • the は弱形 /ðə/
  • dirt の /t/ が未開放（後ろが子音）
