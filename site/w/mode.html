mode

名詞/moʊd/CEFR B1頻度#2201
  1. 【名詞】やり方、方法（a way or manner of doing something）
  2. 【名詞】（ゲームやデバイスの）モード、状態・設定（a setting or state in a game or device）
  3. 【名詞】（統計）最頻値（the value that appears most often in a set of data）
  4. 【名詞】（音楽）モード、調式（a musical scale or key）

⛏️ マインクラフト例文

⛏Switch to survival mode and start your adventure.

サバイバルモードに切り替えて、冒険を始めてください。

/ˈswɪtʃ tə sɚˈvaɪvəl ˈmoʊdən ˈstɑɹt jɚ ədˈvɛntʃɚ/
  • to は機能語で弱形 /tə/。
  • mode の語末子音 /d/ が and の弱形 /ən/ に連結 [ˈmoʊdən] モゥダン
スウィッチ タ サーヴァイヴァル モゥダン スタート ユア アドヴェンチャー

⛏Set it to peaceful mode for creative building.

平和モードに設定して、創作的な建築をします。

/sɛɾɪt tə ˈpiːsfoʊl ˈmoʊd fɚ kriˈeɪtɪv ˈbɪldɪŋ/
  • set の語末子音 /t/ が it の初頭母音 /ɪ/ に連結。米語ではフラップ化 [sɛɾɪt̚]。
  • to は弱形 /tə/。
  • mode /d/ と for /fɚ/ は子音+子音で特に連結なし。for は機能語で弱形。
