fear

動詞・名詞/fɪr/CEFR B1頻度#792
  1. 【動詞】恐れる、怖がる（…が危ないと思う）
  2. 【名詞】恐怖、恐怖心（危機・危険への…）
  3. 【動詞】心配する、～を懸念する（…ではないかと不安に思う）
  4. 【名詞】懸念、気がかり（…ではないかという心配）

⛏️ マインクラフト例文

⛏I fear the Creeper might blow up near my house.

Creeperが家の近くで爆発するかもしれないと心配だ

/aɪ fɪr ðə ˈkriːpər maɪt bloʊ ʌp nɪr maɪ haʊs/
  • fear の語末 /r/ が the の /ə/ に連結 [fɪrðə] フィアザ
  • near の語末 /r/ が my の /maɪ/ に連結 [nɪrmaɪ] ニアマイ
アイ フィアザ クリーパー マイッ ブロウ アップ ニアマイ ハウス

⛏Don't fear—I've set up defenses everywhere.

怖がるな。防御をどこにでも設置した

/doʊnt fɪr aɪv sɛt ʌp dɪˈfɛnsɪz ˈɛvriwɛr/
  • don't の語末 /t/ は未開放で fear に続く
  • fear の語末 /r/ が I've の /aɪ/ に連結 [fɪraɪv]
  • set の語末 /t/ が up の /ʌ/ に連結し、フラップ化 [sɛɾʌp]
ドン フィアアイヴ セラップ ディフェンシズ エブリウェア
