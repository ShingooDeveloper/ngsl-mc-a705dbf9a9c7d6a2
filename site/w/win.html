win

動詞・名詞/wɪn/CEFR A1頻度#376
  1. 【動詞】勝つ、競争や試合に勝利を収める（succeed in a competition）
  2. 【動詞】獲得する、報酬として手に入れる（obtain as a prize or reward）
  3. 【名詞】勝利、勝ち（a successful outcome in a game or competition）

⛏️ マインクラフト例文

⛏I won amazing loot in the dungeon!

ダンジョンですごい戦利品を獲得した！

/aɪ ˈwʌn əˈmeɪzɪŋ lut ɪn ðə ˈdʌndʒən/
  • won amazing → won の語末 n が amazing の ə に連結
  • the → 弱形 /ðə/ でシュワに弱化
アイ ワナメイジング ルート イン ザ ダンジョン

⛏To win, you need to gather resources first.

勝つには、まず資材を集める必要があります。

/tə ˈwɪn ju ˈnɪd̚ə ˈɡæðɚ ɹɪˈsoʊɹsɪz ˈfɚst/
  • to → 機能語は弱形 /tə/ に弱化
  • need to → need の語末 d は未開放 [d̚]、to に連結
タ ウィン ユー ニッァ ギャザー リソーシズ ファースト

⛏Did you win the race yesterday?

昨日のレースで勝ちましたか？

/dɪd̚ ju ˈwɪn ðə ˈɹeɪs ˈjɛstɚdeɪ/
  • Did → /dɪd/ の語末 d は未開放 [d̚]、「ッ」と発音
  • the race → the は弱形 /ðə/ でシュワに弱化
