although

接続詞/ɔːlˈðoʊ/CEFR B1頻度#277
  1. 【接続詞】～にもかかわらず、～だけれども。文や句の前に置き、後続する主張と対比的または矛盾する情報を導入する従属接続詞。（even though、though と同義）

⛏️ マインクラフト例文

⛏Although I want to build a house, I need to find more wood first.

家を建てたいけれど、まず木をもっと見つける必要がある。

[ɔːlˈðoʊ ˈaɪ ˈwɑntə ˈbɪldə ˈhaʊs | ˈaɪ ˈniːdə ˈfaɪnd ˈmɔrwʊd ˈfɝst]
  • want to → want の /t/ と to の /t/ が同化して [wɑntə]→「ウォンタ」
  • build a → build の /d/ が a の弱形 /ə/ に連結して [bɪldə]→「ビルダ」
  • need to → need の /d/ と to の /t/ が同化して [niːdə]→「ニーダ」
  • more wood → more の /r/ が wood に連結
オールドウ アイ ウォンタ ビルダ ハウス アイ ニーダ ファイン モアウッド ファースト

⛏Although I've lost health, I refuse to give up.

ライフが減ってしまったけれど、諦めるつもりはない。

[ɔːlˈðoʊ ˈaɪv ˈlɔst ˈhɛlθ | ˈaɪ rɪˈfjuzə ˈɡɪv ˈʌp]
  • refuse to → refuse の /z/ が to の弱形 /ə/ に連結して [rɪˈfjuzə]→「リフューザ」
オールドウ アイヴ ロスト ヘルス アイ リフューザ ギヴ アップ

⛏Although I've built a tower, I want to make it taller.

塔は建てたけれど、もっと高くしたい。

[ɔːlˈðoʊ ˈaɪv ˈbɪltə ˈtaʊər | ˈaɪ ˈwɑntə ˈmeɪk ɪt ˈtɔlər]
  • built a → built の /t/ が a の弱形 /ə/ に連結して [bɪltə]→「ビルタ」
  • tower I → tower の /r/ が I の /aɪ/ に連結して [taʊəraɪ]→「タウアーライ」
  • want to → want の /t/ と to の /t/ が同化して [wɑntə]→「ウォンタ」
オールドウ アイヴ ビルタ タウアーライ アイ ウォンタ メイク イット トーラー
