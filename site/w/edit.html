edit

動詞・名詞/ˈɛdɪt/CEFR B1頻度#2308
  1. 【動詞】映像・文章・データなどを修正・改編する（edit a video, edit a document）
  2. 【動詞】編集により削除する（edit out mistakes）
  3. 【名詞】編集、修正の行為（perform an edit）

⛏️ マインクラフト例文

⛏I need to edit my build plan before starting.

建設を始める前に、建築計画を編集する必要があります。

[aɪ ˈniːdə ˈɛdɪt maɪ bɪld plæn bɪˈfɔːr ˈstɑːrtɪŋ]
  • need to - 語末の /d/ が to の母音 /ə/ に連結 [ˈniːdə]
  • edit my - 語末の /t/ が次の /m/ 前で未開放 [t̚]
アイ ニーダ エディット マイ ビルド プラン ビフォー スターティング

⛏Can you edit the redstone circuit to make it faster?

レッドストーン回路を編集して、もっと速くできますか？

[kən juː ˈɛdɪt ðə ˈrɛdstoʊn ˈsɝːkɪt tə ˈmeɪkɪt̚ ˈfæstər]
  • Can you - Can の弱形 [kən]
  • make it - make の語末 /k/ が it の母音 /ɪ/ に連結 [ˈmeɪkɪ]、it の /t/ が未開放 [t̚]
カン ユー エディット ザ レッドストーン サーキット タ メイキット ファスター

⛏I'm going to edit out the mistakes in my world design.

ワールドデザイン内の間違いを編集で削除しようと思っています。

[aɪm ˈɡoʊɪŋ tə ˈɛdɪtaʊt ðə məˈsteɪks ɪn maɪ wɝːld dɪˈzaɪn]
  • going to - to の弱形 [tə]
  • edit out - edit の語末 /t/ が out の母音 /aʊ/ に連結 [ˈɛdɪtaʊt]
アイム ゴーイング タ エディタウット ザ ミステイクス イン マイ ワールド ディザイン

💬 よく使う熟語（マイクラ例文）

edit out編集で削除する、編集により除去する

⛏Let me edit out the water leak from my farm.

農場の水漏れを編集で削除しましょう。

[lɛɾəm ˈɛdɪtaʊt ðə ˈwɔɾər lik frəm maɪ fɑːrm]
  • let me - let の /t/ が母音に挟まれてフラップ化 [ɾ] になり me に連結 [lɛɾəm]
  • edit out - edit の語末 /t/ が out の母音に連結 [ˈɛdɪtaʊt]
