hope

動詞・名詞/hoʊp/CEFR A2頻度#249
  1. 【動詞】～することを望む；期待する（良い結果がもたらされることを待つ）
  2. 【名詞】希望；期待（目指すべき良い未来）
  3. 【動詞】（不確実な状況で）期待する；望みをかける
  4. 【名詞】望みのよすが；希望の光（よりどころとなる望み）

⛏️ マインクラフト例文

⛏I hope I can find diamonds before nightfall.

夜までにダイヤモンドが見つかるといいな。

/aɪ ˈhoʊpaɪ kən ˈfaɪnd ˈdaɪəmɑndz bɪˈfɔr ˈnaɪtfɔl/
  • hope の語末 /p/ が I の /aɪ/ に連結 [ˈhoʊpaɪ]
  • can の弱形 /kən/
アイ ホウパイ カン ファインド ダイヤモンズ ビフォー ナイトフォール

⛏I hope we get diamonds out of this mine.

このマインからダイヤが採れるといいな。

/aɪ ˈhoʊp wi ɡɛt ˈdaɪəmɑndzaʊɾəv ðɪs ˈmaɪn/
  • diamonds の /z/ が out の /aʊ/ に連結 [daɪəmɑndzaʊ]
  • out の /t/ が母音に挟まれてフラップ化し連結 [aʊɾəv]。of は弱形 /əv/
アイ ホウプ ウィ ゲット ダイヤモンズアウラヴ ディス マイン

⛏We hope to find a better base location.

もっと良い拠点の場所が見つかるといいな。

/wi ˈhoʊptə ˈfaɪndə ˈbɛtər ˈbeɪs loʊˈkeɪʃən/
  • find の /d/ が a の /ə/ に連結 [ˈfaɪndə]
ウィ ホウプタ ファインダ ベター ベース ロウケイシャン

💬 よく使う熟語（マイクラ例文）

/ðɪs ˈkæsəl dɪˈzaɪn wɪl ˈlʊkəˈmeɪzɪŋ ˈraɪɾaɪ ˈhoʊp soʊ/
  • look の /k/ が amazing の /ə/ に連結 [ˈlʊkə]
  • right の /t/ が I の /aɪ/ に挟まれてフラップ化し連結 [ˈraɪɾaɪ]

⛏We have low health, but hope for the best in this fight!

/wi hæv loʊ ˈhelθ bət ˈhoʊp fər ðə ˈbɛsɾɪn ðɪs ˈfaɪt/

  • the の弱形 /ðə/
