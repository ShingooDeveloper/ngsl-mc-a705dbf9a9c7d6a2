end

動詞・名詞/end/CEFR A1頻度#151
  1. 【動詞】終わる、終わらせる（…end a game）
  2. 【動詞】～で終わる、結果として～になる（…end up in failure）
  3. 【名詞】終わり、終了、完結（…reach the end）
  4. 【名詞】末端、端、先端（…the end of the pickaxe）
  5. 【名詞】目的、意図（…to this end）

⛏️ マインクラフト例文

⛏The night will end when the sun rises.

夜は太陽が上ることで終わるだろう。

/ðə ˈnaɪt wɪlˈɛnd wɛn ðə ˈsʌn ˈraɪzɪz/
  • will end の /l/ が end の /ɛ/ に連結 [wɪlˈɛnd] ウィレン
  • end when の /d/ は未開放、次の /w/ に先立つ
ザ ナイト ウィレン ウェン ザ サン ライジズ

⛏I ended the village raid before dark.

暗くなる前に村へのレイドを終わらせた。

/aɪ ˈɛndəd ðə ˈvɪlɪdʒ ˈreɪd̚ bɪˈfɔr ˈdɑrk/
  • ended the → ended の末尾 /d/ が the の /ð/ に先立つ
  • raid before → raid の /d/ は未開放、before の /b/ に先立つ
アイ エンダッド ザ ビリッジ レイド ビフォー ダーク

⛏The creeper explosion was the end of our base.

クリーパーの爆発が私たちのベースの終わりだった。

/ðə ˈkripər ɪkˈsploʊʒən wəz ðə ˈɛndəv ˈaʊr ˈbeɪs/
  • creeper explosion → creeper の末尾 /r/ が explosion の /ɪ/ に連結
  • end of → end の /d/ が of の /ə/ に連結 [ˈɛndəv] エンダヴ
ザ クリーパー イクスプロウジョン ワズ ザ エンダヴ アウアー ベイス

💬 よく使う熟語（マイクラ例文）

in the end結局、最終的に

⛏In the end, we survived the night.

結局のところ、私たちは夜を乗り切った。

/ɪn ðə ˈɛnd, wi sərˈvaɪvd ðə ˈnaɪt/
  • in the → in の /n/ が the の /ð/ に先立つ
  • the end → the の /ə/ が end の /ɛ/ に連結
  • survived の /d/ が the の /ð/ に先立つ
イン ザ エン ウィ サーヴァイド ザ ナイト
end up～になってしまう、結果として～になる

⛏We ended up losing all our diamonds.

私たちはすべてのダイヤを失ってしまった。

/wi ˈɛndəp ˈluːzɪŋ ɔl aʊr ˈdaɪməndz/
  • ended up → ended の /d/ が up の /ʌ/ に連結 [ˈɛndəp] エンダップ
  • up losing → up の /p/ は未開放、losing の /l/ に先立つ
ウィ エンダップ ルージング オール アウアー ダイマンズ
the end終わり（ゲーム終了画面に表示される言葉）

⛏The end appeared on the screen.

ゲーム終了画面に『The End』が表示された。

/ðə ˈɛndə pɪrd ɑn ðə ˈskriːn/
  • the end → the の /ə/ が end の /ɛ/ に連結
  • end appeared → end の /d/ が appeared の /ə/ に連結 [ˈɛndə] エンダ
ザ エンダ ピアード オン ザ スクリーン
