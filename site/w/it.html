it

代名詞/ɪt/CEFR A1頻度#8
  1. 【代名詞】前述の事物や対象を指す（…I found a diamond. It is very valuable.）
  2. 【代名詞】時間・天候・距離など非人称の主語として機能（…It's nighttime. It takes 30 minutes.）
  3. 【代名詞】形式主語として構文を整える（…It is dangerous to go out when it's dark.）
  4. 【代名詞】形式目的語として補完（…We made it to the village before sunset.）

⛏️ マインクラフト例文

⛏I found obsidian ore. It is very hard to mine.

黒曜石鉱石を見つけた。採掘するのが非常に難しい。

/aɪ ˈfaʊnd ɑbˈsɪdiən ˈɔr. ɪt ɪz ˈvɛri ˈhɑrd tu ˈmaɪn./
  • ore It → /r/ が /ɪ/ に連結 [ˈɔr ɪt]
  • hard to → /d/ が /t/ に連接。hard は /d/ 未開放 [ˈhɑrd̚ tu]
アイ ファウンド アブシディアン オア。イット イズ ベリー ハーッ トゥー マイン。

⛏The Wither is attacking. Can you defeat it?

ウィザーが攻撃している。倒せる？

/ðə ˈwɪðər ɪz əˈtækɪŋ. kæn juː dɪˈfiːt ɪt/
  • the Wither → the は弱形 /ðə/
  • is attacking → /z/ が /ə/ に連結。is 弱形 /ɪz/ [ɪz əˈtækɪŋ]
  • defeat it → /t/ が /ɪ/ に連結 [dɪˈfiːt ɪt]
ザ ウィザー イズアタッキング。キャン ユー ディフィーティット？

⛏We need to craft a bed. Do you know how to make it?

ベッドをクラフトする必要があります。作り方を知っていますか？

/wiː ˈniːd tu ˈkræft ə ˈbɛd. du juː ˈnoʊ haʊ tu ˈmeɪk ɪt/
  • craft a → /t/ が /ə/ に連結。craft 末尾 /t/ [ˈkræft ə]
  • to は弱形 /tə/
  • make it → /k/ が /ɪ/ に連結 [ˈmeɪk ɪt]
ウィー ニード トゥー クラフタ ベッド。ドゥー ユー ノウ ハウ トゥー メイキット？

💬 よく使う熟語（マイクラ例文）

make it成功させる、達成する、間に合わせる

⛏If we run fast, we can make it to the shelter before mobs spawn.

急いで走れば、モブが出現する前にシェルターに着ける。

/ɪf wiː ˈrʌn ˈfæst, wiː kæn ˈmeɪk ɪt tu ðə ˈʃɛltər bɪˈfɔr ˈmoʊbz ˈspɔːn./
  • make it → /k/ が /ɪ/ に連結 [ˈmeɪk ɪt]
  • to the → to は弱形 /tə/, the も弱形 /ðə/
イフ ウィー ラン ファスト、ウィー キャン メイキット タザ シェルター ビフォア モブズ スポーン。
take itそれを受け入れる、（攻撃などを）耐える

⛏The zombie hit me, but I can take it because I have armor.

ゾンビに殴られたけど、鎧があるから耐えられる。

/ðə ˈzɑmbi ˈhɪt mi, bət aɪ kæn ˈteɪk ɪt bɪˈkɔz aɪ ˈhæv ˈɑrmər./
  • take it → /k/ が /ɪ/ に連結 [ˈteɪk ɪt]
  • have armor → /v/ が /ɑ/ に連結 [ˈhæv ˈɑrmər]
ザ ゾンビ ヒット ミー、バット アイ キャン テイキット ビコーズ アイ ハヴァーマー。
call it (a day)今日のところはやめにする

⛏It's getting dark. Let's call it a day and go back to base.

暗くなってきた。今日のところはやめにして基地に帰ろう。

/ɪts ˈɡɛtɪŋ ˈdɑrk. ˈlɛts ˈkɔːl ɪt ə ˈdeɪ ænd ˈɡoʊ ˈbæk tu ˈbeɪs./
  • It's getting → it's 縮約 /ɪts/
  • call it → /l/ が /ɪ/ に連結 [ˈkɔːl ɪt]
  • it a → /t/ が /ə/ に連結。a は弱形 [ɪt ə]
  • go back to → to は弱形 /tə/
イッツ ゲッティング ダーク。レッツ コーリット イッタ デイ アンド ゴー バック タ ベース。
