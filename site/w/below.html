below

前置詞・副詞・形容詞/bɪˈloʊ/CEFR A1頻度#760
  1. 【前置詞】下に、〜より下にある（空間的位置）（...under, beneath...）
  2. 【前置詞】〜より低い（数量・程度）（...less than...）
  3. 【副詞】下へ、下方に（...downward...）
  4. 【形容詞】下の、以下の（...lower...）

⛏️ マインクラフト例文

⛏I found diamonds five blocks below the surface.

表面より5ブロック下でダイヤモンドを見つけた。

/aɪ faʊnd ˈdaɪəmɑndz faɪv blɑks bɪˈloʊ ðə ˈsɝfəs/
  • found の /d/ が未開放になる。
  • the は弱形 /ðə/ に弱化
アイ ファウンダイアモンズ ファイヴ ブロックス ビロー ザ サーフェス

⛏The cave extends far below ground level.

その洞窟は地面レベルより遙か下に延びている。

/ðə keɪv ɪkˈstɛndz fɑr bɪˈloʊ ˈɡraʊnd ˈlɛvəl/
  • The は弱形 /ðə/ に弱化
  • ground の /d/ が未開放になる。
ザ ケイヴ イク・ステンズ ファー ビロー グラウンレベル

⛏We can build a base below the water.

水の下に拠点を建築できる。

/wi kən bɪld ə beɪs bɪˈloʊ ðə ˈwɔtɚ/
  • can は弱形 /kən/ に弱化
  • build の語末 /d/ が a の /ə/ に連結。[bɪl.də]「ビルダ」

💬 よく使う熟語（マイクラ例文）

below ground地下に（採掘や地下構造物を指す）

⛏We are mining below ground to find iron ore.

鉄鉱石を見つけるために地下で採掘している。

/wi ɑr ˈmaɪnɪŋ bɪˈloʊ ˈɡraʊnd tə faɪnd ˈaɪɚn ɔr/

  • to は弱形 /tə/ に弱化
  • find の語末 /d/ が iron の /aɪ/ に連結。[ˈfaɪn.daɪ.ɚn]「ファインダイアン」
