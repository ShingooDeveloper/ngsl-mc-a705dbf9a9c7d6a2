red

形容詞・名詞/rɛd/CEFR A1頻度#799
  1. 【形容詞】赤い、赤色の（The red wool block looks great.）
  2. 【名詞】赤、赤色（Red is my favorite color for decorating.）
  3. 【名詞】赤い染料や顔料（I used red dye to paint the armor.）

⛏️ マインクラフト例文

⛏I built a red armor stand in my base.

私の基地に赤い防具立てを作りました。

/aɪ bɪlt ə rɛd ˈɑrmər stænd ɪn maɪ beɪs/
  • built a → t がフラップ化して [ɾ] になる、[bɪlɾə]→「ビラタ」
  • red armor → d が [ɑ] に連結、[rɛdɑrmər]→「レダーマー」
アイビラタレダーマースタンドインマイベース

⛏I need a red dye to paint my armor.

防具を塗るために赤い染料が必要です。

/aɪ niːd ə rɛd daɪ tə peɪnt maɪ ˈɑrmər/
  • need a → d が [ə] に連結、[niːdə]→「ニーダ」
アイニーダレッドダイタペイントマイアーマー

⛏My red armor is awesome.

私の赤い防具は素晴らしいです。

/maɪ rɛd ˈɑrmər ɪz ˈɔsəm/
  • red armor → d が [ɑ] に連結、[rɛdɑrmər]→「レダーマー」
  • is awesome → z が [ɔ] に連結、[ɪzɔsəm]→「イゾーサム」
マイレダーマーイゾーサム
