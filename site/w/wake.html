wake

動詞・名詞/weɪk/CEFR A1頻度#1505
  1. 【動詞】眠りから目覚める、起きる
  2. 【動詞】〜を目覚めさせる、起こす
  3. 【名詞】通夜、葬礼；（～の後に、という意の前置詞句）in the wake of で使用

⛏️ マインクラフト例文

⛏You need to wake up before you can mine.

採掘する前に起きる必要があります。

/juː niːd tə weɪk ʌp bɪˈfɔːr juː kən maɪn/
  • need to → need の /d/ が to の /ə/ に連結 [niːdə]「ニーダ」
  • wake up → wake の /k/ が up の /ʌ/ に連結 [weɪkʌp]「ウェイカップ」
  • to は機能語で弱形 /tə/
ユー ニーダ ウェイカップ ビフォア ユー カン マイン

⛏Don't wake the zombie while mining.

採掘中、ゾンビを起こさないでください。

/doʊnt weɪk ðə ˈzoʊmbi waɪl ˈmaɪnɪŋ/
  • the は機能語で弱形 /ðə/
  • wake の /k/ は子音 /ð/ に続くため連結なし
ドウント ウェイク ザ ゾンビ ワイル マイニング

⛏I woke up and started building my house.

起きて家を建てました。

/aɪ woʊk ʌp ənd ˈstɑɹtəd ˈbɪldɪŋ maɪ haʊs/
  • woke up → woke の /k/ が up の /ʌ/ に連結 [woʊkʌp]「ウォークアップ」
  • started → /ɹ/ と /t/ の間の /t/ がフラップT化して [ɾ]「ラ」発音
  • and は弱形 /ənd/ で「アン」
アイ ウォークアップ アン スターラッド ビルディング マイ ハウス

💬 よく使う熟語（マイクラ例文）

wake up to（問題・状況に）気づく、（現実を）直視する

⛏I finally woke up to the danger of creepers.

やっとクリーパーの危険性に気づきました。

/aɪ ˈfaɪnəli woʊk ʌp tə ðə ˈdeɪndʒər əv ˈkɹipərz/
  • woke up → woke の /k/ が up の /ʌ/ に連結 [woʊkʌp]「ウォークアップ」
  • up to → /p/ と /t/ が連続。up の /p/ は未開放 [ʌp̚]
  • danger of → danger の語末 /r/ が母音化（米英では /ɚ/）し、of の /əv/ に連結して [deɪndʒɚəv]「デインジャーアヴ」
  • the は弱形 /ðə/
アイ ファイナリー ウォークアップタ ザ デインジャーアヴ クリーパーズ
