extension

名詞/ɪkˈstɛnʃən/CEFR B1頻度#2183
  1. 【名詞】拡張、延長（機能やコンテンツを追加するプロセス）
  2. 【名詞】追加機能、拡張プラグイン（…mod extension として使用）
  3. 【名詞】延長部分、増築部分（…building extension の意味）
  4. 【名詞】延長期間、継続期間（…extension of the deadline）
  5. 【名詞】延長コード、接続ケーブル（…cable extension として使用）

⛏️ マインクラフト例文

⛏This extension adds new blocks and mobs to the game.

このエクステンション（拡張機能）がゲームに新しいブロックとモブを追加します。

[ðɪs ɪkˈstɛnʃən ˈædz njuː ˈblɑks ənd ˈmoʊbz tə ðə ˈɡeɪm]
  • to は機能語で弱形 /tə/ に弱化
  • the も弱形 /ðə/ に弱化
ディス イクステンション アッズ ニュー ブロックス アンド モーブズ タ ザ ゲイム

  • need の /d/ が an の /ə/ に連結し [ˈniːdən] ニーダン
