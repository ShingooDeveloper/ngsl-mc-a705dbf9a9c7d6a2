color

名詞・動詞/ˈkʌlər/CEFR A1頻度#377
  1. 【名詞】色、色合い（…color）
  2. 【名詞】（顔料・染料などの）着色材（…color）
  3. 【動詞】色を付ける、着色する（…color）

⛏️ マインクラフト例文

⛏What color is this leather armor?

この革の鎧は何色ですか？

/wɑt ˈkʌlər ɪz ðɪs ˈleðər ˈɑrmər/
  • 「color is」で color の語末 /r/ が is の /ɪ/ に連結 [ˈkʌlərɪz] コラリズ
  • 「leather armor」で leather の語末 /r/ が armor の /ɑ/ に連結 [ˈleðərˈɑrmər] レーザー アーマー
ワット コラリズ ディス レーザー アーマー

⛏I need to color the concrete to make it match.

マッチさせるためにコンクリートを着色する必要があります。

/aɪ ˈniːd tə ˈkʌlər ðə ˈkɑŋkrit tə meɪk ɪt ˈmætʃ/
  • 「need to」で need の語末 /d/ が to の弱形 /ə/ に連結 [ˈniːdə] ニーダ
  • 「make it」で make の語末 /k/ が it の /ɪ/ に連結 [meɪkɪt] メイキット
  • 「to」と「the」は機能語で弱形 /tə/, /ðə/（to は2回出現）
