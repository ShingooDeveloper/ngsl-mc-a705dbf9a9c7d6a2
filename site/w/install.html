install

動詞・名詞/ɪnˈstɔl/CEFR B1頻度#1921
  1. 【動詞】（ソフトウェア・機器などを）設置する、インストールする（…をコンピュータやシステムに取り付けたり配置したりする）
  2. 【動詞】（人を職位に）就任させる、任命する（…が公式な職位につくようにさせる）
  3. 【名詞】設置物、インストール版（…取り付けられたもの）

⛏️ マインクラフト例文

⛏You should install OptiFine to boost your FPS in Minecraft.

マインクラフトでフレームレートを上げるためにOptiFineをインストールすべきだ。
