operation

名詞/ˌɑːpəˈreɪʃən/CEFR B1頻度#847
  1. 【名詞】採掘・建築・クラフトなど一連の作業（…a process consisting of coordinated steps…）
  2. 【名詞】自動装置・機械が機能している状態（…the functioning or working condition of a system…）
  3. 【名詞】大規模なプロジェクトや企業活動（…a large-scale organized activity…）
  4. 【名詞】数学における計算処理（…a mathematical calculation or procedure…）

⛏️ マインクラフト例文

⛏Mining operations need a lot of stone.

採掘操作には大量の石が必要です。

/ˈmaɪnɪŋ ˌɑːpəˈreɪʃənz niːd ə ˈlɑt əv ˈstoʊn/
  • need a: 語末 /d/ が母音 /ə/ に連結 [ˈniːdə] → 「ニーダ」
  • lot of: /t/ がフラップT化 [ɾ]、"of" は弱形 /əv/ [ˈlɑɾəv] → 「ラーラヴ」
マイニング・オペレーションズ・ニーダ・ラーラヴ・ストーン

⛏Please start a chicken-farm operation immediately.

ニワトリ農場の操作をすぐに始めてください。

/pliːz stɑːrt ə ˈtʃɪkən ˌfɑːrm ˌɑːpəˈreɪʃən ɪˈmiːdɪətli/
  • start a: 語末 /t/ がフラップT化 [ɾ]、"a" は弱形 /ə/ [ˈstɑːɾə] → 「スターラ」
  • operation immediately: 語末 /n/ が母音 /ɪ/ に連結 [ˌɑːpəˈreɪʃən ɪˈmiːdɪətli] → 「オペレーション・イミーディットリ」
プリーズ・スターラ・チキン・ファーム・オペレーション・イミーディットリ

⛏Set up a redstone operation to automate farming.

自動農場の操作をセットアップしてください。

/sɛt ʌp ə ˈɹɛdˌstoʊn ˌɑːpəˈreɪʃən tə ˈɑːtəˌmeɪt ˈfɑːrmɪŋ/
  • set up: 語末 /t/ が母音 /ʌ/ に連結 [sɛt ʌp] → 「セット・アップ」
セット・アップ・ア・レッドストーン・オペレーション・タ・オートメイト・ファーミング
