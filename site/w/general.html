general

形容詞・名詞/ˈdʒɛnərəl/CEFR A2頻度#403
  1. 【形容詞】全体的で詳細でない。個々の部分を省く（…idea, overview）
  2. 【形容詞】多くの人や物に共通する。広くあてはまる（…knowledge, skills）
  3. 【形容詞】多様な領域にかかわる。特定分野に限定されない（…store, tool）
  4. 【形容詞】通常は、おおむね。大概の場合（…in general）

⛏️ マインクラフト例文

⛏The general idea is to mine deeper for better ores.

一般的な考え方は、より良い鉱石のためにもっと深く採掘することです。

/ðə ˈdʒɛnərəl aɪˈdɪə ɪz tə ˈmaɪn ˈdipər fɔr ˈbɛtər ˈɔrz/
  • general idea → 語末 /l/ が母音 /aɪ/ に連結 [ˌdʒɛnəɹəl aɪˈdiə] ジェネラル アイディア
  • is to → is は弱形 /ɪz/、to は弱形 /tə/。両者間に z と t が続く（連結なし）
ザ ジェネラル アイディア イズ タ マイン ディーパー フォー ベター オアーズ

⛏In general, you should avoid the Nether until you have diamonds.

一般的に、ダイヤモンドを手に入れるまでネザーを避けるべきです。

/ɪn ˈdʒɛnərəl ju ˈʃʊd əˈvɔɪd ðə ˈnɛðər ənˈtɪl ju hæv ˈdaɪmənds/
  • in general → in は弱形 /ɪn/。general は /dʒ/ で始まる子音なので連結なし

more general比較級/mɚ ˈdʒɛnərəl/

Creative mode rules are more general than survival mode restrictions.

クリエイティブモードのルールはサバイバルモード制限より広範である。

/kriˈeɪtɪv ˈmoʊd ˈrulz ɑr mɚ ˈdʒɛnrəl ðæn sərˈvaɪvəl ˈmoʊd rɪkˈstrɪkʃənz/
  • more は弱形 /mɚ/ で弱化。比較構文では通常弱形が使われる
  • than は /n/ で終わり、survival の /s/ に子音+子音で連続。連結なし
クリエイティヴ モード ルーズ アー マー ジェネラル ザン サーヴァイヴァル モード リクストリクションズ

Building basic shelters requires more general skills than creating complex redstone.

基本的なシェルター建築は複雑なレッドストーン建築より広い範囲のスキルが必要だ。

/ˈbɪldɪŋ ˈbeɪsɪk ˈʃɛltərz rɪˈkwaɪrz mɚ ˈdʒɛnrəl skɪlz ðæn kriˈeɪtɪŋ kəmˈplɛks ˈrɛdˌstoʊn/
  • basic の /k/ と shelters の /ʃ/ は子音+子音。連結なし
  • requires の /z/ と more の /m/ は子音+子音。連結なし
  • than の /n/ と creating の /k/ は子音+子音。連結なし

most general最上級/ˈmoʊst ˈdʒɛnərəl/

Collecting wood is the most general task any new player should do first.

木を集めることは、すべての新規プレイヤーが最初にすべき最も一般的なタスクだ。

/kəˈlɛktɪŋ ˈwʊd ɪz ðə ˈmoʊst ˈdʒɛnərəl ˈtæsk ˈɛni ˈnu ˈpleɪər ʃʊd ˈdu ˈfɝst/
  • Collecting は /ŋ/ で終わり（velar nasal）、the の /ð/ に先行。nasalと fricativeで子音+子音
  • most の /st/ と general の /dʒ/ は子音+子音。最上級では most が強調されることが多い
  • should と do は /d/+/d/ で同一子音が連続（gemination）
コレクティング ウッド イズ ザ モースト ジェネラル タスク エニー ニュー プレイアー シュッド ドゥー ファースト

Understanding Minecraft mechanics at the most general level helps you learn faster.

マインクラフトのメカニクスを最も基本的なレベルで理解することが、より速く学ぶのに役立つ。

/ˌʌndərˈstændɪŋ ˈmaɪnkræft məˈkænɪks æt ðə ˈmoʊst ˈdʒɛnərəl ˈlɛvəl hɛlps ju ˈlɝn ˈfæstər/
  • the は弱形 /ðə/。most の /m/ に先行。母音+子音で滑らかに接続
  • most の /st/ と general の /dʒ/ は子音+子音。連結なし
  • level は /əl/ で終わり（syllabic l）、helps の /h/ に先行。子音+子音

/ɪn ˈdʒɛnərəl ˈrɛdˌstoʊn ˈmɛkənɪzəmz ɑr kəmˈplɛks tə bɪld/

  • to build → to は弱形 /tə/。build は /b/ で子音。連結なし
