ask

動詞/æsk/CEFR A1頻度#133
  1. 【動詞】（人に）質問する、尋ねる（ask someone a question）
  2. 【動詞】～を求める、懇願する（ask for help）
  3. 【動詞】人を招待する、誘う（ask someone to dinner）

⛏️ マインクラフト例文

⛏Can you ask the villager about enchanted books?

エンチャント本について村人に尋ねてもいいですか？

/kæn juː æsk ðə ˈvɪlɪdʒɚ əˈbaʊt ɪnˈtʃɑntɪd bʊks/
  • can の末尾 /n/ が you の /j/ に連結 [kæn juː]
  • the は機能語の弱形 /ðə/
  • about は弱形 /əˈbaʊt/
キャン ユー アスク ザ ヴィリジャー アバウト インチャンティド ブックス

⛏She asked if we could find diamonds in this cave.

彼女は、この洞窟でダイヤモンドが見つかるかどうかを聞きました。

/ʃi æskt ɪf wiː kəd faɪnd ˈdaɪəmɑndz ɪn ðɪs keɪv/
  • if は機能語の弱形 /ɪf/
  • could は機能語の弱形 /kəd/
  • in の末尾 /n/ が this の /ð/ に連結 [ɪn ðɪs]

🔤 活用形・複数形

asks三人称単数現在形/æsks/

She always asks if it's safe to go out at night.

彼女は夜に外に出ても安全かいつも確認する。

/ʃi ˈɔːlweɪz ˈæsks ɪf ɪts ˈseɪf tə ˈɡoʊ aʊt‿æt naɪt/
  • She → 弱形 /ʃi/
  • asks if → /s/ と母音 /ɪ/ の間に word boundary があり、通常は有声化しない
  • it's → 短縮形、is の有声摩擦音 /z/ を保持
  • to go → to は弱形 /tə/
シ オールウェイズ アスク イフ イツ セイフ タ ゴー アウアッ ナイト

He never asks for help when mining diamonds.

彼はダイヤを採掘中、決して助けを求めない。

/hi ˈnevɚ ˈæsks fɚ help wen ˈmaɪnɪŋ ˈdaɪməndz/
  • He → 弱形 /hi/
  • never asks → /r/ 脱落後、/s/ が無声子音で変化なし
  • asks for → 語末 /s/ が無声摩擦音 /f/ に先行、連結なし
  • for → 弱形 /fɚ/
  • when → 通常形

  • asking for → 語末 /ŋ/ が無声摩擦音 /f/ に先行、連結なし
