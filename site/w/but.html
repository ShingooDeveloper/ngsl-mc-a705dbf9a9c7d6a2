but

接続詞・前置詞/bʌt/CEFR A1頻度#24
  1. 【接続詞】(対比) しかし、だが、けれど。前文と異なる情報を示す（…but I need…）
  2. 【接続詞】(only but で) 〜だけである、〜でしかない。制限の意味（…only but a few…）
  3. 【前置詞】〜を除いて、〜以外に。除外を表す（everyone but him）
  4. 【前置詞】(all but で) ほぼ、ほとんど。ほぼ完全に（all but complete）

⛏️ マインクラフト例文

⛏I want to build a house, but I need more wood.

家を建てたいのですが、もっと木が必要です。

/aɪ ˈwɑnt tə ˈbɪld ə ˈhaʊs bʌt aɪ ˈniːd mɔr ˈwʊd/
  • want to の /t/ が /ə/ に連結 [ˈwɑnə]
  • build a の /d/ が /ə/ に連結 [ˈbɪldə]
