user

名詞/ˈjuːzɚ/CEFR A2頻度#1112
  1. 【名詞】ゲームやデバイスを使う人（…a person who plays a game or operates a device…）
  2. 【名詞】サーバーのアカウント。プレイヤー（…an account on a server; a player…）
  3. 【名詞】複数のプレイヤー、ユーザーグループ（…players; group of players on a server…）

⛏️ マインクラフト例文

⛏A new user should read the server rules first.

新しいユーザーは最初にサーバールールを読むべきだ。

/ə ˈnu ˈjuːzɚ ˈʃʊd ˈrid ðə ˈsɝvɚ ˈrulz ˈfɝst/
  • A user：弱形 /ə/ は母音のような /j/ に続く [əˈjuːzɚ] アユーザー
  • read the：語末 /d/ の後に /ð/ が続き未開放になる [ˈrid̚ ðə] リッザ
  • server rules：語末 /ɚ/ と /r/ の間は境界。rules の /z/ が次の音に続く [ˈsɝvɚ ˈrulz] サーヴァー・ルールズ
ア・ニュー・ユーザー・シュッド・リッザ・サーヴァー・ルールズ・ファースト

⛏Every user in this world can build anything.

このワールドのすべてのユーザーは何でも建築できる。

/ˈɛvri ˈjuːzɚ ɪn ðɪs ˈwɝld kən ˈbɪld ˈɛniˌθɪŋ/
  • Every user：/i/ が /j/ に自然に連なる [ˈɛvri ˈjuːzɚ] エヴリ・ユーザー
  • can build：弱形 /kən/、語末 /n/ の後に /b/ が続く [kən ˈbɪld] カン・ビルド
  • anything：語末 /ŋ/ で文が終わる [ˈɛniˌθɪŋ] エニシング
エヴリ・ユーザー・イン・ディス・ワールド・カン・ビルド・エニシング

⛏That user has admin powers to protect our base.

そのユーザーは私たちの基地を守るための管理者権限を持っている。

/ðæt ˈjuːzɚ hæz ˈædmɪn ˈpaʊɚz tə prəˈtɛkt ˈaʊər ˈbeɪs/
  • That user：語末 /t̚/ は未開放で /j/ に続く [ðæt̚ ˈjuːzɚ] ザット・ユーザー
  • user has：/ɚ/ の後に /h/ が続く [ˈjuːzɚ hæz] ユーザー・ハズ（h は無声音）
  • to protect：弱形 /tə/、/t/ が /p/ に先行 [tə prəˈtɛkt] タ・プロテクト
ザット・ユーザー・ハズ・アドミン・パワーズ・タ・プロテクト・アウアー・ベイス

💬 よく使う熟語（マイクラ例文）

end user最終ユーザー。実際にゲームやサービスを使う一般的なプレイヤー

⛏As an end user, I don't care how the server plugin code works.

最終ユーザーとして、サーバープラグインのコードがどのように機能するかは関心がない。

/æz ən ˈɛnd ˈjuːzɚ aɪ ˈdoʊnt ˈkɛr haʊ ðə ˈsɝvɚ ˈplʌɡɪn ˈkoʊd ˈwɝks/
  • end user：語末 /d/ が /j/ に連結 [ˈɛnd ˈjuːzɚ] エンド・ユーザー
  • an end：冠詞 an は /ən/、end は /ɛ/ で始まる [ən ˈɛnd] アン・エンド
  • server plugin：語末 /ɚ/ の後に /p/ が続く。plugin の /ŋ/ は閉鼻音 [ˈsɝvɚ ˈplʌɡɪn] サーヴァー・プラグイン
アズ・アン・エンド・ユーザー・アイ・ドウント・ケア・ハウ・ザ・サーヴァー・プラグイン・コウド・ワークス
