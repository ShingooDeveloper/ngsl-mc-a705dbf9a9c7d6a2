admit

動詞/ədˈmɪt/CEFR B2頻度#1031
  1. 【動詞】（過ち・失敗など）を認める、白状する（...が事実であることを認める）
  2. 【動詞】（人を）入場させる、認める（...への出入りを許可する）
  3. 【動詞】（事実・状況）を認める、受け入れる（異を唱えずに認める）

⛏️ マインクラフト例文

⛏I admit I built this structure terribly.

この構造をひどく作ってしまったことを認めます。

[aɪ ədˈmɪɾaɪ ˈbɪlt ðɪs ˈstrʌkʃɚ ˈtɛɾəbli]
  • admit I → admit の語末 /t/ が I /aɪ/ に連結し、母音と母音の間で /t/ はフラップ化: [ədˈmɪɾaɪ]
  • structure terribly → structure /ɚ/ と terribly /t/ の間に linking なし。terribly 内の /r/ はフラップ化して [ɾ]
アイ アドミラアイ ビルト ディス ストラクチャー テリブリ

⛏You have to admit this ore is rare.

この鉱石が稀だということを認めざるを得ません。

[juː ˈhæf.tə ədˈmɪt ðɪs ˈɔɹ ɪz ˈrɛɹ]
  • have to → have /v/ が to の弱形 /tə/ に連結: [ˈhæftə]
  • to admit → 弱形 to /tə/ と admit /ə/ が接続。特に assimilation なし
  • ore is → ore の /ɔɹ/ と is /ɪz/ の境界。r が母音 /ɪ/ に引き継がれ [ɔɹɪz]
ユー ハフタ アドミット ディス オア イズ レア

⛏Players should admit mining is truly addictive.

プレイヤーはマイニングが本当に中毒性があることを認めるべきです。

[ˈpleɪɚz ˈʃʊd ə ˈdmɪt ˈmaɪnɪŋ ɪz ˈtruːli əˈdɪktɪv]
  • should admit → should /d/ が admit /ə/ に連結: [ˈʃʊd ə]
  • admit mining → admit の /t/ が mining の /m/ に接する子音+子音の境界。admit の /t/ は未開放 [t̚]
  • truly addictive → truly /i/ と addictive /ə/ は両者母音なので特に linking なし
プレイアーズ シュドア ドミット マイニング イズ トゥーリ アディクティブ

💬 よく使う熟語（マイクラ例文）

admit defeat敗北を認める、降参する

⛏We finally admitted defeat when the dragon killed us all.

ドラゴンに全員殺されたとき、私たちはついに降参を認めた。

[wi ˈfaɪnəli ədˈmɪɾəd dɪˈfit ˈwɛn ðə ˈdɹæɡən ˈkɪld əs ˈɔl]
  • admitted defeat → admitted /ɪd/ の中の /t/ が /ə/ に連結してフラップ化 [ɾ]、その後 /əd/ 接尾辞。語末 /d/ と defeat の /d/ が接する際、最初の /d/ は未開放 [t̚]
  • dragon killed → /ŋ/ と /k/ の間に linking なし
  • killed us → killed /d/ が us /ə/ に連結: [ˈkɪld əs]

⛏He won't admit he destroyed the farm on purpose.

彼は農場を意図的に破壊したことを認めようとしない。

[hi ˈwoʊnt ədˈmɪt hi dɪsˈtɹɔɪd ðə ˈfɑrm ɑn ˈpɝpəs]
  • won't admit → won't /t/ と admit /ə/ が接する子音+母音の境界。won't の /t/ は未開放 [t̚]
  • admit he → admit /t/ と he /h/ は子音+子音で special linking なし
  • destroyed the → destroyed /d/ と the /ð/ の弱形。特に linking なし
