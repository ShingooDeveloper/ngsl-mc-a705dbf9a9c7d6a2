request

名詞・動詞/rɪˈkwɛst/CEFR B1頻度#1008
  1. 【名詞】求め、要望。（～を求める）
  2. 【名詞】頼みごと、願い。（～のお願い）
  3. 【動詞】～に～を求める、～を要求する。
  4. 【動詞】～にお願いする、～をリクエストする。

⛏️ マインクラフト例文

⛏Request iron ore from our supply team.

補給チームから鉱石をリクエストしてください。

/rɪˈkwɛst ˈaɪɚn ˈoɚ fɹəm ˈaʊɚ səˈplaɪ ˈtim/
  • request /t/ が iron /aɪ/ に連結 [rɪˈkwɛstaɪɚn]
  • from は前置詞で弱形 /fɹəm/ に
リクウェスタイアン オア フラム アウァー サプライ ティーム

⛏She requested that we gather wood before the storm arrives.

彼女は嵐が来る前に木を集めるよう要求した。

/ʃi rɪˈkwɛstəd ðæt wi ˈɡæðɚ ˈwʊd bɪˈfoɹ ðə ˈstoɹm əˈɹaɪvz/
  • before /oɹ/ が the の弱形 /ðə/ に滑らかに連結
シー リクウェステッド ザット ウィー ゲザー ウッド ビフォア ザ ストーム アライブズ

⛏The server admin will accept your request for more building area.

サーバー管理人がさらに多くの建築エリアのリクエストを受け入れます。

/ðə ˈsɚvɚ ˈædmɪn wɪl əkˈsɛpt jɔɹ rɪˈkwɛst fɔɹ ˈmoɹ ˈbɪldɪŋ ˈɛɹiə/
  • building /ŋ/ が area /ɛ/ に連結 [ˈbɪldɪŋɛɹiə]
  • for は前置詞で弱化して /fɚ/ に
ザ サーバー アドミン ウィル アクセプト ユアー リクウェスト フォー モア ビルディングエリア

💬 よく使う熟語（マイクラ例文）

make a requestリクエストを出す、願いを述べる

⛏I need to make a request for better tools at the crafting table.

クラフティングテーブルでもっと良い道具をリクエストする必要があります。

/aɪ ˈnid tə ˈmeɪk ə rɪˈkwɛst fɔɹ ˈbɛtɚ ˈtulz æt ðə ˈkɹæftɪŋ ˈteɪbəl/
  • need /d/ が to の弱形 /tə/ に→未開放で連結 [ˈniːd̚ə]
  • make /k/ が a の弱形 /ə/ に連結 [ˈmeɪkə]
