pattern

名詞・動詞/ˈpætɚn/CEFR A2頻度#881
  1. 【名詞】反復する配置や構造。視覚的な規則性（ブロックの模様、地形の配置等における規則的な繰り返し）
  2. 【名詞】従うべきモデルやテンプレート（建築設計図、クラフトレシピ、マイニング戦略等のガイド）
  3. 【名詞】一連の行動や出来事の規則的な流れ（鉱脈生成やモブスポーン等における配置パターン）
  4. 【動詞】～に基づいて設計する、～の様式を真似る（建築家や先達の様式を踏襲する）

⛏️ マインクラフト例文

⛏Patterns of stone bricks make a strong wall.

石レンガのパターンは丈夫な壁を作ります。

[ˈpætɚnz əv stoʊn brɪks meɪk ə strɔŋ wɔl]
  • Patterns の末尾 /z/ が of の /ə/ に連結 [ˈpætɚnz əv]→「パターンザヴ」
  • of の /v/ が stone の /s/ に連結しない（子音同士）
パターンザヴ ストーン ブリックス メイク ア ストロング ウォール

⛏I followed a pattern to mine efficiently.

効率的に採掘するためにパターンに従いました。

[aɪ ˈfɑloʊd ə ˈpætɚn tə maɪn ɪˈfɪʃəntli]
  • followed の /d/ が a の /ə/ に連結 [ˈfɑloʊd ə]→「フォロードア」
  • pattern の /n/ が to の弱形 /tə/ に連結 [ˈpætɚn tə]→「パターンタ」
アイ フォロード ア パターンタ マイン イフィシャントリー

⛏The redstone pattern activates all the doors at once.

そのレッドストーンのパターンはすべてのドアを一度に開きます。

[ðə ˈrɛdstoʊn ˈpætɚn ˈæktɪveɪts ɔl ðə dɔrz æt wʌns]
  • redstone の /n/ が pattern の /p/ に連結しない（子音同士）
  • pattern の /n/ が activates の /æ/ に連結→「パターナクティベイツ」のように聞こえる
ザ レッドストーン パターナクティベイツ オール ザ ドーズ アット ワンス
