coach

名詞・動詞/koʊtʃ/CEFR B1頻度#1339
  1. 【名詞】コーチ、トレーナー（競技や技能の指導者）
  2. 【名詞】指導者、教練者
  3. 【動詞】（…を）教える、訓練する
  4. 【動詞】（…に）指導をする、コーチをする

⛏️ マインクラフト例文

⛏You need a good coach for mining tips.

採掘のテクニックを学ぶには、良いコーチが必要だ。

/ju ˈniːdə ˈɡʊd ˈkoʊtʃ fər ˈmaɪnɪŋ ˈtrɪps/
  • need の /d/ が a の /ə/ に連結 [ˈniːdə]
  • for は弱形 /fər/
ユー ニーダ グッド コウチ ファー マイニング トリップス

⛏Hire a coach to help everyone survive!

みんなが生き残るのを手伝ってくれるコーチを雇おう！

/ˈhaɪrə ˈkoʊtʃ tə ˈhɛlpˌɛvriˌwʌn sərˈvaɪv/
  • hire の /r/ が a の /ə/ に連結 [ˈhaɪrə]
  • to は弱形 /tə/
  • help の /p/ が everyone の /ɛ/ に連結 [ˈhɛlpˌɛvriˌwʌn]
ハイラ コウチ タ ヘルペヴリワン サー ヴァイヴ

⛏The best coach will set goals and teach strategy!

最高のコーチは目標を設定して戦略を教えるだろう。

/ðə ˈbɛst ˈkoʊtʃ wɪl sɛt ˈɡoʊlzən ˈtitʃ ˈstrætədʒi/
  • the は弱形 /ðə/
  • goals の /z/ が and の /ə/ に連結 [ˈɡoʊlzən]
ザ ベスト コウチ ウィル セット ゴウルズエン ティーチ ストラテジー
