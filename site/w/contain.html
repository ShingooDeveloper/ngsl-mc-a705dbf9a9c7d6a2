contain

動詞/kənˈteɪn/CEFR A2頻度#662
  1. 【動詞】（容器などが）〜を含む・入っている・保持する（アイテムやリソースなど）
  2. 【動詞】（内に）〜を持つ・保有する・所持する
  3. 【動詞】〜を抑制する・制御する・広がりを防ぐ（溶岩や爆発など）

⛏️ マインクラフト例文

⛏This chest contains valuable diamonds and emeralds.

このチェストには貴重なダイヤモンドと翡翠が入っている。

/ðɪs ˈtʃɛst kənˈteɪnz ˈvæljəbəl ˈdaɪəmənz ənd ˈɛmərəldz/
  • 「diamonds」末尾 /z/ が「and」の弱形に連結 [ˈdaɪəmənz‿ən]
ディス・チェスト・カンテインズ・ヴァリュアブル・ダイアモンズン・エメラルズ

⛏The inventory doesn't contain the items I need.

このインベントリには必要なアイテムが入っていない。

/ðə ɪnˈvɛntəri ˈdʌzənt kənˈteɪn ðə ˈaɪtəmz aɪ niːd/
  • 「items」末尾 /z/ が「I」の母音に連結 [ˈaɪtəmz‿aɪ]
  • 「doesn't」末尾 /t/ が「contain」の冒頭 /k/ に未開放で続く
ザ・インベントリ・ダズント・カンテイン・ザ・アイテムズアイ・ニード

⛏We need to contain the lava flow before it spreads to the village.

溶岩が村に広がる前に、その流れを食い止める必要がある。

/wi niːd tə kənˈteɪn ðə ˈlɑvə floʊ bɪˈfɔr ɪt spɹɛdz tə ðə ˈvɪlɪdʒ/
  • 「need」末尾 /d/ が「to」の弱形に連結 [niːd‿ə]
  • 「spreads」末尾 /z/ が「to」の弱形に連結 [spɹɛdz‿ə]
ウィ・ニーダ・カンテイン・ザ・ラバ・フロー・ビフォー・イット・スプレッザ・ザ・ヴィレッジ

💬 よく使う熟語（マイクラ例文）

contain oneself自分を抑制する・我慢する・落ち着く

⛏I can barely contain myself when I find diamonds!

ダイヤを見つけた時、興奮を抑えられない！

/aɪ kæn ˈbeərli kənˈteɪn maɪˈsɛlf wɛn aɪ faɪnd ˈdaɪəmənz/
  • 「when」末尾 /n/ が「I」の母音に連結 [wɛn‿aɪ]
  • 「find」末尾 /d/ が「diamonds」の冒頭 /d/ に連結 [faɪnd‿ˈdaɪ-]
アイ・キャン・ベアリー・カンテイン・マイセルフ・ウェンアイ・ファインダイアモンズ
