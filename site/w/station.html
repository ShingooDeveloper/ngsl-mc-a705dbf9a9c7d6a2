station

名詞・動詞/ˈsteɪʃən/CEFR B1頻度#708
  1. 【名詞】人々が集まる、または特定の目的のための建物や施設（例：駅、警察署、採掘ステーション）
  2. 【名詞】軍隊や警察などが駐在する場所、またはそこでの職務地点
  3. 【名詞】社会的な身分や立場
  4. 【動詞】～を特定の場所に配置する、配置された状態にする

⛏️ マインクラフト例文

⛏I built a mining station near the diamond ore.

ダイヤモンド鉱石の近くにマイニングステーションを建設しました。

/aɪ bɪlt ə ˈmaɪnɪŋ ˈsteɪʃən nɪr ðə ˈdaɪmənd ɔr/
  • built a → /t/ が母音の前でフラップT [ɾ] になり [bɪɾə]
  • near the → near の /r/ が the の弱形 /ðə/ に連結 [ˈnɪrðə]
  • the diamond → the は弱形 /ðə/
アイ ビルラ マイニング ステイション ニアザ ダイモンド オア

⛏This breeding station helps me raise cows and sheep together.

このブリーディングステーションは、牛と羊を一緒に育てるのに役立っています。

/ðɪs ˈbriːdɪŋ ˈsteɪʃən ˈhɛlps mi reɪz kaʊz ən ʃip təˈɡɛðər/
  • cows and → cows の /z/ が and の /ə/ に連結、and は弱形 /ən/ [kaʊzən]
  • sheep together → sheep の /p/ は未開放で together の弱い /tə/ に繋がる

⛏We need to establish a main station as our base.

メインステーションを確立し、それを基地とする必要があります。

/wi nid tə ɪˈstæblɪʃ ə ˈmeɪn ˈsteɪʃən əz ˈaʊr beɪs/
  • need to → to は機能語の弱形 /tə/
  • a main → a は弱形 /ə/
  • as our → as の /z/ が our の /aʊr/ に連結 [ˈəzaʊr]

/aɪ nid tə sɛt ʌp ə nu ˈsteɪʃən fɔr ˈmaɪnɪŋ əbˈsɪdiən ˈseɪfli/

  • set up a → set の /t/ が up の /ʌ/ の前でフラップT [ɾ] になり、up の /p/ が a の /ə/ に連結 [ˈsɛɾʌpə]
  • for mining → for は弱形 /fɔr/、/r/ が mining の /m/ に連結 [ˈfɔrmaɪn...]
