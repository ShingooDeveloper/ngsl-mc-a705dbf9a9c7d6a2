any

限定詞・代名詞/ˈɛni/CEFR A1頻度#82
  1. 【限定詞】（肯定文）どんな…でも、どれでも（…any block, any material）
  2. 【限定詞】（否定文）…がない（…don't have any）
  3. 【限定詞】（疑問文）何か、いくつか（…have you seen any）
  4. 【代名詞】どれでも、何でも（…you can use any）

⛏️ マインクラフト例文

⛏You can craft any tool with the right materials.

適切な材料があれば、どんな道具でも作製できます。

/juː kən ˈkræft ˈɛni ˈtuːl wɪð ðə ˈraɪt̚ məˈtɛɹiəlz/

  • the は弱形 /ðə/
