amaze

動詞/əˈmeɪz/CEFR B1頻度#1521
  1. 【動詞】～を驚かせる、びっくりさせる（非常に驚かせる、深い印象を与える）
  2. 【動詞】～を圧倒する、度肝を抜く（優れた技巧や規模で心を奪う）

⛏️ マインクラフト例文

⛏I was amazed when I discovered a hidden diamond cave.

隠れたダイヤモンド洞窟を発見したとき、私は驚いた。

/aɪ wəz əˈmeɪzd wɛn aɪ dɪsˈkʌvərd ə ˈhɪdn ˈdaɪəmənd keɪv/
  • was の /z/ が amazed の /ə/ に連結 [wəz əˈmeɪzd]→「ワザメイズド」
  • when の /n/ が I の /aɪ/ に連結 [wɛn aɪ]→「ウェナイ」
  • discovered の /d/ が a の /ə/ に連結 [dɪsˈkʌvərd ə]→「ディスカヴァーダ」
アイ ワザメイズド ウェナイ ディスカヴァーダ ヒドン ダイアモンド ケイヴ

⛏Your building skills amaze everyone on the server.

あなたの建築スキルはサーバー上の全員を驚かせている。

/jɔr ˈbɪldɪŋ skɪlz əˈmeɪz ˈɛvriwʌn ɑn ðə ˈsɝvər/
  • skills の /z/ が amaze の /ə/ に連結 [skɪlz əˈmeɪz]→「スキルザメイズ」
  • the は弱形 /ðə/
ユア ビルディング スキルザメイズ エヴリワン オン ザ サーバー

⛏The mob spawner amazed us with its unexpected size.

モブスポーナーは予期しない大きさで私たちを驚かせた。

/ðə mɑb ˈspɔnər əˈmeɪzd ʌs wɪð ɪts ʌnɪkˈspɛktəd saɪz/
  • spawner の /r/ が amazed の /ə/ に連結 [ˈspɔnər əˈmeɪzd]→「スポーナラメイズド」
  • amazed の /d/ が us の /ʌ/ に連結 [əˈmeɪzd ʌs]→「アメイズダス」
  • its の /s/ が unexpected の /ʌ/ に連結 [ɪts ʌnɪkˈspɛktəd]→「イッサンエクスペクティド」
ザ モブ スポーナラメイズダス ウィズ イッサンエクスペクティド サイズ

💬 よく使う熟語（マイクラ例文）

be amazed at～に驚く、～に感動する

⛏I was amazed at how fast you built that house.

あなたがそれほど速く家を建てたことに驚いた。

/aɪ wəz əˈmeɪzd æt haʊ fæst ju bɪlt ðæt haʊs/
  • was の /z/ が amazed の /ə/ に連結 [wəz əˈmeɪzd]→「ワザメイズド」
  • amazed の /d/ が at の /æ/ に連結し、全体で [wəz əˈmeɪzd æt]→「ワザメイズダット」
アイ ワザメイズダット ハウ ファスト ユー ビルト ザット ハウス
be amazed by～に圧倒される、～に驚かされる

⛏Everyone was amazed by the Nether portal design.

誰もがネザーポータルのデザインに驚いた。

/ˈɛvriwʌn wəz əˈmeɪzd baɪ ðə ˈnɛðər ˈpoʊrtəl dɪˈzaɪn/
  • was の /z/ が amazed の /ə/ に連結 [wəz əˈmeɪzd]→「ワザメイズド」
  • the は弱形 /ðə/
