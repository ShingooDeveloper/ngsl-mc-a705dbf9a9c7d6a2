addition

名詞/əˈdɪʃən/CEFR B1頻度#706
  1. 【名詞】加算、足し算（数学的な操作または計算結果）
  2. 【名詞】追加、付け足し（既存の物に加えられる行為または物）
  3. 【名詞】増築部分、拡張部分（建物などへの追加構造）
  4. 【名詞】『in addition to』の形で「〜に加えて」（〜の他に）

⛏️ マインクラフト例文

⛏I built an addition to the main base.

メインの拠点に増築部分を追加した。

[aɪ ˈbɪɾən əˈdɪʃən tə ðə ˈmeɪn ˈbeɪs]
  • 語末子音 /t/ が母音 /ə/ に連結してフラップ化 [ˈbɪɾən] ビラン
  • to は弱形 /tə/
  • the は弱形 /ðə/
アイ ビラン アディション タ ザ メイン ベイス

⛏I finished my cobblestone house addition.

丸石の家の増築部分を完成させた。

[aɪ ˈfɪnɪʃt̚ maɪ ˈkɑbəlstoʊn ˈhaʊsə ˈdɪʃən]
  • finished の語末 /t/ は未開放 [ˈfɪnɪʃt̚]
  • 語末子音 /s/ が母音 /ə/ に連結 [ˈhaʊsə] ハウサ
アイ フィニシュト マイ カバルストウン ハウサディション

💬 よく使う熟語（マイクラ例文）

in addition to〜に加えて、さらに〜

⛏In addition to building a house, we need to create a farm.

家を建てることに加えて、農場も作る必要があります。

[ɪn əˈdɪʃən tə ˈbɪldɪŋ ə ˈhaʊs, wiː ˈniːd tə kriˈeɪt ə ˈfɑrm]
  • to は機能語で弱形 /tə/
  • 機能語 to が繰り返し弱化
イン アディション タ ビルディング ア ハウス、ウィー ニード タ クリエイト ア ファーム
in additionさらに、その上

⛏We built the main house. In addition, we created a workshop.

メインの家を建てました。さらに、作業所も建築しました。

[wiː ˈbɪlt̚ ðə ˈmeɪn ˈhaʊs. ɪn əˈdɪʃən, wiː kriˈeɪtɪdə ə ˈwɜrkʃɑp]
  • built の語末 /t/ は未開放 [ˈbɪlt̚]
  • the は弱形 /ðə/
  • created a—語末子音 /d/ が母音 /ə/ に連結
