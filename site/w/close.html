close

動詞・形容詞/kloʊz/CEFR A2頻度#238
  1. 【動詞】shut（…を閉じる；ドアやチェストなど）
  2. 【形容詞】near（近い；…に接近している）
  3. 【形容詞】intimate/tight（親密な；密接した）
  4. 【動詞】end/finish（終わる；終える）

⛏️ マインクラフト例文

⛏Close the chest before the mobs find it.

モブに見つかる前にチェストを閉じてください。

/ˈkloʊz ðə ˈtʃɛst bɪˈfɔr ðə ˈmɑbz ˈfaɪnd ɪt/
  • close の z が the の ə に連結 [ˈkloʊzə]
  • before の r が the の ə に連結 [bɪˈfɔrə]
  • find の d が it の ɪ に連結 [ˈfaɪndɪt]
