write

動詞・名詞/raɪt/CEFR A1頻度#120
  1. 【動詞】ペンなどで文字や単語を紙に記す（…I write my name on the paper）
  2. 【動詞】手紙や本などを作成して記す、著作する（…She writes guides for new players）
  3. 【動詞】看板などの表面に文字を記す（…write on a sign）
  4. 【名詞】書く行為、文字を記すこと
  5. 【動詞】記録する、書き記す（…write down coordinates）

⛏️ マインクラフト例文

⛏I write down the recipe in my notebook.

ノートにレシピを書き留める。

/aɪ ˈraɪt̚ ˈdaʊn ðə ˈrɛsəpi ɪn maɪ ˈnoʊɾbʊk/
  • write down：write の語末子音 /t/ が未開放 [t̚]。下の /d/ が続くため。
  • the：機能語として弱形 /ðə/ に弱化。recipe の /r/ は子音なので連結なし。
  • notebook：の t が母音に挟まれてフラップ化 [ɾ]「ラ」。
アイ ライッ ダウン ザ レセピ イン マイ ノーラブック

⛏You can write signs to share information with other players.

看板に書いて他のプレイヤーと情報を共有できる。

/juː kən ˈraɪt̚ ˈsaɪnz tə ˈʃɛr ɪnˈfɔrmeɪʃən wɪð ˈʌðɚ ˈpleɪɚz/
  • can：弱形 /kən/ に弱化。
  • write signs：write の /t/ が未開放 [t̚]。下が /s/ で子音。
  • to：機能語は弱形 /tə/ に弱化「タ」。
  • other：弱形 /ɚ/ で後ろの r-colored schwa。
ユー カン ライッ サインズ タ シェア インフォメーション ウィズ アザー プレイアーズ

⛏Players write helpful tips and tricks in community forums.

プレイヤーはコミュニティフォーラムに役に立つコツを書く。

/ˈpleɪɚz ˈraɪt̚ ˈhɛlpfəl ˈtɪps ənd ˈtrɪks ɪn kəˈmjuːnəti ˈfɔrəmz/
  • write helpful：write の /t/ が未開放 [t̚]。下が /h/ で子音。
  • and：弱形 /ənd/ → /ən/ に弱化「アン」。
  • community：second syllable は /ˈmjuː/ で，mi はアメリカ英語では /mi/ ではなく /mju/（ミューニティ）。
