all

限定詞・代名詞・形容詞・副詞/ɔːl/CEFR A1頻度#32
  1. 【限定詞・代名詞】すべての、全部の（…）
  2. 【代名詞】全部、それら全て（…）
  3. 【形容詞】完全な、全体の（…）
  4. 【副詞】完全に、全く（…）

⛏️ マインクラフト例文

⛏All the diamonds are deep underground.

すべてのダイヤモンドは地下深くにある。

/ɔːl ðə ˈdaɪmənz ɑr ˈdiːp ˌʌndərˈɡraʊnd/
    オール ザ ダイマンズ アー ディープ アンダーグラウンド

    ⛏We need all the resources for this build.

    このビルドにはすべてのリソースが必要だ。

    /wi niːd ɔːl ðə rɪˈsɔrsəz fər ðɪs bɪld/
    • need の /d/ が all の /ɔ/ に連結 [niːdɔːl]
    • the は弱形 /ðə/
    • for は弱形 /fər/
    ウィ ニードール ザ リソーシズ ファ ディス ビルド

    ⛏I've collected all the blocks we need.

    必要なブロックをすべて集めた。

    /aɪv kəˈlɛktəd ɔːl ðə blɑks wi niːd/
    • collected の /d/ が all の /ɔ/ に連結 [kəˈlɛktədɔːl]
    • the は弱形 /ðə/
    アイヴ コレクティドオール ザ ブロックス ウィ ニード

    💬 よく使う熟語（マイクラ例文）

    all day一日中、朝から晩まで

    ⛏We mined ore all day yesterday.

    昨日は一日中、鉱石を採掘した。

    /wi maɪnd ɔr ɔːl ˈdeɪ ˈjɛstərdeɪ/
    • ore の /r/ が all の /ɔ/ に連結 [ɔrɔːl]

    all right大丈夫、良好な状態

    ⛏The base looks all right now.

    この拠点は今、良好な状態に見える。

    /ðə ˈbeɪs lʊks ɔːl ˈraɪt naʊ/
    • looks の /s/ が all の /ɔ/ に連結 [lʊksɔːl]
    ザ ベイス ルックソール ライト ナウ

    全部で50個のダイヤモンドを見つけた。

    /wi faʊnd ˈfɪfti ˈdaɪmənz ɪn ɔːl/

    ウィ ファウンド フィフティ ダイマンズ イノール
