what

代名詞・限定詞/wɑt/CEFR A1頻度#50
  1. 【代名詞】疑問詞として不明な対象や事物を尋ねる（…what do you want?）
  2. 【代名詞】不定関係詞。「…なもの」という意味（…I don't know what this block is）
  3. 【限定詞】感嘆文で驚きや強い評価を表す（…what a great discovery!）
  4. 【副詞的用法】相手の発言に対する問い返しや戸惑いを示す（…What?）

⛏️ マインクラフト例文

⛏What are you mining for?

何のために掘ってるの？

/wɑt ɑr juː ˈmaɪnɪŋ fɔr/
  • What are → what /wɑt/ の /t/ が are /ɑr/ の /ɑ/ に連結してフラップ化 [wɑɾ] ワラ
  • are you → are /ɑr/ の /r/ が you /juː/ の /j/ に接続 [ɑɚj] アーユー
  • for → 機能語は弱形 /fər/ に弱化 [fɚ] ファー
ワラ ユー マイニング ファー

⛏I don't know what this block is.

このブロックが何なのか分からない。

/aɪ ˈdoʊnt noʊ wɑt ðɪs blɑk ɪz/
  • what this → what /wɑt/ の /t/ は子音 /ð/ の直前で未開放 [wɑt̚] ウォッ
  • don't → /nt/ は語末で未開放 [doʊn̚t̚] ドーンッ
アイ ドーンッ ノー ウォッ ディス ブラッキズ

⛏What a great discovery!

なんて素晴らしい発見だ！

/wɑt ə ˈɡreɪt dɪsˈkʌvəri/
  • What a → what /wɑt/ の /t/ が a /ə/ に連結してフラップ化 [wɑɾə] ワラ
  • great discovery → great /ɡreɪt/ の /t/ は子音 /d/ の直前で未開放 [ɡreɪt̚] グレイッ
ワラ グレイッ ディスカヴァリー

What if...?もし…だったら？という仮定の表現

⛏What if creepers come tonight?

もし今夜クリーパーが来たら？

/wɑt ɪf ˈkriːpərz kʌm təˈnaɪt/
  • What if → what /wɑt/ の /t/ が if /ɪf/ の /ɪ/ に連結してフラップ化 [wɑɾɪf] ワリフ
  • to → 機能語は弱形 /tə/ に弱化 [tə] タ
ワリフ クリーパーズ カム タナイト
What about...?…はどう？という提案や質問

⛏What about mining diamonds together?

ダイヤモンド採掘を一緒にしない？

/wɑt əˈbaʊt ˈmaɪnɪŋ ˈdaɪəmɑndz təˈɡɛðər/
  • What about → what /wɑt/ の /t/ が about /əˈbaʊt/ の /ə/ に連結してフラップ化 [wɑɾə-ˈbaʊt] ワラバウト
  • to → 機能語は弱形 /tə/ に弱化 [tə] タ
ワラバウト マイニング ダイアモンズ タゲザー
What's the...?…は何？という確認や質問

⛏What's the plan for building this castle?

この城を建てるプランは？

/wʌts ðə ˈplæn fɔr ˈbɪldɪŋ ðɪs ˈkæsəl/
  • What's → is が短縮して /s/。what + is = /wʌts/ [wʌts] ワッツ
  • the → 機能語は弱形 /ðə/ [ðə] ザ
  • for → 機能語は弱形 /fər/ に弱化 [fɚ] ファー
ワッツ ザ プラン ファー ビルディング ディス キャスル
