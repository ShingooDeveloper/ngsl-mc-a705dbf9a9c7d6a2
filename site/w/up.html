up

副詞・動詞・前置詞・名詞/ʌp/CEFR A1頻度#51
  1. 【副詞】上に，上へ；空間的・方向的に上昇する（rise up など）
  2. 【副詞】増加する，高くなる（level up など）
  3. 【動詞】上げる，増加させる；掘り出す（dig up, build up など）
  4. 【前置詞】～の上に，～までの間
  5. 【名詞】上昇，増加；上向きの状態

⛏️ マインクラフト例文

⛏I dig up blocks of stone.

石のブロックを掘り出す。

/aɪ ˈdɪɡ ʌp blɑks əv ˈstoʊn/
  • dig の /ɡ/ が up の /ʌ/ に連結
  • of が弱形 /əv/ で発音
アイ ディガップ ブロックス アヴ ストーン
