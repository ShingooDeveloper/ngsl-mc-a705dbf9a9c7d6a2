build

動詞・名詞/bɪld/CEFR A2頻度#220
  1. 【動詞】（建物などを）建てる・構築する（…を建設する）
  2. 【動詞】（段階的に）構成する・発展させる（…を形成する）
  3. 【名詞】建築様式・構造（…の造り）
  4. 【名詞】（人の）体格・体つき（…の体型）

⛏️ マインクラフト例文

⛏I'm building a stone house near the river.

川の近くに石の家を建築中です。

/aɪm ˈbɪldɪŋ ə stoʊn haʊs nɪr ðə ˈrɪvər/
  • the は弱形 /ðə/
アイム ビルディング ア ストォン ハウス ニア ザ リヴァ

⛏We need to build a redstone farm to get automatic harvesting.

自動収穫するため、レッドストーン装置を建築する必要があります。

/wi niːd tə bɪld ə ˈrɛdstoʊn fɑːrm tə ɡɛt ˌɔːtəˈmætɪk ˈhɑːrvɛstɪŋ/
  • to は弱形 /tə/ に弱化
  • build a → 語末子音 /d/ が母音 /ə/ に連結、フラップ化 [ˈbɪɾə]
ウィ ニード タ ビルラ レッドストーン ファーム タ ゲット オートマティック ハーベスティング

⛏They built a bridge across the lava lake yesterday.

昨日、溶岩湖を渡る橋を建築しました。

/ðeɪ bɪlt ə ˈbrɪdʒ əˈkrɔs ðə ˈlɑvə leɪk ˈjɛstərdeɪ/
  • built a → 語末子音 /t/ が母音 /ə/ に連結、フラップ化 [ˈbɪɾə]
  • the は弱形 /ðə/
ゼイ ビルラ ブリッジ アクロス ザ ラヴァ レイク イェスタデイ

💬 よく使う熟語（マイクラ例文）

build up積み上げる、段階的に構築する

⛏I'm building up my resources before starting the big project.

大きなプロジェクトを始める前に、リソースを積み上げています。

/aɪm ˈbɪldɪŋ ʌp maɪ ˈrɛsɔːrsɪz bɪˈfɔr ˈstɑːrtɪŋ ðə bɪɡ ˈprɑːdʒɛkt/
  • the は弱形 /ðə/
アイム ビルディング アップ マイ レソーシーズ ビフォア スターティング ザ ビッグ プロジェクト
build out広げる、拡張する

⛏Let's build out the farm to double our production.

生産量を2倍にするため、ファームを拡張しましょう。

/lɛts bɪld aʊt ðə fɑːrm tu ˈdʌbəl ˈaʊr prəˈdʌkʃən/
  • build out → 語末子音 /d/ が母音 /aʊ/ に連結、フラップ化 [bɪɾˈaʊt]
  • to は弱形 /tə/ に弱化
  • the は弱形 /ðə/
レッツ ビラウト ザ ファーム タ ダブル アワ プロダクション
build on〜を基にして構築する、〜の上に建築する

⛏We should build on bedrock to make sure the structure is safe.

構造が安全であることを確認するため、基盤岩の上に建築すべきです。

/wi ʃʊd bɪld ɑn ˈbɛdˌrɑk tu meɪk ʃʊr ðə ˈstrʌktʃər ɪz seɪf/
  • build on → 語末子音 /d/ が母音 /ɑ/ に連結、フラップ化 [ˈbɪɾɑn]
  • to は弱形 /tə/ に弱化
  • the は弱形 /ðə/
