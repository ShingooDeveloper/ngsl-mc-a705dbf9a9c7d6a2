task

名詞・動詞/tæsk/CEFR A2頻度#852
  1. 【名詞】困難な仕事、任務（a difficult task）
  2. 【名詞】やるべきこと、課題（the task ahead）
  3. 【名詞】責務、役割（the task of leadership）
  4. 【動詞】（人に）仕事を割り当てる（task someone with something）

⛏️ マインクラフト例文

⛏I need to complete this task before mining ends.

採掘が終わる前にこのタスクを完了する必要があります。

/aɪ ˈniːd tə kəmˈplit ðɪs tæsk bɪˈfoɾ ˈmaɪnɪŋ ˈɛndz/
  • to → 機能語は弱形 /tə/
  • before → 内部フラップ化 /bɪˈfoɾ/
アイ ニード タ コンプリート ディス タスク ビフォーラ マイニング エンズ

⛏Your main task is to gather resources quickly.

あなたの主なタスクは、リソースを素早く集めることです。

/jɔɾ ˈmeɪn tæsk ɪz tə ˈɡæðər ˈɹɛsɔɹsɪz ˈkwɪkli/
  • to → to 弱形 /tə/
ヨーラー メイン タスク イズ タ ギャザー リソースズ クイックリー

⛏Complete this task to unlock new abilities!

このタスクを完了して、新しい能力をアンロック！

/kəmˈplit ðɪs tæsk tə ʌnˈlɑk ˈnu əˈbɪləɾɪz/
  • to → to 弱形 /tə/
コンプリート ディス タスク タ アンロック ニュー アビラリズ

💬 よく使う熟語（マイクラ例文）

take on a taskタスクを引き受ける

⛏We took on the task of building a stronghold.

私たちは要塞を建設するタスクを引き受けました。

/wi ˈtʊk ɑn ə ˈtæsk əv ˈbɪldɪŋ ə ˈstɹɔŋhoʊld/
  • took on → took の末尾子音が on の先頭母音に連結→「トゥカン」
  • of → 弱形 /əv/
ウィ トゥカン ア タスク アヴ ビルディング ア ストロングホールド
set a taskタスクを割り当てる

⛏The admin set a task to rebuild the bridge.

管理者は橋を再建するタスクを割り当てました。

/ðə ˈædmɪn ˈsɛt ə ˈtæsk tu riˌbɪld ðə ˈbrɪdʒ/
  • set a → set /sɛt/ の末尾 t が a /ə/ に連結→「セター」
  • to → to 弱形 /tə/
ザ アドミン セター タスク ター リビルド ザ ブリッジ
task at hand目前のタスク、当面の課題

⛏The task at hand is defeating the boss.

目前のタスクはボスを倒すことです。

/ðə ˈtæsk ət ˈhænd ɪz dɪˈfiːtɪŋ ðə ˈbɑs/
  • task at → task /k/ が at /ə/ に連結→「タスカ」
  • at hand → at の末尾 t が未開放→「アッ」
ザ タスカッ ハンド イズ ディフィーティング ザ バース
