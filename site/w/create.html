create

動詞/kriˈeɪt/CEFR B1頻度#385
  1. 【動詞】〜を作成する、作り出す（…を産出する）
  2. 【動詞】（新しい状況や環境を）生じさせる、作り出す（…を引き起こす）

⛏️ マインクラフト例文

⛏I will create a new world with survival mode.

サバイバルモードで新しいワールドを作成します。

[aɪ wəl kriˈeɪtə njuː wɝld wɪð sərˈvaɪvəl moʊd]
  • will は機能語なので弱形 /wɪl/ → /wəl/
  • create の語末子音 /t/ が a の /ə/ に連結 [kriˈeɪtə]「クリエイター」
アイ ウル クリエイター ニュー ワール ウィズ サーヴァイヴァル モード

  • can は機能語なので弱形 /kæn/ → /kən/
  • create の語末子音 /t/ が a の /ə/ に連結 [kriˈeɪtə]「クリエイター」
  • made の語末子音 /d/ がフラップ化し of の /ə/ に連結 [ˈmeɪɾəv]「メイラヴ」
  • wood の語末子音 /d/ がフラップ化し and の /ə/ に連結 [wʊɾən]「ウラン」
