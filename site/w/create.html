create

動詞/kriˈeɪt/CEFR B1頻度#385
  1. 【動詞】〜を作成する、作り出す（…を産出する）
  2. 【動詞】（新しい状況や環境を）生じさせる、作り出す（…を引き起こす）

⛏️ マインクラフト例文

⛏I will create a new world with survival mode.

サバイバルモードで新しいワールドを作成します。

[aɪ wəl kriˈeɪtə njuː wɝld wɪð sərˈvaɪvəl moʊd]
  • will は機能語なので弱形 /wɪl/ → /wəl/
  • create の語末子音 /t/ が a の /ə/ に連結 [kriˈeɪtə]「クリエイター」
アイ ウル クリエイター ニュー ワール ウィズ サーヴァイヴァル モード

⛏Can you help me create a house made of wood and stone?

木と石で作った家を作成するのを手伝ってくれますか？

[kən juː hɛlp mi kriˈeɪtə haʊs ˈmeɪɾəv wʊɾən stoʊn]
  • can は機能語なので弱形 /kæn/ → /kən/
  • create の語末子音 /t/ が a の /ə/ に連結 [kriˈeɪtə]「クリエイター」
  • made の語末子音 /d/ がフラップ化し of の /ə/ に連結 [ˈmeɪɾəv]「メイラヴ」
  • wood の語末子音 /d/ がフラップ化し and の /ə/ に連結 [wʊɾən]「ウラン」
カン ユー ヘルプ ミー クリエイター ハウス メイラヴ ウラン ストーン

⛏She created an underground base beneath the mountain.

彼女は山の下に地下基地を建設しました。

[ʃi kriˈeɪɾən ʌndərˈɡraʊnd beɪs bɪˈniːθ ðəˈmaʊntən]
  • created の語末子音 /d/ がフラップ化し an の /ə/ に連結 [kriˈeɪɾən]「クリエイラン」
  • the は弱形 /ðə/ で mountain の /m/ に連結 [ðəˈmaʊntən]「ザマウンテン」
シー クリエイラン アンダーグラウンド ベース ビニース ザマウンテン

💬 よく使う熟語（マイクラ例文）

この城は石ブロックだけでゼロから作成しました。
