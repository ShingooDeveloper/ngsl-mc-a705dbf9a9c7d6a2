set

動詞・名詞/sɛt/CEFR A1頻度#184
  1. 【動詞】（物を）置く、配置する（…に置く。コマンドで難易度やスポーン地点を設定）
  2. 【動詞】（装置・罠などを）セットアップする、準備する
  3. 【名詞】一組、セット（道具やブロックの組み合わせ）
  4. 【名詞】グループ、集合（同じ種類の物を集めたもの）

⛏️ マインクラフト例文

⛏I set my spawn point at the base.

私は自分のスポーン地点を基地に設定した。

/aɪ ˈsɛt maɪ ˈspoʊn pɔɪnt æt ðə ˈbeɪs/
  • set my — 語末子音 /t/ が未開放で次の /m/ に接続 [t̚ m]「セッマイ」
  • at the — 語末子音 /t/ が未開放で /ð/ に接続 [t̚ ð]「アッザ」
アイ セッマイ スポウン ポイント アッザ ベイス

⛏We set up traps around the village.

私たちは村の周りに罠をセットアップした。

/wiː sɛt ʌp træps əˈraʊnd ðə ˈvɪlɪdʒ/
  • set up — 語末子音 /t/ が未開放で次の /ʌ/ に接続 [t̚ ʌ]「セッ アップ」
  • around the — 語末子音 /d/ が /ð/ に接続 [d ð]「ァラウンザ」
ウィー セッ アップ トラップス ァラウンザ ヴィリッジ

⛏This is a complete set of iron tools.

これは鉄道具一式です。

/ðɪs ɪz ə kəmˈplit sɛt əv ˈaɪrən tulz/
  • is a — 語末子音 /z/ が母音 /ə/ に連結 [z ə]「ズア」
  • set of — /t/ が母音に挟まれフラップ化 [ɾ] し次の /ə/ に連結「セラヴ」。of は弱形 /əv/
ディス イズア コムプリット セラヴ アイアン トゥールズ

set up拠点やトラップなどを準備・構築する

⛏We need to set up a base before nightfall.

日没前に基地を構築する必要があります。

/wiː niːd tə ˈsɛt ʌp ə ˈbeɪs bɪˈfɔr ˈnaɪtˌfɔl/
  • need to — to が弱形 /tə/ に弱化「ニータ」
  • set up — 語末子音 /t/ が未開放で次の /ʌ/ に接続 [t̚ ʌ]「セッ アップ」
  • before nightfall — /r/ が母音に連結「ビフォア」
