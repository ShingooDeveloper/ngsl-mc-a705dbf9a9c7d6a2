package

名詞・動詞/ˈpækɪdʒ/CEFR A2頻度#1316
  1. 【名詞】郵便や配送の対象となる梱包された荷物（a package arrived at the base）
  2. 【名詞】複数の関連した商品やサービスをセットでまとめたもの（a software package）
  3. 【動詞】商品などを梱包する、箱に詰める、包装する（package items for trade）

⛏️ マインクラフト例文

⛏I packaged the diamonds in a chest for storage.

ダイヤをストレージ用のチェストに梱包した。

/aɪ ˈpækɪdʒd ðə ˈdaɪməndz ɪn ə tʃɛst fɚ ˈstɔrɪdʒ/
  • the は弱形 /ðə/
  • a は弱形 /ə/
アイ パッキジド ザ ダイマンズ イン ア チェスト ファー ストーリッジ

⛏Each package contains five stacks of cobblestone.

各パッケージは5スタックの丸石を含んでいる。

/ˈiːtʃ ˈpækɪdʒ kənˈteɪnz faɪv stæks əv ˈkɑblˌstoʊn/
