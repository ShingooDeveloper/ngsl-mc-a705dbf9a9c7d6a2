function

動詞・名詞/ˈfʌŋkʃən/CEFR B1頻度#701
  1. 【名詞】機能、働き（intended purpose or action）
  2. 【名詞】職務、役割（a person's role or job）
  3. 【動詞】機能する、作用する（to work correctly or operate）
  4. 【動詞】～として機能する、役立つ（to serve as or act in a specified way）

⛏️ マインクラフト例文

⛏The furnace's main function is to smelt ores.

かまどの主な機能は鉱石を精錬することだ。

/ðə ˈfɝːnɪsɪz ˈmeɪn ˈfʌŋkʃənɪz tə ˈsmɛɾɔːrz/
  • function-is: 語末子音 /n/ が is の /ɪ/ に連結 [ˈfʌŋkʃənɪz]
  • smelt-ores: 母音-/t/-母音で米式フラップT [ˈsmɛɾɔːrz]
ザ ファーニシズ メイン ファンクションイズ タ スメロオーズ

  • function-as: 語末子音 /n/ が as の /æ/ に連結 [ˈfʌŋkʃənæz]
