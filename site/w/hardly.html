hardly

副詞/ˈhɑːrdli/CEFR B1頻度#1337
  1. 【副詞】ほぼ～ない、ほぼ～しない（否定的意味を強める）
  2. 【副詞】辛うじて、やっと（ほぼ…だけ、すんでのこと）
  3. 【副詞】～するとすぐに（「Hardly had...when」構文で使用）
  4. 【副詞】厳しく、ぶっきらぼうに（人の言い方や態度に対して、古い用法）

⛏️ マインクラフト例文

⛏I hardly found any diamonds in that cave.

その洞窟でほぼダイヤモンドを見つけられなかった。

/aɪ ˈhɑːrdli ˈfaʊnd ˈɛni ˈdaɪəmɑndz ɪn ðæt keɪv/
  • found の /d/ が any の /ɛ/ に連結 [faʊndɛni] ファウンデニ
  • that の /t/ は未開放化 [ðæt̚] ザッ
アイ ハードリ ファウンデニ ダイヤモンズ イン ザッ ケイヴ

⛏You can hardly survive without a wooden pickaxe.

木のツルハシなしではほぼ生き残れない。

/juː kən ˈhɑːrdli sərˈvaɪv wɪðˈaʊt ə ˈwʊdən ˈpɪkˌæks/
  • can は機能語として弱形 /kən/ に弱化 [kən] カン
  • without の /t/ が a の /ə/ に連結 [wɪðˈaʊtə] ウィザウター
ユー カン ハードリ サーバイヴ ウィザウター ウッデン ピッカックス

⛏Hardly had we finished mining when a Warden appeared.

採掘をやっと終えるとすぐにウォーデンが現れた。

/ˈhɑːrdli həd wiː ˈfɪnɪʃt ˈmaɪnɪŋ wɛn ə ˈwɔːrdən əˈpɪrd/
  • had は機能語として弱形 /həd/ に弱化 [həd] ハッド
  • a は機能語の冠詞として弱形 /ə/ に弱化 [ə] ア
  • when の /n/ が a の /ə/ に連結 [wɛnə] ウェナ
  • Warden の /ən/ が appeared の /ə/ に連結 [ˈwɔːrdənəˈpɪrd] ウォーデナピアド
ハードリ ハッド ウィ フィニシ マイニング ウェナ ウォーデナピアド

💬 よく使う熟語（マイクラ例文）

hardly ever滅多に～ない、ほぼ決して～ない

⛏Creepers hardly ever spawn in my base anymore.

クリーパーはもう私の基地にはほぼスポーンしない。

/ˈkriːpərz ˈhɑːrdli ˈɛvər speɪn ɪn maɪ beɪs ˌɛniˈmoːr/
  • Creepers の /z/ と hardly は子音+子音で分かれる [ˈkriːpərz ˈhɑːrdli] クリーパーズ ハードリ
  • spawn の /n/ が in の /ɪ/ に連結 [speɪnɪn] スペイニン
クリーパーズ ハードリ エバー スペイニン マイ ベイス エニモア
hardly aほぼ…ない、…とは言えない（存在の否定）
