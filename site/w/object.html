object

名詞・動詞/ˈɑbdʒɛkt/CEFR B1頻度#940
  1. 【名詞】見たり触ったりできる物、アイテム（…a thing you can see or touch）
  2. 【名詞】目的、意図（…a purpose or aim）
  3. 【動詞】反対する、異議を唱える（…to express disagreement）

⛏️ マインクラフト例文

⛏I mined an object made of obsidian.

黒曜石でできたアイテムを採掘した。

[aɪ ˈmaɪnɾən ˈɑbdʒɛkt ˈmeɪɾəv ɑbˈsɪdiən]
  • 「an」は機能語で弱形 /ən/
  • 「mined an」— 語末 /d/ が次の母音に連結し、フラップ化 [ˈmaɪnɾən]
  • 「made of」— /d/ がフラップ化し、弱形 /əv/ と連結 [ˈmeɪɾəv]
アイ マインラン オブジェクト メイラヴ アブスィディアン

⛏All players object to this unfair rule.

すべてのプレイヤーがこの不公正なルールに反対する。

[ɔl ˈpleɪɚz əbˈdʒɛk̚tə ðɪs ənˈfɛɚ rul]
  • 「to」は機能語で弱形 /tə/ に弱化
  • 「players object」— 語末 /z/ が母音 /ə/ に連結 [ˈpleɪɚzə-]
  • 「object to」— object の /t/ が未開放になり、to の弱形 /tə/ と連結 [əbˈdʒɛk̚tə]
オール プレイヤーズ ァブジェッタ ディス アンフェア ルール

⛏The object inside the chest is valuable.

チェストの中のアイテムは貴重だ。

[ðə ˈɑbdʒɛɾɪn ˈsaɪd ðə tʃɛst ɪz ˈvæljəbəl]
  • 「the」は機能語で弱形 /ðə/
  • 「object inside」— 語末 /t/ が次の母音に連結し、フラップ化 [ˈɑbdʒɛɾɪn]
  • 「the chest」— 「the」は弱形 /ðə/ のまま
