actually

副詞/ˈæktʃuəli/CEFR A2頻度#290
  1. 【副詞】実は、事実上（意外な現実や事実を述べる）（…である）
  2. 【副詞】本当に、実際に（強調して言う）（…なのだ）
  3. 【副詞】ところで、実のところ（新しい情報や見解を導入する）

⛏️ マインクラフト例文

⛏I actually found diamonds deep underground.

実は地下深くでダイヤを見つけた。

/aɪ ˈæktʃuəli ˈfaʊnd ˈdaɪməndz ˈdiːp ˌʌndɚˈɡraʊnd/
  • deep の語末 /p/ が underground の /ʌ/ に連結して [ˈdiːpʌ] 「ディーパ」に聞こえる
  • underground は un-ˈDER-ground で第2音節に弱ストレス、第3音節に強ストレス、中間の er は r-colored schwa /ɚ/
  • found と diamonds の間は未開放の /d/ で diamonds の /d/ に続く
アイ　アクチュアリ　ファウンド　ダイマンズ　ディープアンダーグラウンド

⛏We can actually make obsidian with water and lava.

実は水と溶岩で黒曜石が作れる。

/wi kən ˈæktʃuəli ˈmeɪk əbˈsɪdiən wɪð ˈwɑɾɚ ənd ˈlɑvə/
  • can は機能語で弱形 /kən/「カン」
  • make の語末 /k/ が obsidian の /ə/ に連結して [ˈmeɪkə]「メイカ…」
  • water の /t/ が米式英語で [ɾ] に flap 化して [ˈwɑɾɚ]「ウォーラー」
  • and は弱形 /ən/ またはシラビック n /n̩/「ン」
ウィ　カン　アクチュアリ　メイカブシディアン　ウィズ　ウォーラー　アン　ラーヴァ

⛏Actually, you need a pickaxe to mine this ore block.

実は、つるはしが必要だよ（このオア鉱石を採掘するには）。

/ˈæktʃuəli juː niːd ə ˈpɪkæks tə ˈmaɪn ðɪs ˈɔɹ blɑk/
  • need の語末 /d/ が a の /ə/ に連結して [niːdə] 「ニーダ」
  • to は機能語で弱形 /tə/「タ」、mine の /m/ に自然に続く
  • ore と block の間は連結がなく、単語境界が明確
アクチュアリ　ユー　ニーダ　ピックアックス　タ　マイン　ディス　オー　ブロック

💬 よく使う熟語（マイクラ例文）

Actually, ...実は…、ところで…

⛏Actually, we should build the farm first.

実は最初に農場を建築すべきだ。

/ˈæktʃuəli wi ʃəd ˈbɪld ðə ˈfɑrm fɚst/
  • Actually が文頭で高いピッチで開始、強ストレス
  • should は助動詞で弱形 /ʃəd/「シュド」に縮約
アクチュアリ　ウィ　シュド　ビルド　ザ　ファーム　ファースト
It's actually ...実は…だ、本当は…である

⛏It's actually faster to use a stone pickaxe.

実は石のつるはしの方が速い。

/ɪts ˈæktʃuəli ˈfæstɚ tə ˈjuːz ə ˈstoʊn ˈpɪkæks/
  • It's の語末 /s/ が actually の /ə/ に連結して [ɪtsə]「イッツァ」
  • to は弱形 /tə/「タ」、use の /j/ に自然に続く
イッツァクチュアリ　ファスター　タ　ユーズ　ア　ストーン　ピックアックス
