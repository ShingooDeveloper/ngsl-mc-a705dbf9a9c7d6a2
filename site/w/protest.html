protest

動詞・名詞/pɹoʊˈtɛst/CEFR B1頻度#1705
  1. 【動詞】抗議する、異議を唱える（…に対して反対の意思を表明する）
  2. 【動詞】（innocence などを）公式に主張する、断定する
  3. 【名詞】抗議、異議、苦情（公式な反対意見の表明）
  4. 【名詞】抗議運動、デモ（集団による反対活動）

⛏️ マインクラフト例文

⛏I protest against this unfair ban!

この不公正な禁止に抗議します！

/aɪ pɹoʊˈtɛst əˈɡɛnst ðɪs ʌnˈfɛɹ bæn/
  • protest の語末子音 /t/ が against の母音 /ə/ に連結 [pɹoʊˈtɛstə]
  • against の語末 /t/ が this の /ð/ に未開放で詰まる
アイ プロテスター ゲンスト ディス アンフェア バン

⛏Players protested the server restart.

プレイヤーたちがサーバー再起動に抗議した。

/ˈpleɪɚz pɹoʊˈtɛstɪd ðə ˈsɜɹvɚ ˈɹiːstɑɹt/
  • the は機能語で弱形 /ðə/、server の /s/ に連結し [ðəˈsɜɹvɚ]
プレイヤーズ プロテスティド ザサーバー リースター ト

⛏I had to make a protest sign.

抗議看板を作らなければならなかった。

/aɪ hæd tə meɪkə ˈpɹoʊtɛst saɪn/
  • to は機能語で弱形 /tə/、make の /m/ に連結
  • make の語末子音 /k/ が a の弱形 /ə/ に連結 [ˈmeɪkə]
アイ ハッド タメイカ プロテスト サイン

/wiː pɹoʊˈtɛst əˈɡɛnst ðə njuː bæn/
