only

副詞・形容詞/ˈoʊnli/CEFR A1頻度#73
  1. 【副詞】ただ、だけ、のみ（only one など）
  2. 【副詞】たったの、ほんの（only three）
  3. 【形容詞】唯一の、ただ一つの（the only one）

⛏️ マインクラフト例文

⛏I have only one diamond left.

ダイヤが1個だけ残っている。

/aɪ hæv ˈoʊnli wʌn ˈdaɪmənd lɛft/
  • have /hæv/ の語末 /v/ が only /ˈoʊnli/ に連結 [hæˈvoʊnli]
アイ ハヴオウンリー ワン ダイマンド レフト

⛏You can only reach the End by defeating the dragon.

エンドには竜を倒してのみ到達できます。

/juː kən ˈoʊnli riːtʃ ðə ˈɛnd baɪ dɪˈfiːtɪŋ ðə ˈdræɡən/
  • can /kən/ は弱形
  • the /ðə/ は弱形（2つ出現）
ユー カン オウンリー リーチ ザ エンド バイ ディフィーティング ザ ドラゴン

⛏Only trees can provide wood in this forest biome.

この森バイオームでは、木だけが木材を提供できます。

/ˈoʊnli triːz kən prəˈvaɪd wʊd ɪn ðɪs ˈfɔrɪst ˈbaɪoʊm/
  • can /kən/ は弱形
  • wood /wʊd/ の /d/ が in /ɪn/ に連結 [wʊˈdɪn]
オウンリー トリーズ カン プロヴァイド ウッディン ジス フォリスト バイオウム

ここではダイヤだけでなくエメラルドも見つかります。

/nɑt ˈoʊnli ˈdaɪmənz bət ˈɔlsoʊ ˈɛmərəldz kən bi ˈfaʊnd hɪr/
