component

名詞・形容詞/kəmˈpoʊnənt/CEFR B1頻度#1484
  1. 【名詞】より大きな物を構成する個々の部分・要素（…a part or element of something larger…）
  2. 【名詞】機械・装置・システムの部品（…a piece of equipment or part…）
  3. 【名詞】混合物や複合体を作っている基本的な構成要素（…an essential element that makes up a whole…）
  4. 【形容詞】全体を構成する、部品としての、構成要素の（…constituent, making up a larger system…）

⛏️ マインクラフト例文

⛏I need every component for this redstone machine.

このレッドストーンマシンの部品をすべて集める必要があります。

/aɪ ˈniːd ˈɛvɹi kəmˈpoʊnənt fɔr ðɪs ˈɹɛdstoʊn məˈʃin/
  • need の語末子音 /d/ が every の母音 /ɛ/ に連結 [ˈniːdɛvɹi] ニーディエヴリ
  • component の語末子音 /t/ が for の子音 /f/ に直続、/t/ は未開放 [kəmˈpoʊnən(t)̚ fɔr] カムポウナン フォア
アイ ニーディエヴリ クムポウナン フォア ディス レッドストウン マシーン

⛏Each component of the farm must work together.

農場のそれぞれの部品が一緒に機能しなければなりません。

/iːtʃ kəmˈpoʊnənt əv ðə fɑrm mʌst wɜrk təˈɡɛðər/
  • component の語末子音 /t/ が of の弱形 /əv/ の母音 /ə/ に連結し、フラップT [ɾ] に変わる [kəmˈpoʊnən ɾəv]
  • of の /v/ が the の弱形 /ðə/ に直続する [əv ðə]
  • work の /k/ が together の /t/ に直続し、/k/ の開放が遅延する
イーチ クムポウナナヴ ザ ファーム マスト ワーク タゥゲザー

⛏The component parts are copper and redstone dust.

その構成部品は銅とレッドストーンダストです。

/ðə kəmˈpoʊnənt pɑrts ɑr ˈkɑpər ənd ˈɹɛdstoʊn dʌst/
  • the の弱形 /ðə/ が component の子音 /k/ に直続する [ðə kəmˈpoʊnənt] ザカムポウナント
  • component の語末子音 /t/ が parts の子音 /p/ に直続し、/t/ の開放が遅延する [ən(t)̚ pɑrts]
  • and の /n/ が dust の子音 /d/ に直続し、/n/ の開放が遅延する [ən(d)̚ dʌst]
ザ クムポウナント パーツ アー カッパー アン レッドストウン ダスト

💬 よく使う熟語（マイクラ例文）

component part構成部分、部品

⛏Each wooden block is a component part of the structure.

各木のブロックは構造の構成部分です。

/iːtʃ ˈwʊdən blɑk ɪz ə kəmˈpoʊnənt pɑrt əv ðə ˈstrʌktʃər/
  • component の語末子音 /t/ が part の子音 /p/ に直続し、/t/ の開放が遅延する [ən(t)̚ pɑrt]
  • part の語末 /t/ が of の弱形 /əv/ の母音に連結し、フラップT [ɾ] に変わる [pɑɾəv]。of の /v/ が the に直続する [əv ðə]
イーチ ウッデン ブロック イズ ア クムポウナント パーラヴ ザ ストラクチャー
key component重要な部品、主要な構成要素

⛏The piston is a key component of any automatic system.

ピストンはあらゆる自動システムの重要な部品です。

/ðə ˈpɪstən ɪz ə ki kəmˈpoʊnənt əv ˈɛni ɔːˈtɑmætɪk ˈsɪstəm/
  • piston の語末 /n/ が is の /ɪ/ に連結する [ən ɪz]
  • key の語末 /i/ が component の /k/ に続き、母音-子音で通常の連結 [ki kəmˈpoʊnənt]
ザ ピストン イズ ア キー クムポウナント アヴ エニ オートマティック システム
