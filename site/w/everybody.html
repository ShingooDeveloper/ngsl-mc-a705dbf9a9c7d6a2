everybody

代名詞/ˈɛv.ɚ.i.ˌbɑ.di/CEFR A1頻度#931
  1. 【代名詞】全ての人，皆，誰もが（every person を総称する最基本用法）
  2. 【代名詞】（グループ・場面における）全員，一同

⛏️ マインクラフト例文

⛏Everybody has to build a shelter before nightfall.

誰もが夜が来る前にシェルターを作らなければならない。

[ˈɛv.ɚ.i.ˌbɑ.di ˈhæz.tə ˈbɪld ə ˈʃɛl.tɚ bɪˈfɔr ˈnaɪt.fɔl]
  • "has" の /z/ が "to" の /tə/（弱形）に連結 [ˈhæztə] ハズタ。to は機能語で弱化
エヴリバディ ハズタ ビルド ア シェルター ビフォア ナイトフォル

⛏Everybody loves exploring the Nether.

誰もがネザーの探索が好きだ。

[ˈɛv.ɚ.i.ˌbɑ.di ˈlʌvz.ɪk.ˈsplɔr.ɪŋ ðə ˈnɛð.ɚ]
  • "loves" の /z/ が "exploring" の /ɪ/ に連結 [ˈlʌvzɪ]
エヴリバディ ラヴズ イクスプロアリング ザ ネザー

⛏Everybody needs at least one diamond.

誰もが少なくともダイヤモンド1個が必要だ。

[ˈɛv.ɚ.i.ˌbɑ.di ˈniːdz.æt ˈlist wʌn ˈdaɪ.mənd]
  • "needs" の /z/ が "at" の /æ/ に連結 [ˈniːdzæt] ニーズ アット。子音+母音で自然に繋がる
エヴリバディ ニーズ アット リスト ワン ダイマンド

💬 よく使う熟語（マイクラ例文）

everybody knows誰もが知っている，常識だ

⛏Everybody knows you can't survive without a pickaxe.

つるはしなしでは生き残れないなんて誰もが知ってる。

[ˈɛv.ɚ.i.ˌbɑ.di ˈnoʊz.ju kænt sɚˈvaɪv wɪðˈaʊt ə ˈpɪk.æks]
  • "knows" の /z/ が "you" の /j/ に連結 [ˈnoʊzju] ノウズ ユー。you の /j/ は glide（子音機能を持つ半母音）
エヴリバディ ノウズ ユー キャント サヴァイヴ ウィザウト ア ピッカックス
