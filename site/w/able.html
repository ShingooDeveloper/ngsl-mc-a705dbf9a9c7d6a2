able

形容詞/ˈeɪbəl/CEFR A2頻度#270
  1. 【形容詞】(～する)ことができる、可能である。be able to～で使用する最頻用法
  2. 有能な、能力がある、有力な
  3. 手腕がある、実力のある

⛏️ マインクラフト例文

⛏You should be able to find diamonds if you dig deep enough.

十分に掘り進めば、ダイヤモンドを見つけることができるはずです。

/juː ˈʃʊd bi ˈeɪbəl tə ˈfaɪnd ˈdaɪmənz ɪf juː ˈdɪɡ ˈdip ɪˈnʌf/
  • to が弱形 /tə/ に弱化
ユー シュド ビー エイバル タ ファイン ダイマンズ イフ ユー ディグ ディープ イナフ

⛏The builder was able to construct a fortress in one day.

そのビルダーは1日で要塞を建設することができました。

/ðə ˈbɪldɚ wəz ˈeɪbəl tə kənˈstrʌkt ə ˈfɔɚtəs ɪn ˈwʌn deɪ/
  • the が弱形 /ðə/ に弱化
  • was が弱形 /wəz/ に弱化
  • to が弱形 /tə/ に弱化
  • construct の /t/ が a の /ə/ に連結
ザ ビルダー ウァズ エイバル タ コンストラクト ア フォーティス イン ワン デイ

⛏I wasn't able to escape from the creeper before it exploded.

クリーパーが爆発する前に逃げることができませんでした。

/aɪ ˈwɑzənt ˈeɪbəl tə ɪˈskeɪp frəm ðə ˈkɹipɚ bɪˈfoɹ ɪt ɪkˈsploʊdɪd/
  • wasn't の /nt/ がほぼ /n/ に近く発音され able に続く
  • the が弱形 /ðə/ に弱化
  • it の語末子音 /t/ が未開放で exploded の母音に流れる [ɪt̚ ɪkˈsploʊdɪd]
アイ ウァズント エイバル タ イスケイプ フラム ザ クリーパー ビフォー イッ エクスプロウディド

🔤 活用形・複数形

more able比較級[mɔɹ ˈeɪbəl]

I'm more able to mine deep in the caves now.

今はより深い洞窟で採掘できるようになった。

[aɪm mɔɹ ˈeɪbəl tə maɪn dip ɪn ðə keɪvz naʊ]
  • more の語末 /r/ が able の /eɪ/ に連結 [mɔɹˈeɪbəl] モアエイバル
  • able の語末 /l/ が to の /ə/ に連結。to は弱形 /tə/ [ˈeɪbəltə] エイバルタ
  • the は弱形 /ðə/ でザと発音 [ðə keɪvz] ザ ケイヴズ
アイム モアエイバルタ マイン ディープ イン ザ ケイヴズ ナウ

She's more able at fighting Endermen than before.

彼女は以前よりもエンダーマンとの戦闘がより上手だ。

[ʃiz mɔɹ ˈeɪbəl æt ˈfaɪtɪŋ ˈɛndɚmɛn ðæn bɪˈfɔɹ]
  • more の語末 /r/ が able の /eɪ/ に連結 [mɔɹˈeɪbəl] モアエイバル
  • able の語末 /l/ が at の /æ/ に連結。liquid /l/ は母音の前で明確に発音される [ˈeɪbəl æt] エイバル アット
  • than の /n/ が before の /b/ に続く。子音+子音なので linking なし。before は第2音節が重音 [ðæn bɪˈfɔɹ] ザン ビフォア
シーズ モアエイバル アット ファイティング エンダーメン ザン ビフォア
