wage

名詞・動詞/weɪdʒ/CEFR B1頻度#1754
  1. 【名詞】給料、賃金（…earn a good wage）
  2. 【名詞】労働に対する金銭的報酬、対価
  3. 【動詞】～を実行する、遂行する（…wage a campaign）
  4. 【動詞】（特に戦闘・争いを）繰り広げる、継続する（…wage war）

⛏️ マインクラフト例文

⛏The miners on this server earn a wage for each diamond they find.

このサーバーの採掘者は見つけたダイアモンドごとに給料をもらいます。

/ðə ˈmaɪnərz ɑn ðɪs ˈsɝvɚ ɝn ə ˈweɪdʒ fɚ ˈitʃ ˈdaɪəmɑnd ðeɪ ˈfaɪnd/
  • earn の語末 /n/ が a の /ə/ に連結して [ɝnə]
  • for の語末 /r/ が each の /i/ に連結して [fəɹˈitʃ]
ザ マイナーズ アン ディス サーバー アーナ ウェイジ フォーイッチ ダイアモンド ザェイ ファイン

⛏We waged a fierce battle against the Wither boss last night.

昨晩、私たちはウィザーボスとの激しい戦闘を繰り広げました。

/wi ˈweɪdʒd ə ˈfɪrs ˈbætəl əˈɡɛnst ðə ˈwɪðɚ bɑs læst naɪt/
  • waged の語末 /d/ が a の /ə/ に連結して [ˈweɪdʒd ə]
ウィ ウェイジド ア フィアス バッタル アゲンス ザ ウィザー バース ラース ナイト

⛏Their wages increased after discovering a large diamond vein.

大きなダイアモンド鉱脈を発見した後、彼らの給料は増えました。

/ðɛr ˈweɪdʒɪz ɪnˈkrist ˈæftɚ dɪˈskʌvɚɪŋ ə ˈlɑrdʒ ˈdaɪəmɑnd veɪn/
  • discovering の語末 /ŋ/ が a の /ə/ に連結して [vɚɪŋ ə]
ザア ウェイジズ インクリース アフター ディスカバーイング ア ラージ ダイアモンド ベイン

💬 よく使う熟語（マイクラ例文）

wage war（～に対して）戦いを繰り広げる、激しく争う

⛏The redstone engineers wage war against lag and server crashes.

レッドストーン技術者たちはラグとサーバークラッシュに対して戦っている。

/ðə ˈrɛdˌstoʊn ˌɛndʒɪˈnɪrz ˈweɪdʒ wɔr əˈɡɛnst læɡ ənd ˈsɝvɚ ˈkræʃɪz/
  • war の語末 /r/ が against の /ə/ に連結して [wɔrə]
  • and の弱形 /ənd/ で /d/ が server の /s/ に続く（連結なし）
