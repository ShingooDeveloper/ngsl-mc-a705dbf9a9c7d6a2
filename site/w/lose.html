lose

動詞/luz/CEFR A1頻度#304
  1. 【動詞】(物を)失う、なくす、置き忘れる（lose sth）
  2. 【動詞】(人を)失う、(人を)見失う（lose sb）
  3. 【動詞】(競技やゲームに)負ける（lose to sb）
  4. 【動詞】(時間・チャンスなど)を無駄にする（lose time）

⛏️ マインクラフト例文

⛏I lost my pickaxe in the cave.

洞窟でピッケルをなくした。

/aɪ ˈlɑst maɪ ˈpɪkæks ɪn ðə ˈkeɪv/
  • "the cave" — the は弱形 /ðə/ で cave に連なる [ðəˈkeɪv] ザケイヴ
アイ ロスト マイ ピッカックス イン ザ ケイヴ

⛏We lost the battle against the Wither.

ウィザーとの戦いに負けた。

/wi ˈlɑst ðə ˈbætəl əˈɡɛnst ðə ˈwɪðər/
  • "lost the" — lost の /t/ が未開放で次の /ð/ に続く [ˈlɑsðə] ロスザ
  • "the Wither" — the は弱形 /ðə/ で Wither に連なる [ðəˈwɪðər] ザウィザー
ウィ ロスザ ザ バティル アゲンスザ ザウィザー

⛏You'll lose all your tools if you fall in lava.

マグマに落ちたらすべてのツールをなくすよ。

/juːl ˈluz ɔl jɔr ˈtulz ɪf ju ˈfɔl ɪn ˈlɑvə/
  • "tools if" — tools の /z/ が if の /ɪ/ に連結 [tulzɪf] トゥルズイフ
  • "fall in" — fall の /l/ が in の /ɪ/ に連結 [fɔlɪn] フォーリン
ユール ルーズ オール ユア トゥルズイフ ユー フォーリン ラーヴァ

💬 よく使う熟語（マイクラ例文）

lose track of…を見失う、追跡を失う

⛏I lost track of the Enderman in the dark.

暗闇でエンダーマンを見失った。

/aɪ ˈlɑst træk əv ðə ˈɛndərmæn ɪn ðə dɑrk/
  • "track of" — track の /k/ が of の /ə/ に連結 [træk əv] トラックアヴ
アイ ロスト トラックアヴ ザ エンダーマン イン ザ ダーク
lose your mind気が狂う、激怒する

⛏I'll lose my mind if another Creeper blows up my base!

もう一体のクリーパーが基地を爆発させたら、気が狂う！

/aɪl ˈluz maɪ ˈmaɪnd ɪf əˈnʌðər ˈkriːpər bloʊz ʌp maɪ ˈbeɪs/
  • "if another" — if の /f/ が another の /ə/ に連結 [ɪfəˈnʌðər] イフアナザー
  • "blows up" — blows の /z/ が up の /ʌ/ に連結 [bloʊzʌp] ブローズアップ
アイル ルーズ マイ マインド イフアナザー クリーパー ブローズアップ マイ ベイス
