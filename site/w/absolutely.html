absolutely

副詞/ˈæbsəluːtli/CEFR B1頻度#1075
  1. 【副詞】完全に、絶対に（～なしでは不可能なこと）
  2. 【副詞】疑いなく、確実に（～することは確かだ）
  3. 【副詞】本当に、その通り（～に同意する・驚きを表す）

⛏️ マインクラフト例文

⛏I absolutely need diamonds for a pickaxe.

つるはしをクラフトするのにダイヤが絶対に必要だ。

/aɪ ˈæbsəluːtli niːd ˈdaɪməndz fɚ ə ˈpɪkæks/
  • need の /d/ が未開放で diamonds の /d/ に直結 [niːd̚ˈdaɪməndz]
  • for の弱形 /fɚ/ の /r/ が a に連結 [fɚə]
アイ アブソルーットリ ニーダイマンズ ファ ピカックス

⛏That redstone door is absolutely brilliant to build!

あのレッドストーンドアは本当に建築が素晴らしい！

/ðæt ˈrɛdˌstoʊn dɔr ɪz ˈæbsəluːtli ˈbrɪljənt tə ˈbɪld/
  • door の /r/ が is の /ɪ/ に連結 [dɔrɪz]
  • brilliant の /t/ が未開放で、to の弱形 /tə/ に続く [ˈbrɪljənt̚ tə]
ザット レッドストーン ドーリズ アブソルーットリ ブリリャント タ ビルド

⛏We can absolutely defeat the Wither with diamond armor.

ダイヤアーマーでウィザーを絶対に倒せる。

/wi kən ˈæbsəluːtli dɪˈfit ðə ˈwɪðər wɪð ˈdaɪmənd ˈɑrmər/
  • can の弱形 /kən/ の /n/ が absolutely の /æ/ に連結 [kənˈæbsəluːtli]
  • diamond の /d/ が armor の /ɑ/ に連結 [ˈdaɪməndˈɑrmər]
