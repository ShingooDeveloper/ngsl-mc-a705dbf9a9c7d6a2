argument

名詞/ˈɑɹɡjəmənt/CEFR B1頻度#877
  1. 【名詞】けんか、言い争い、議論（意見が対立することで生じる）
  2. 【名詞】論拠、根拠、論証（主張を支持する理由や証拠）
  3. 【名詞】論題、主張の内容（議論や会話の対象になるテーマ）

⛏️ マインクラフト例文

⛏We had an argument about which direction to dig for diamonds.

私たちはダイヤモンドを掘る方向についてけんかした。

/wiː hæd ən ˈɑɹɡjəmənt əˈbaʊt ˈwɪtʃ dɪˈɹɛkʃən tə dɪɡ fɔɹ ˈdaɪəmənz/
  • had an - 語末子音 /d/ が母音 /ə/ に連結 [hæd ən]
  • argument about - 語末子音 /t/ が母音 /ə/ に連結
  • to dig - 機能語 to の弱形 /tə/
ウィー ハッダン アーギュメンタバウト ウィッチ ディレクション タ ディグ フォー ダイアモンズ

⛏His argument was that we should build a mob farm for experience.

彼の主張は、経験値を得るためにモブファームを作るべきだというものだった。

/hɪz ˈɑɹɡjəmənt wəz ðæt wiː ʃʊd bɪld ə mɑb fɑɹm fɔɹ ɪkˈspɪɹiəns/
  • argument was - 語末子音 /t/ が半母音 /w/ に連結
  • build a - 語末子音 /d/ が母音 /ə/ に連結 [bɪld ə]
ヒズ アーギュメンワズ ザット ウィー シュッド ビルダ マブ ファーム フォー イクスピアリエンス

⛏I don't accept that argument—mining straight down is too dangerous.

その主張は認めない。真っすぐ掘るのは危険だ。

/aɪ ˈdoʊnt ækˈsɛpt ðæt ˈɑɹɡjəmənt ˈmaɪnɪŋ streɪt daʊn ɪz tu ˈdeɪndʒərəs/
  • don't accept - 語末子音 /t/ が母音 /æ/ に連結 [ˈdoʊnt ækˈsɛpt]
  • straight down - 語末子音 /t/ は未開放（次の /d/ の前で詰まる）
アイ ドウンタクセプト ザット アーギュメント マイニング ストレイッ ダウン イズ トゥー デインジャラス

💬 よく使う熟語（マイクラ例文）

get into an argumentけんかになる、議論になる

⛏We got into an argument about the best way to fight the Ender Dragon.

エンダードラゴンと戦う最良の方法についてけんかになった。

/wiː ɡɑt ˈɪntu ən ˈɑɹɡjəmənt əˈbaʊt ðə bɛst weɪ tə faɪt ðə ˈɛndər ˈdræɡən/
  • got into - 語末子音 /t/ が母音 /ɪ/ に連結
  • into an - 末尾 /u/ が次の母音 /ə/ に向かう流れ
ウィー ガティントゥアン アーギュメンタバウト ザ ベスト ウェイ タ ファイト ザ エンダー ドラゴン
settle an argumentけんかを終わらせる、議論を決着させる

⛏Let's settle this argument with a PvP duel to see who's the better builder.

PvPの決闘で誰がより優れたビルダーか決めてけんかを終わらせよう。

/lɛts ˈsɛtəl ðɪs ˈɑɹɡjəmənt wɪð ə ˈpiːviːpiː ˈduːəl tə ˈsiː ˈhuːz ɪz ðə ˈbɛtər ˈbɪldər/
  • settle this - 語末子音 /l/ が /ð/ に連結
  • to see - 機能語 to の弱形 /tə/
