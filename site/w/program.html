program

動詞・名詞/ˈproʊɡræm/CEFR B1頻度#239
  1. 【名詞】計画・プロジェクト（自動化システムやレッドストーン回路の設計）
  2. 【名詞】予定・スケジュール（今週やシーズンの目標）
  3. 【動詞】プログラミング・設計する（システムやメカニズムを自動化する）

⛏️ マインクラフト例文

⛏I need to program this redstone circuit to open the door automatically.

このレッドストーン回路を自動的にドアを開くようにプログラミングする必要があります。

/aɪ ˈniːd tə ˈproʊɡræm ðɪs ˈɹɛdˌstoʊn ˈsɝː.kɪt tə ˈoʊ.pən ðə ˈdɔɹ ˌɑː.təˈmæ.tɪ.kli/
  • need to: d /d/ が to の弱形 /tə/ に連結 [ˈniːdə]
  • circuit の語末 t は未開放 [ˈsɝː.kɪt̚]
  • to は機能語なので弱形 /tə/
アイ ニーダ タ プロウグラム ディス レッドストウン サーキッ タ オウペン ザ ドール アータマティカリー

⛏My program for this week is to build a mega base.

今週の計画はメガベースを建設することです。

/maɪ ˈproʊɡræm ˈfɔɹ ðɪs ˈwiːk ˈɪz tə ˈbɪld ə ˈmeɪ.ɡə ˈbeɪs/
  • is to: z /z/ が to の弱形 /tə/ に連結 [ɪz tə]
  • build a: d /d/ が a の弱形 /ə/ に連結 [ˈbɪldə]
マイ プロウグラム フォー ディス ウィーク イズタ ビルダ メイガ ベイス
