retail

名詞・動詞/ˈrɛˌteɪl/CEFR B1頻度#2034
  1. 【名詞】小売業、小売販売（一般消費者への直接販売）
  2. 【動詞】小売りで売る、小売りで販売する（商品を客に直接販売する）
  3. 【名詞】小売価格（消費者向けの販売価格）

⛏️ マインクラフト例文

⛏I'm building a retail shop in the village to sell diamonds.

私は村にリテール店を建設してダイヤモンドを売ることにしている。

/aɪm ˈbɪldɪŋ ə ˈrɛˌteɪl ʃɑp ɪn ðə ˈvɪlɪdʒ tə sɛl ˈdaɪməndz/
  • shop の /p/ が in の /ɪ/ に連結して [ʃɑ.pɪn] ショピン
  • sell の /l/ と diamonds の /d/ は consonant-consonant で、sell の /l/ が未開放 [sɛl̚]
  • 機能語 a、the、to は弱形（/ə/、/ðə/、/tə/）に弱化
アイム ビルディング ア レテイル ショピン ザ ビリッジ タ セル ダイモンズ

⛏Players retail their crafted items at the marketplace.

プレイヤーたちは、作成したアイテムをマーケットプレイスで販売している。

/ˈpleɪərz ˈrɛˌteɪl ðɛr ˈkræftɪd ˈaɪtəmz æt ðə ˈmɑrkɪtˌpleɪs/
  • retail の /l/ と their の /ð/ は consonant-consonant で接続なし [ˈrɛˌteɪl ðɛr]
  • crafted の /d/ は unreleased のまま items に続き [ˈkræf.tɪd̚ ˈaɪtəmz] クラフティッ アイテムズ
  • at の /t/ も unreleased で the の /ðə/ に続く [æt̚ ðə] アッザ
プレイアーズ レテイル ザー クラフティッ アイテムズ アッ ザ マーケットプレイス

⛏Many servers retail rare enchanted weapons to new players.

多くのサーバーは、レアなエンチャント武器を新しいプレイヤーに販売している。

/ˈmeɪni ˈsɜrvərz ˈrɛˌteɪl rɛr ɪnˈtʃæntɪd ˈwɛpənz tə nu ˈpleɪərz/
  • rare の末尾 /r/ が enchanted の /ɪ/ に連結して [rɛrɪn]
  • servers の /z/ と retail の /r/ は consonant-consonant で接続なし
  • to は弱形 /tə/ で タと発音、new の /u/ に母音として続く

💬 よく使う熟語（マイクラ例文）

retail for（商品が）〜の値段で売られる、〜で販売される

⛏Enchanted tools retail for high prices here.

エンチャントされたツールはここでは高い値段で販売される。

/ɪnˈtʃæntɪd tulz ˈrɛˌteɪl fɔr haɪ ˈpraɪsɪz hɪr/
  • for の /r/ が high の /aɪ/ に連結して [fɔɹaɪ] フォーハイ
  • high の /aɪ/ と prices の /p/ は vowel-consonant で smooth に移行
  • tools の /z/ が retail の /r/ は consonant-consonant で接近はなし
