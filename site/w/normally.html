normally

副詞/ˈnɔɚməli/CEFR B1頻度#1421
  1. 【副詞】通常は、普通は（…というのが一般的だ）
  2. 【副詞】標準的な方法で、正常に（…のやり方で）

⛏️ マインクラフト例文

⛏I normally gather wood first when I start a new world.

新しいワールドを始める時、私は通常はまず木を集めます。

/aɪ ˈnɔɚməli ˈɡæðɚ wʊd fɝst wɛn aɪ stɑɹt ə ˈnjuː wɝld/
  • start a → start の語末 /t/ が未開放で、次の弱形 a に連結 [stɑɹt̚ə]（スターア）
アイ ノーマリ ギャザー ウッド ファースト ウェン アイ スターア ニュー ワールド

⛏The bow normally shoots arrows in a straight line.

弓は通常、矢をまっすぐに発射します。

/ðə ˈboʊ ˈnɔɚməli ʃuːts ˈɛroʊz ɪn ə ˈstreɪt ˈlaɪn/
  • in a → in の語末 /n/ が弱形 a に連結 [ɪnə]（イナ）
ザ ボウ ノーマリ シューツ エローズ イナ ストレイト ライン

⛏You normally need a shield to protect against creepers.

クリーパーから身を守るために、通常は盾が必要です。

/juː ˈnɔɚməli ˈniːd ə ʃild tə prəˈtɛkt əˈɡɛnst ˈkripɚz/
  • need a → need の語末 /d/ が弱形 a に連結 [niːdə]（ニーダ）
ユー ノーマリ ニーダ シールド タ プロテクト アゲンスト クリーパーズ

💬 よく使う熟語（マイクラ例文）

not normally通常は…しない、普通は…ない

⛏Diamonds don't normally appear on the surface.

ダイヤは通常、地表には現れません。

/ˈdaɪməndz ˈdoʊnt ˈnɔɚməli əˈpɪr ɑn ðə ˈsɝfɪs/
  • don't normally → don't の /t/ が未開放で normally の /n/ に先行 [doʊnt̚ˈnɔɚməli]
  • appear on → appear の語末 /r/ が on の /ɑ/ に連結 [əˈpɪrɑn]（アピアーオン）
ダイマンズ ドント ノーマリ アピアーオン ザ サーフィス
