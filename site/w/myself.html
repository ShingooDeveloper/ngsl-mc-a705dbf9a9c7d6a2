myself

代名詞/maɪˈsɛlf/CEFR A1頻度#668
  1. 【代名詞】（再帰用法）自分自身。動作の対象が話者自身である場合に使う（I built it myself.）
  2. 【代名詞】（強調用法）〜私自身が。主語や述語を強調する際に用いる（I myself don't know.）
  3. 【代名詞】単独で使われて自分自身を指す。by myself で1人で、助けなしに（I can do it by myself.）

⛏️ マインクラフト例文

⛏I built the tower myself without help.

塔を自分で建てた、助けなしに。

/aɪ ˈbɪlt ðə ˈtaʊər maɪˈsɛlf wɪðˈaʊt ˈhɛlp/
  • the は機能語で弱形 /ðə/
アイ ビル（っ）ザ タウアー マイセルフ ウィザウ（っ）ヘルプ

⛏I myself prefer night mining to cave exploring.

私自身は、洞窟探索より夜間採掘を好む。

/aɪ maɪˈsɛlf prɪˈfɝ naɪt ˈmaɪnɪŋ tə keɪv ɪkˈsplɔrɪŋ/
  • cave [keɪv] と exploring [ɪkˈsplɔrɪŋ] の間で、cave の /v/ が exploring の /ɪ/ に繋がる
  • to は機能語で弱形 /tə/

⛏By myself, I can survive in the desert for days.

1人では、砂漠で数日生き残ることができる。

/baɪ maɪˈsɛlf aɪ kən sərˈvaɪv ɪn ðə ˈdɛzərt fɚ deɪz/
  • can は助動詞で弱形 /kən/
  • the は機能語で弱形 /ðə/
  • for は弱形 /fɚ/ で r 色彩シュワ
