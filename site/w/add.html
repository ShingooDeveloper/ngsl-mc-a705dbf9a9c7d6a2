add

動詞/æd/CEFR A1頻度#413
  1. 【動詞】（数字や数量を）足す・加える、合計する（perform addition）
  2. 【動詞】（別のもの・情報・機能を）加える・追加する、含める（combine or include）
  3. 【動詞】「さらに言う」「付け加える」、追記する（state or remark additionally）

⛏️ マインクラフト例文

⛏I'll add stone blocks to repair the house.

家を直すために石ブロックを加えます。

/aɪl ˈæd ˈstoʊn ˈblɑks tə rɪˈpɛr ðə ˈhaʊs/
  • to は弱形 /tə/
アイル・エド・ストーン・ブロックス・ター・リペア・ザ・ハウス

⛏You need to add vines to make it look better.

見た目を良くするために蔓を加える必要があります。

/ju ˈniːd tə ˈæd ˈvaɪnz tə ˈmeɪk ɪt ˈlʊk ˈbɛtər/
  • to は弱形 /tə/（2回出現）
  • better: /t/ がフラップ化 [ˈbɛɾər] ベラー
ユー・ニード・ター・エド・ヴァインズ・ター・メイク・イット・ルック・ベラー

⛏Add redstone repeaters to speed up the signal.

信号を速くするためにレッドストーンリピーターを追加してください。

/ˈæd ˈrɛdˌstoʊn rɪˈpiːtərz tə ˈspiːd ʌp ðə ˈsɪɡnəl/
  • to は弱形 /tə/
  • speed up: 語末子音 /d/ が母音 /ʌ/ に連結 [spiːdʌp] スピーダップ
エド・レッドストーン・リピーターズ・ター・スピーダップ・ザ・シグナル

💬 よく使う熟語（マイクラ例文）

add up合計する、つじつまが合う

⛏These resources don't add up to enough for the final build.

これらのリソースは最終的な建築に必要な量に足りない。

/ðiz ˈrɛsɔrsɪz ˈdoʊnt ˈæd ʌp tə ɪˈnʌf fər ðə ˈfaɪnəl ˈbɪld/
  • add up: 語末子音 /d/ が母音 /ʌ/ に連結 [ˈæ.dʌp] エダップ
  • to は弱形 /tə/
ディーズ・レソーシーズ・ドーント・エダップ・ター・イナフ・ファー・ザ・ファイナル・ビルド
add to増やす、増加させる、～を高める

⛏Dark oak trees add to the spooky atmosphere of the haunted mansion.

ダークオークの樹はお化け屋敷の不気味な雰囲気を高めます。

/ˈdɑrk ˈoʊk ˈtriz ˈæd tə ðə ˈspuki ˈætməsˌfɪr əv ðə ˈhɔnɾɪd məˈnʃən/
  • add to: 弱形 /tə/ [ˈæd tə] エダ

⛏Don't forget to add in the lighting system when designing the base.

基地を設計するときに照明システムを忘れずに含めてください。

/ˈdoʊnt ˈfɔrɡɛt tə ˈæd ɪn ðə ˈlaɪtɪŋ ˈsɪstəm wɛn dɪˈzaɪnɪŋ ðə ˈbeɪs/
  • forget to: 語末 /t/ が未開放で弱形 /tə/ に連接 [ˈfɔrɡɛt̚ tə] フォーゲッター
  • add in: 語末子音 /d/ が母音 /ɪ/ に連結 [ˈæ.dɪn] エディン
ドーント・フォーゲッター・エディン・ザ・ライティング・システム・ウェン・ディザイニング・ザ・ベース
