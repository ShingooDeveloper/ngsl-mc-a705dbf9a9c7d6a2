flat

形容詞・名詞/flæt/CEFR A1頻度#1074
  1. 【形容詞】平らな、平坦な（…terrain is flat）
  2. 【形容詞】退屈な、つまらない（…the mining feels flat）
  3. 【形容詞】（声・音が）フラットな、低い
  4. 【名詞】平らな面、平面
  5. 【名詞】（イギリス英語）アパート

⛏️ マインクラフト例文

⛏I built a flat base to prevent mobs from spawning.

私はモブのスポーンを防ぐために平らなベースを建築した。

/aɪ bɪlɾə flæt beɪs tə prɪˈvɛnt mobz frəm ˈspɑnɪŋ/
  • built a → built の末尾子音 /t/ が a の /ə/ に連結してフラップT [bɪlɾə]
  • to は弱形 /tə/
  • from は弱形 /frəm/
アイ ビルラア フラット ベイス タ プリヴェント モブズ フラム スポーニング

/flæt ˈtɛreɪn ɪz ˈizər tə ɪkˈsplɔr ən ˈnævɪɡeɪt/
  • terrain is → terrain の末尾子音 /n/ が is の母音 /ɪ/ に連結
  • to は弱形 /tə/
  • and は弱形 /ən/
