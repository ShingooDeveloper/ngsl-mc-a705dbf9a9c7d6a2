option

名詞/ˈɑpʃən/CEFR B1頻度#858
  1. 【名詞】選択肢（選択可能な複数の可能性）（…you have several options…）
  2. 【名詞】選択権、決定権（…the option to build something…）
  3. 【名詞】（ゲーム・システムの）設定項目、オプション機能（…adjust graphics options…）

⛏You can set multiple options for defending against mobs.
