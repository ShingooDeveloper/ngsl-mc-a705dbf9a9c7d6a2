project

名詞・動詞/ˈprɑdʒɛkt/CEFR B1頻度#517
  1. 【名詞】計画、企画。建築や基盤開発に関する構想（building project など）。
  2. 【動詞】計画する、企画する。プロジェクトを立案する。
  3. 【動詞】投げる、発射する。矢など物体を飛ばす（project arrows など）。
  4. 【動詞】予測する、計算する。結果や方向を推測する。

⛏️ マインクラフト例文

⛏I have a huge mining project.

私は大規模な採掘プロジェクトを持っている。

[aɪ hæv ə ˈhjudʒ ˈmaɪnɪŋ ˈprɑdʒɛkt]
  • have a：v が a の母音に連結 [hæ və]
