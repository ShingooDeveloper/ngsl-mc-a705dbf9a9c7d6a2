hate

動詞・名詞/heɪt/CEFR A2頻度#1377
  1. 【動詞】（人・物・事柄を）強く嫌う、憎む。嫌悪感を持つ。
  2. 【動詞】（…することを）嫌がる、つらい。（…するのが不本意だ）
  3. 【名詞】強い嫌悪感、憎しみ、敵意。

⛏️ マインクラフト例文

⛏I hate losing diamonds in the nether.

ネザーでダイヤモンドを失うのが嫌だ

/aɪ ˈheɪt̚ ˈluːzɪŋ ˈdaɪməndz ɪn ðə ˈnɛðɚ/
  • hate の t が losing の l の前で未開放になる [aɪ ˈheɪt̚ ˈluːzɪŋ]「アイ ヘイッ ルーズィング」
  • the は弱形 /ðə/ [ɪn ðə]「イン ザ」
アイ ヘイッ ルーズィング ダイマンズ イン ザ ネザー

⛏I hate this mob spawner.

このモブスポーナーが嫌だ

/aɪ ˈheɪt̚ ðɪs ˈmoʊb ˈspɑnɚ/
  • hate の t が this の th の前で未開放になる [aɪ ˈheɪt̚ ðɪs]「アイ ヘイッ ディス」
アイ ヘイッ ディス モーブ スパナー

⛏I hate mobs.

モブが嫌だ

/aɪ ˈheɪt̚ ˈmoʊbz/
  • hate の t が mobs の m の前で未開放になる
アイ ヘイッ モーブズ

💬 よく使う熟語（マイクラ例文）

hate it when（…というのが）嫌だ。（…することが本当に嫌だ）

⛏I hate it when my pickaxe breaks.

つるはしが壊れるのが本当に嫌だ

/aɪ ˈheɪt̚ ɪt̚ wɛn maɪ ˈpɪkˌæks ˈbreɪks/
  • hate の t が it の i の前で未開放 [aɪ ˈheɪt̚ ɪt̚]「アイ ヘイッ イッ」
  • it の t が when の w の前で未開放 [ɪt̚ wɛn]「イッ ウェン」
アイ ヘイッ イッ ウェン マイ ピッカックス ブレイクス
hate to say（不適切な情報を伝える前置き）…なことを言うのは嫌だが。

⛏I hate to say it, but you died.

悪いけど、君は死んだ

/aɪ ˈheɪt̚ tə ˈseɪ ɪt̚, bət ju ˈdaɪd/
  • hate の t が to の t の前で未開放になり、to は弱形 /tə/ [aɪ ˈheɪt̚ tə]「アイ ヘイッ タ」
  • but の t が you の y の前で未開放 [bət ju]「バッ ユー」
アイ ヘイッ タ セイ イット、バッ ユー ダイド
