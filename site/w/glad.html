glad

形容詞/ɡlæd/CEFR A2頻度#1367
  1. 【形容詞】喜びに満ちた、嬉しい（…が嬉しい、…に満足している）
  2. 【形容詞】（古い用法）明るい、光り輝く

⛏️ マインクラフト例文

⛏I'm so glad I found these diamonds while mining.

採掘中にダイヤモンドを見つけたので本当に嬉しい。

[aɪm soʊ ˈɡlæd aɪ ˈfaʊnd̚ ðiːz ˈdaɪməndz waɪl ˈmaɪnɪŋ]
  • glad の /d/ が I の /a/ に連結して [ˈɡlæ daɪ]
  • found の d が未開放で these に続く
アイム ソウ グラッダイ ファウン ディーズ ダイマンズ ワイル マイニング

⛏They were glad to see the new chest full of supplies.

彼らは物資がいっぱい詰まった新しいチェストを見て嬉しかった。

[ðeɪ wɚ ˈɡlæd tə siː ðə ˈnjuː tʃɛst ˈfʊləv səˈplaɪz]
  • to は弱形 /tə/、the は弱形 /ðə/、of は弱形 /əv/
  • full の /l/ が of の /ə/ に連結
ゼイ ウェ グラッ ター シー ザ ニュー チェスト フルアヴ サプライズ

⛏I'd be glad to help you build this structure if you want.

もし望むなら、この構造物を建設するのを手伝うのは喜んでします。

[aɪd̚ bi ˈɡlæd tə ˈhɛlpju ˈbɪld̚ ðɪs ˈstrʌktʃɚ ɪf ju wɑnt]
  • to は弱形 /tə/ で help の h に続く
  • build の d が未開放で this に続く
アイ ビー グラッ ター ヘルプユー ビル ディス ストラクチャー イフ ユー ワント
