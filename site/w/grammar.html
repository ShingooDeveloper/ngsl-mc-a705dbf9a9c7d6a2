grammar

名詞/ˈɡræmɚ/CEFR B1頻度#2683
  1. 【名詞】言語の規則や構造。文や文章を正しく作るための法則（the system of rules and structures for using a language correctly）
  2. 【名詞】言語体系や言葉の使い方。文法的な知識（the structure or principles of a language）
  3. 【名詞】(比喩的に) あるシステムの法則や基本原則。ゲームやプログラミングの規則（the basic rules or structure of a system）
  4. 【名詞】文法を扱う書籍や教科書（a book that teaches the rules of grammar）

⛏You should study the grammar before using redstone commands.

レッドストーンコマンドを使う前に、文法を勉強したほうがいいです。

  • the は冠詞で弱形 /ðə/ に弱化。grammar の初頭の子音 /ɡ/ に接続。
  • before の末尾 /r/ から using の初頭へ自然に流れる。
