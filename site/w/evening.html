evening

名詞/ˈiːvnɪŋ/CEFR A2頻度#678
  1. 【名詞】夕方から夜中までの時間帯（…昼間の反対）
  2. 【名詞】夜のイベント・集い（…night out や gathering）

⛏️ マインクラフト例文

⛏It takes all evening to mine these diamonds.

このダイヤを掘るのに一晩中かかる

/ɪt̚ ˈteɪks ɑːl ˈiːvnɪŋ tə ˈmaɪn ðiːz ˈdaɪmənds/
  • takes の /s/ が all の /ɑ/ に連結
  • to は弱形 /tə/
イッ テイクサール イーブニング タ マイン ディーズ ダイマンズ

⛏We work on the house in the evening.

私たちは夜間に家で作業をする

/wiː ˈwɝk ɑn ðə ˈhaʊs ɪn ðə ˈiːvnɪŋ/
  • work の /k/ が on の /ɑ/ に連結
  • house の /s/ が in の /ɪ/ に連結
  • the は弱形 /ðə/
ウィ ワーカン ザ ハウシン ザ イーブニング

⛏Mobs spawn in the evening, so we need to prepare.

モブは夜間にスポーンするから準備が必要だ

/ˈmoʊbz spɔn ɪn ðə ˈiːvnɪŋ soʊ wiː ˈniːd tə prɪˈpɛr/
  • the は弱形 /ðə/
  • need の /d/ が to の /ə/ に連結
モウズ スポーン イン ザ イーブニング ソー ウィ ニーダ プリペア
