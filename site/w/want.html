want

動詞・名詞/wɑnt/CEFR A1頻度#75
  1. 【動詞】〜が欲しい、〜したい（want to do...）
  2. 【動詞】求める、〜を必要とする（～を探す）
  3. 【名詞】欲望、願望
  4. 【名詞】不足、欠乏

⛏️ マインクラフト例文

⛏I want to build a Nether portal.

ネザーゲートを作りたいです。

/aɪ ˈwɑnt tə ˈbɪld ə ˈneðɚ ˈpoɹtəl/
  • to: 機能語として弱形 /tə/
  • build → a: build の語末 /d/ が a の /ə/ に連結
アイ ワントタ ビルダ ネザー ポータル

⛏She wants diamond pickaxes for mining.

彼女は採掘用のダイヤのつるはしが欲しいです。

/ʃi ˈwɑnz ˈdaɪmənd ˈpɪkæksɪz fɚ ˈmaɪnɪŋ/
  • wants -s: 濁音化して /z/ [ˈwɑnz]
  • for: 機能語として弱形 /fə/
シー ワンズ ダイモンド ピックアクシズ ファ マイニング
