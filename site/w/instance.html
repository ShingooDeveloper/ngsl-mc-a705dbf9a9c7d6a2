instance

名詞・動詞/ˈɪnstəns/CEFR B1頻度#862
  1. 【名詞】例、事例、具体的な場合（…の場合として）
  2. 【名詞】（コンピュータ）インスタンス、実行中のサーバーやプログラムの個別実行体
  3. 【動詞】例として挙げる、例示する（…の一例を示す）

⛏️ マインクラフト例文

⛏In that instance, I gave up mining.

その場合、私は採掘を諦めました。

/ɪn ðæt ˈɪnstəns, aɪ ˈɡeɪv ʌp ˈmaɪnɪŋ/
  • that instance → t が未開放で i へ「ザッ インスタンス」
  • gave up → v が ʌ に連結「ゲイヴァップ」
イン ザッ インスタンス、アイ ゲイヴァップ マイニング

⛏This server instance crashed yesterday.

このサーバーインスタンスは昨日クラッシュしました。

/ðɪz ˈsɜrvər ˈɪnstəns ˈkræʃt ˈjɛstərdeɪ/
  • This instance → s が i に連結し z に声帯化「ディズインスタンス」
  • crashed yesterday → t が未開放で j へ「クラッシュ イェスタデイ」
ディズインスタンス クラッシュ イェスタデイ

⛏I'll give you an instance of how mobs spawn.

モブがどのようにスポーンするかの一例を教えます。

/aɪl ɡɪv juː ən ˈɪnstəns əv ˌhaʊ moʊbz sˈpoʊn/
  • give you → v が j に連結「ギブユー」
  • an instance → n が i に連結「アニンスタンス」
  • of mobs → v が m に連結「ラムモブズ」
  • mobs spawn → z が s に連結「モブススポーン」
アイル ギブユー アニンスタンス ラヴ ハウ モブズ スポーン

💬 よく使う熟語（マイクラ例文）

for instance例えば、（特に…の場合として）

⛏Finding diamonds is difficult; for instance, I found them only at Y-level 11.

ダイヤを見つけることは難しいです。例えば、私はY レベル11でしか見つけられませんでした。

/ˈfaɪndɪŋ ˈdaɪməndz ɪz ˈdɪfɪkəlt fɔr ˈɪnstəns aɪ ˈfaʊnd ðɛm ˈoʊnli æt waɪ ˈlɛvəl ɪˈlɛvən/
  • is difficult → z が d に連結し未開放「イッディ」
  • For instance → r が i に連結「フォーインスタンス」
  • at Y-level → t が w に連結「アッワイ」
ファインディング ダイマンズ イッディフィカルト フォーインスタンス アイ ファウンド ゼム オンリー アッワイ レベル イレヴン
an instance of…の一例、…の一つの場合

⛏That's an instance of how redstone can be used creatively.

それはレッドストーンをどのように創造的に使えるかの一例です。

/ðæts ən ˈɪnstəns əv ˌhaʊ ˈrɛdˌstoʊn kæn bi ˈjuzd kriˈeɪtɪvli/
  • That's an → s が ə に連結「ザッスアン」
  • of how → v が h に連結「ラハウ」
  • creatively → v が l に連結「クリエイティヴリー」
ザッスアン インスタンス ラハウ レッドストーン キャン ビー ユーズド クリエイティヴリー
