wish

動詞・名詞/wɪʃ/CEFR A2頻度#523
  1. 【動詞】～を望む、～があったらいいのにと思う（欲しいと願う）
  2. 【動詞】（if句と共に）～だったらいいのに、～ならいいのに（～が起こることを希望する）
  3. 【名詞】願い、希望（何かが起こることへの欲求）

⛏️ マインクラフト例文

⛏I wish I could take it.

それを持ち運びたいんだ。

/aɪ ˈwɪʃ aɪ ˈkʊd ˈteɪkɪt/
  • take it → take の語末子音 /k/ が it の /ɪ/ に連結し [ˈteɪkɪt] テイキット
  • could の語末 /d/ は未開放で詰まる音
アイ ウィシュ アイ クド テイキット

⛏I wish to find ore here.

ここで鉱石を見つけたい。

/aɪ ˈwɪʃ tə ˈfaɪndɔɹ hɪɹ/
  • to → 機能語として弱形 /tə/ に弱化
  • find ore → find の語末子音 /d/ が ore の /ɔ/ に連結し [ˈfaɪndɔɹ] ファインドア
アイ ウィシュ タ ファインドア ヒア

⛏His wish is to get a netherite tool.

彼の願いはネザライトツールを手に入れることだ。

/hɪz ˈwɪʃ ɪz tə ˈɡɛtə ˈnɛðəɹaɪt ˈtuːl/
  • to → 機能語として弱形 /tə/ に弱化
  • get a → get の語末子音 /t/ が a の弱形 /ə/ に連結し [ˈɡɛtə] ゲタ
ヒズ ウィシュ イズ タ ゲタ ネザライト トゥール

💬 よく使う熟語（マイクラ例文）

wish for～を熱望する、～を強く望む

⛏I wish for an enchanted pickaxe above all else.

何よりも魔法のピッケルが欲しい。

/aɪ ˈwɪʃ fɔɹ ən ɛnˈtʃɑntəd ˈpɪkˌæks əˈbʌvɔl ˈɛls/
  • wish for → wish の /ʃ/ と for の /f/ は子音+子音で連結なし。for が弱形に変わる
  • for an → for の語末 /r/ が an に連結
  • above all → above の語末 /v/ が all の /ɔ/ に連結し [əˈbʌvɔl]
アイ ウィシュ フォーラン エンチャンテッド ピックアックス アバヴォル エリス
as you wishかしこまりました、お好きなように

⛏Build another farm? As you wish!

別のファームを作る？かしこまりました！

/ˈbɪldəˈnʌðɚ ˈfɑɹm æz jə ˈwɪʃ/
  • build another → build の語末子音 /d/ が another の /ə/ に連結し [ˈbɪldəˈnʌðɚ]
  • as you → as の /z/ と you の /j/ は子音+子音で連結なし
ビルド アナザー ファーム アズ ユ ウィッシュ
wish list欲しいもののリスト

⛏My wish list includes an elytra and a beacon.

私の欲しいものリストにはエリトラとビーコンが含まれています。

/maɪ ˈwɪʃ ˌlɪst ɪnˈkludz ən əˈlɪtrə ənd ə ˈbikən/
  • wish list → wish の /ʃ/ と list の /l/ は子音+子音で連結なし
  • list の語末 /t/ は未開放で詰まる音。includes と明確に区切られる
マイ ウィシュ リスト インクルーズ アン アリトラ アンド ア ビーコン
