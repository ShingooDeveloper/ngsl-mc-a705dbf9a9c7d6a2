issue

名詞・動詞/ˈɪʃuː/CEFR B1頻度#217
  1. 【名詞】問題・懸案事項（解決が必要な事態、取り組むべき課題）
  2. 【名詞】発行版・号（雑誌や出版物の一冊分）
  3. 【動詞】正式に配付・交付する（証書や指令を出す）
  4. 【動詞】（法律・医学的に）必然的に生じる、（から）発生する

⛏️ マインクラフト例文

⛏We have an issue with the redstone not working.

レッドストーンが動作しないという問題があります。

/wiː əv ən ˈɪʃuː wɪð ðə ˈrɛdˌstoʊn nɑt̚ ˈwɜrkɪŋ/
  • have が unstressed で弱形 /əv/ に。have an は [əv ən]「アヴ アン」
  • an の /n/ が issue の /ɪ/ に連結 [ən ˈɪ]「エニ」
  • not の /t/ は未開放で [t̚]「ナッ」、working へ続く
ウィー アヴ アン イシュー ウィズ ザ レッドストゥン ナッ ワーキング

⛏The admin will issue new tools to every player.

管理人は全プレイヤーに新しい道具を配付します。

/ðə ˈædmɪn wɪl ˈɪʃuː nu tulz tə ˈɛvri ˈpleɪər/
  • the が unstressed で弱形 /ðə/。the admin は [ðə ˈædmɪn]「ザ アドミン」
  • will が unstressed で [wɪl] のまま。issue へ [wɪl ˈɪ]「ウィル イ」
  • to が unstressed で弱形 /tə/。to every は [tə ˈɛv]「タ エ」
ザ アドミン ウィル イシュー ニュー トゥールズ タ エヴリ プレイヤー

/ðə ˈmeɪn ˈɪʃuː ɪz ðæt̚ ˈmoʊbz spɔn tuː kloʊs tə ˈbeɪs/
