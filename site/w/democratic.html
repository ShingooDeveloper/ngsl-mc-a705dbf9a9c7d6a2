democratic

形容詞/ˌdɛməˈkrætɪk/CEFR B2頻度#2543
  1. 【形容詞】民主的な、民主制度に基づく（全員が意思決定に平等に参加する…）
  2. 【形容詞】公正で平等な、独裁的でない（すべてのメンバーが意見を言う機会を持つ…）
  3. 【形容詞】集団参加型の、協力的な（指導者が一方的に決定するのではなく、みんなで決める…）

⛏️ マインクラフト例文

⛏We held a democratic vote before changing the server rules.

サーバーのルールを変更する前に民主的な投票を行いました。

/wi hɛld ə ˌdɛməˈkrætɪk voʊt bɪˈfɔr ˈtʃeɪndʒɪŋ ðə ˈsɜrvər ˈrulz/

  • vote before → vote の /t/ が未開放で before に連接。[voʊt̚bɪˈfɔr] ヴォッビフォア
