file

名詞・動詞/faɪl/CEFR A2頻度#833
  1. 【名詞】（コンピュータやサーバーの）ファイル、保存されたデータや情報（…に属するドキュメント）
  2. 【名詞】一行、一列（…が一行をなす人や物）
  3. 【動詞】（情報や書類を）ファイルに保存する、記録する、提出する（…をファイルに入れる）
  4. 【動詞】一列に並んで移動する、行進する（…が行列をなして進む）

⛏️ マインクラフト例文

⛏Players file into the mine together.

プレイヤーたちは鉱山に一列になって進む。

/ˈpleɪɚz ˈfaɪl ˈɪntu ðə ˈmaɪn təˈɡɛðɚ/
  • file into: 語末の子音 /l/ が母音 /ɪ/ に連結 [ˈfaɪlɪntu]
  • the: 弱形 /ðə/
プレイヤーズ ファイリント ザ マイン トゥゲザー

⛏We should file our materials in the chest.

我々は材料をチェストに保管整理するべきだ。

/wiː ʃʊd ˈfaɪl aʊɚ məˈtɪriəlz ɪn ðə ˈtʃɛst/
  • file our: 語末の子音 /l/ が母音 /aʊ/ に連結 [ˈfaɪlaʊɚ]
  • the: 弱形 /ðə/
ウィ シュッド ファイラウアー マテリアルズ イン ザ チェスト

⛏I filed away my collection of ores.

鉱石の収集品を大切に保管しておいた。

/aɪ ˈfaɪld əˈweɪ maɪ kəˈlɛkʃən əv ˈɔrz/
  • filed away: 語末の子音 /d/ が母音に挟まれてフラップ化 [ˈfaɪɾəˈweɪ]
  • of: 機能語の弱形 /əv/

💬 よく使う熟語（マイクラ例文）

file away（大切なものや情報を）保管する、きちんと整理して保存する

⛏Let's file away these blueprints for the project.

プロジェクト用のこれらの設計図をきちんと保管しておこう。

/lɛts ˈfaɪl əˈweɪ ðiz ˈbluːprɪnts fɚ ðə ˈprɑdʒɛkt/

  • for the: for の弱形 /fɚ/、the の弱形 /ðə/
