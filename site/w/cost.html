cost

名詞・動詞/kɔst/CEFR A1頻度#251
  1. 【名詞】価格、費用（何かを手に入れるか達成するのに必要な金銭やアイテム）
  2. 【名詞】代償、代価（何かを成し遂げるために失われるもの、払う必要がある犠牲）
  3. 【動詞】かかる（時間、金銭、労力が必要とされる状態にある。〜するのに〜が必要とされる）

⛏️ マインクラフト例文

⛏Enchanting an item costs a lot of resources.

アイテムに魔法をかけるには膨大なリソースが必要です。

/ɪnˈtʃæntɪŋ ən ˈaɪtəm ˈkɔsts ə ˈlɑɾəv rɪˈzɔɹsəz/
  • 「costs a」→ costs の末尾 /s/ が a の /ə/ に接続→「コスツア」
  • 「lot of」→ lot の /t/ がフラップ化 [ɾ]、of の弱形 /əv/ に続く→「ロータヴ」
エンチャンティング アン アイタム コスツ ア ロータヴ リソーシズ

⛏Building a castle requires materials that cost a lot.

城を建設するには多くのマテリアルが必要です。

/ˈbɪldɪŋ ə ˈkæsəl rɪˈkwaɪɹz məˈtɪɹiəlz ðæt kɔst ə ˈlɑɾət/
  • 「a castle」→ a と castle が接続→「アキャスル」
  • 「cost a lot」→ cost の /t/ が a の /ə/ に接続、lot の /t/ がフラップ化→「コスタ ロータット」
ビルディング ア キャスル リクワイアズ マテリアルズ ザット コスト ア ロータット
