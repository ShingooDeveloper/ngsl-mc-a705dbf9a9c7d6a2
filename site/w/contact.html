contact

動詞・名詞/ˈkɑn.tæk̚t/ (n.), /kənˈtæk̚t/ (v.)CEFR B1頻度#599
  1. 【名詞】接触、物が接触する状態（…two blocks in contact…）
  2. 【名詞】連絡先、連絡を取る相手（…save a player's contact…）
  3. 【動詞】接触する（…contact the ground…）
  4. 【動詞】連絡する、連絡を取る（…contact your team…）

⛏️ マインクラフト例文

⛏Contact the others if you see any mobs.

敵モブを見かけたら他のプレイヤーに連絡してください。

/kənˈtæk̚t ðə ˈʌðɚz ɪf juː siː ˈɛni mɑbz/
  • the → 弱形 /ðə/ に弱化
  • others if → others の語末 /z/ が if の /ɪ/（母音）に連結 [ˈʌðɚ zɪf]
コンタクト・ザ・アザーズ・イフ・ユー・シー・エニー・モブズ

⛏I'll contact my team before we start mining.

採掘を始める前にチームに連絡します。

/aɪl kənˈtæk̚t maɪ ˈtim bɪˈfɔɹ wi ˈstɑɹt̚ ˈmaɪ.nɪŋ/
  • contact my → contact の語末 /t/ が子音 /m/ に先行して未開放 [kənˈtæk̚ maɪ]
  • start mining → start の語末 /t/ が子音 /m/ に先行して未開放 [ˈstɑɹt̚ ˈmaɪ.nɪŋ]
アイル・コンタクト・マイ・ティーム・ビフォー・ウィ・スタート・マイニング

⛏Make contact with the redstone circuit.

レッドストーン回路と接触を作ってください。

/meɪk ˈkɑn.tæk̚t wɪð ðə ˈrɛd.stoʊn ˈsɜr.kɪt̚/
  • contact with → contact の語末 /t/ が子音 /w/ に先行して未開放 [ˈkɑn.tæk̚ wɪð]
  • the → 弱形 /ðə/ に弱化
  • circuit → circuit の語末 /t/ が語境界で未開放
メイク・カンタクト・ウィズ・ザ・レッドストーン・サーキット
