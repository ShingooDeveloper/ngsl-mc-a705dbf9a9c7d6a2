move

動詞・名詞/muːv/CEFR A1頻度#199
  1. 【動詞】（物・人を）動かす、移動させる（...から...へ）
  2. 【動詞】（自分が）動く、移動する、（拠点を）移す
  3. 【動詞】（感情を）動かす、心を打つ（...に）
  4. 【動詞】提案する、動議を提出する
  5. 【名詞】動き、移動、行動

⛏️ マインクラフト例文

⛏I need to move this block to build my house.

自分の家を建てるために、このブロックを移動させる必要がある。

[aɪ niːd̚tə muːvðɪs blɑk̚tə bɪldmaɪ haʊs]
  • need の /d/ が to の /t/ に未開放で接続、to は弱形 /tə/
  • move の /v/ が this の /ð/ に連結 [muːvðɪs]
  • block の /k/ が to の /t/ に未開放で接続 [blɑk̚tə]
  • build の /d/ が my の /m/ に連結 [bɪldmaɪ]
アイ ニードゥタ ムーヴディス ブラックゥタ ビルドマイ ハウス

⛏The blocks are moved by the piston every second.

ブロックはピストンによって毎秒移動させられる。

[ðə blɑkzɑr ˈmuːvdbaɪðə ˈpɪs.tən ˈɛv.ə.ri ˈsɛk.ənd]
  • the は弱形 /ðə/ ザ
  • blocks の /z/ が are の /ɑ/ に連結 [blɑkzɑr]
ザ ブラックザー ムーヴドバイザ ピストン エヴェリ セカンド

⛏Move away from the creeper or it will explode!

クリーパーから離れろ、さもないと爆発するぞ！

[muːvəˈweɪ frəmðə ˈkriːpərɔrɪt̚wəlɪkˈsploʊd]
  • move の /v/ が away の /ə/ に連結 [muːvəˈweɪ]
  • from は弱形 /frəm/
  • creeper の /r/ が or の /ɔ/ に連結、or の /r/ が it の /ɪ/ に連結
  • it の /t/ が will の /w/ に未開放で接続、will は弱形 /wəl/
ムーヴアウェイ フラムザ クリーパオーリット ウリクスプロード

💬 よく使う熟語（マイクラ例文）

make a move行動を起こす、手を打つ

⛏We need to make a move before the creeper finds us.

クリーパーに見つかる前に、私たちは行動を起こす必要がある。

[wi niːd̚tə meɪkə muːvbɪˈfɔrðə ˈkriːpərfaɪndzəz]
  • need の /d/ が to の /t/ に未開放で接続、to は弱形
  • make の /k/ が a の /ə/ に連結 [meɪkə]
  • before は語末が /r/
  • finds の /z/ が us に続く
ウィ ニードゥタ メイカ ムーヴビフォーザ クリーパーファインズアズ
move on先へ進む、次へ移動する

⛏Let's move on to the next biome to find diamonds.

ダイヤモンドを探すために、次のバイオームへ進もう。

[lɛts muːvɑn̚tə ðə nɛks̚tbaɪˈoʊm̚tə faɪn̚d ˈdaɪ.mənz]
  • move の /v/ が on の /ɑ/ に連結 [muːvɑn]
  • on の /n/ が to の /t/ に未開放で接続、to は弱形
  • next の /t/ が biome の /b/ に未開放で接続
  • biome の /m/ が to の /t/ に未開放で接続
レッツ ムーヴオン タ ザ ネクスト バイオーム タ ファインド ダイモンズ
move out移動して出ていく、引っ越す

⛏We should move out of this desert before nightfall.

夜落前に、この砂漠から出ていくべき。

[wi ʃəd̚muːvaʊt̚əvðɪzˈdɛz.ər̚tbɪˈfɔrˈnaɪt.fɔl]
  • should の /d/ が move の /m/ に未開放で接続（弱化形 /ʃəd/）
  • move の /v/ が out の /aʊ/ に連結 [muːvaʊt]
  • out の /t/ が of の /ə/ に連結
  • this の /z/ が desert の /d/ に連結、desert の /t/ が before の /b/ に未開放で接続
ウィ シュド ムーヴアウト アヴディズ デザート ビフォーナイトフォール
move in移動して入る、引っ越す

⛏I built a new house in the village, so I'm going to move in tomorrow.

村に新しい家を建てたので、明日引っ越す予定だ。

[aɪ bɪl̚tə nu haʊsɪn ðə ˈvɪl.ɪʒsoʊaɪmˈɡoʊ.ɪŋ̚tə muːvɪn̚təˈmɑ.roʊ]
  • built の /t/ が a の /ə/ に連結（未開放で [bɪl̚tə]）
  • house の /s/ が in の /ɪ/ に連結 [haʊsɪn]
  • going の /ŋ/ が to の /t/ に未開放で接続、to は弱形
  • move の /v/ が in の /ɪ/ に連結、in の /n/ が tomorrow の /t/ に未開放で接続
