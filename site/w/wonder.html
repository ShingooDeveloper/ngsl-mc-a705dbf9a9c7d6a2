wonder

動詞・名詞/ˈwʌndər/CEFR B1頻度#664
  1. 【動詞】〜かなと思う、疑問に思う（～について思考する、～を知りたいと思う）
  2. 【動詞】不思議に思う、驚く（～に驚く、～が奇妙だと感じる）
  3. 【名詞】不思議なもの、奇跡（珍しく素晴らしいもの、見る価値のあるもの）

⛏️ マインクラフト例文

⛏I wonder if there's a cave below this area.

この地域の下に洞窟があるのかな。

/aɪ ˈwʌndərɪf ðɛɹzə keɪv bɪˈloʊ ðɪs ˈɛɹiə/
  • wonder の r が if の i に連結 [ˈwʌndərɪf] ワンダリフ
  • there's の z が a の弱形 /ə/ に連結 [ðɛɹzə] デアーザ
アイ ワンダリフ デアーザ ケイヴ ビロー ディス エリア

⛏I wonder about the behavior of Endermen here.

ここでのエンダーマンの行動について不思議に思う。

/aɪ ˈwʌndərəbaʊt ðə bɪˈheɪvjərəv ˈɛndərmɛn hɪr/
  • wonder の r が about の ə に連結 [ˈwʌndərəbaʊt] ワンダーラバウト
  • behavior の r が of の ə に連結。of は弱形 /əv/ [bɪˈheɪvjərəv] ビヘイヴャーラヴ
アイ ワンダーラバウト ザ ビヘイヴャーラヴ エンダーメン ヒア
