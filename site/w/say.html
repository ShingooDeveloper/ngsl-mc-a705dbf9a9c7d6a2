say

動詞・名詞/seɪ/CEFR A1頻度#29
  1. 【動詞】話す、言葉で表現する（何かを伝える、相手に告げる）
  2. 【動詞】述べる、主張する（～だと言える、～だと思う）
  3. 【動詞】（命令形で）言いなさい、発声しなさい
  4. 【名詞】発言権、意見を述べる機会（have a say in～など）

⛏️ マインクラフト例文

⛏I say we need a lot of diamonds.

ダイヤはたくさん必要だと思うよ。

[aɪ ˈseɪ wiː ˈniːd ə ˈlɑɾəv ˈdaɪ.əndz]
  • need a → need の /d/ が a の /ə/ に連結[ˈniːdə] ニーダ
  • a lot → a の /ə/ が lot の /l/ に連結[ə ˈlɑt] ア ロット
  • lot of → lot の /t/ が of の /ə/ に挟まってフラップT化[ˈlɑɾəv] ロラヴ
  • of diamonds → of は弱形 /əv/、diamonds の /d/ に続く[əv ˈdaɪ.əndz] オヴ ダイモンズ
アイ セイ ウィー ニーダ ア ロラヴ ダイモンズ

⛏I'd say we need to build a shelter.

夜が来る前に拠点を作った方がいいと思う。

[aɪd ˈseɪ wiː ˈniːd tə ˈbɪld ə ˈʃɛl.tər]
  • I'd say → I'd の /d/ が say の /s/ に未開放で接続[aɪd̚ ˈseɪ] アイッ セイ
  • need to → need の /d/ が to の /ə/ に連結、to は弱形[ˈniːdə] ニーダ
  • to build → to の /ə/ が build の /b/ に連結[tə ˈbɪld] タ ビルド
  • build a → build の /d/ が a の /ə/ に連結[ˈbɪldə] ビルダ
アイッ セイ ウィー ニーダ タ ビルダ シェルター

⛏Tell me what you say about the Creeper!

クリーパーについて君は何て言ってるの？

[ˈtɛl mi wʌt̚ juː ˈseɪ ə.ˈbaʊt ðə ˈkri.pər]
  • what you → what の /t/ が未開放で you の /j/ に接続[wʌt̚ juː] ワット ユー
  • you say → you の /u:/ と say の /s/ が続く。連結なし[juː ˈseɪ] ユー セイ
  • say about → say の /eɪ/ が about の /ə/ に続く[ˈseɪ ə.ˈbaʊt] セイ ア バウト
  • about the → about の /t/ が未開放で the の /ð/ に接続、the は弱形[ə ˈbaʊt̚ ðə] ア バット ザ
テル ミー ワット ユー セイ ア バット ザ クリーパー
