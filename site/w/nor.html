nor

接続詞/nɔr/CEFR B1頻度#965
  1. 【接続詞】neither と対で使い、2つの事柄を等しく否定する（neither X nor Y で「X でもなく Y でもない」）
  2. 【接続詞】前の否定を受けて、追加の否定を表す（～でもなく、～もない）

⛏️ マインクラフト例文

⛏You can't make this tool nor can I craft it alone.

このツールは作れず、また1人では組み立てることもできない

/juː kænt meɪk ðɪs tuːl nɔr kən aɪ kræft ɪt əˈloʊn/
  • can I → /n/ + /aɪ/ → 連結。can の弱形 /kən/ の語末 /n/ が I /aɪ/ に連なり [kə naɪ] → カナイ
  • craft it → /t/ + /ɪ/ → フラップT。craft の語末 /t/ が it の /ɪ/ に連結し、フラップT [ɾ] となり [kræf ɾɪt] → クラフリット
  • it alone → /t/ + /ə/ → フラップT。it の語末 /t/ が alone の /ə/ に連結し、フラップT [ɾ] となり [ɪ ɾə ˈloʊn] → イラローン
ユー キャント メイク ザス トゥール ナー カナイ クラフリット イラローン

  • can I → /n/ + /aɪ/ → 連結。can の弱形 /kən/ の語末 /n/ が I /aɪ/ に連なり [kə naɪ] → カナイ

⛏This ore is rare, nor did I find any similar deposits easily.

この鉱石は稀で、また同様な鉱脈も簡単には見つけられない

/ðɪs ɔr ɪz reɪr nɔr dɪd aɪ faɪnd ˈɛni ˈsɪmələr dɪˈpoʊzɪts ˈiːzɪli/
  • ore is → /r/ + /ɪ/ → 連結。ore の語末 /r/ が is の /ɪ/ に連なり [ɔr ɪz] → オリズ
