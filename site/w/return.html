return

動詞・名詞/rɪˈtɜrn/CEFR A2頻度#336
  1. 【動詞】（どこかに）戻る、帰る（…に戻る）
  2. 【動詞】（物を）返す、返却する（…に返す）
  3. 【動詞】（物が）戻ってくる、返ってくる
  4. 【名詞】帰着、返還、返金（…への帰還）
  5. 【名詞】利益、利益率（投資からの…）

⛏️ マインクラフト例文

⛏I returned to the surface after mining diamonds deep underground.

深い地下でダイヤを採掘した後、地表に戻った。

/aɪ rɪˈtɜrnd tə ðə ˈsɜrfɪs ˈæftər ˈmaɪnɪŋ ˈdaɪmənz dip ˈʌndərɡraʊnd/
    アイ リターン タザ サーフィス アフター マイニング ダイマンズ ディープ アンダーグラウンド

    ⛏Return the diamond tools to the main storage chest.

    ダイヤのツールをメインのストレージチェストに返しておいて。

    /rɪˈtɜrn ðə ˈdaɪmənd tulz tə ðə ˈmeɪn ˈstɔrɪdʒ tʃɛst/
      リターン ザ ダイマンド トゥールズ タザ メイン ストレッジ チェスト

      ⛏The player's items returned to the spawn point after he died in lava.

      プレイヤーがラヴァで死んだ後、彼のアイテムはスポーンポイントに戻ってきた。

      /ðə ˈpleɪərz ˈaɪtəmz rɪˈtɜrnd tə ðə ˈspoʊn ˌpɔɪnt ˈæftər hi daɪd ɪn ˈlɑvə/
      • player's items → 語末の /z/ が次の /aɪ/ に連結 [ˈpleɪərz aɪtəmz]
      ザ プレイヤーズアイタムズ リターン タザ スポーン ポイント アフター ヒー ダイド イン ラヴァ

      💬 よく使う熟語（マイクラ例文）

      return to base基地に戻る

      ⛏We should return to base before nightfall to stay safe.

      安全を保つために日没前に基地に戻るべきです。

      /wi ʃʊd rɪˈtɜrn tə beɪs bɪˈfɔr ˈnaɪtˌfɔl tə steɪ seɪf/
      • to base → 弱形 /tə/ で発音される
      ウィー シュド リターン タ ベース ビフォア ナイトフォール タ ステイ セイフ
      return to spawnスポーンに戻る

      ⛏If you press this button, you will return to spawn instantly.

      このボタンを押すと、すぐにスポーンに戻ります。

      /ɪf ju prɛs ðɪs ˈbʌtən, ju wɪl rɪˈtɜrn tə ˈspoʊn ˈɪnstəntli/
      • return to spawn → 弱形 /tə/ で発音される
      • spawn instantly → 語末の /n/ が次の /ɪ/ に連結 [ˈspoʊnɪnstəntli]
