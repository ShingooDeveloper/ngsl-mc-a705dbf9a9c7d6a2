return

動詞・名詞/rɪˈtɜrn/CEFR A2頻度#336
  1. 【動詞】（どこかに）戻る、帰る（…に戻る）
  2. 【動詞】（物を）返す、返却する（…に返す）
  3. 【動詞】（物が）戻ってくる、返ってくる
  4. 【名詞】帰着、返還、返金（…への帰還）
  5. 【名詞】利益、利益率（投資からの…）
