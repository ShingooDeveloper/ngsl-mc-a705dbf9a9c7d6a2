rest

動詞・名詞/ɹɛst/CEFR A2頻度#527
  1. 【動詞】眠る、休む
  2. 【動詞】（…に）置く、支える（…one's head on a pillow）
  3. 【名詞】休息、睡眠
  4. 【名詞】残り、残余部分（…the rest of...）

⛏️ マインクラフト例文

⛏I need a rest after mining diamonds.

ダイヤを採掘した後、休憩が必要だ。

/aɪ ˈniːd ə ˈɹɛst ˈæftɚ ˈmaɪnɪŋ ˈdaɪməndz/
  • need a → 語末子音 /d/ が母音 /ə/ に連結 [ˈniːdə] ニーダ
  • a は機能語で弱形 /ə/
アイ ニーダ レスト アフター マイニング ダイモンズ

⛏Set off to find the rest of the materials.

残りの材料を見つけるために出発する。

/ˈsɛt ɔf tə ˈfaɪnd ðə ˈɹɛst əv ðə məˈtɪriəlz/
  • set off → 語末 /t/ がフラップ化して /ɔ/ に連結 [ˈsɛɾ ɔf] セラオフ
  • to は機能語で弱形 /tə/
  • of は機能語で弱形 /əv/
セラオフ タ ファインド ザ レスト アヴ ザ マテリアルズ

⛏Let me rest a bit before the next fight.

次の戦いの前に少し休ませてください。

/lɛt mi ˈɹɛst ə ˈbɪt bɪˈfɔr ðə ˈnɛkst ˈfaɪt/
  • Let me → 語末 /t/ がフラップ化して /i/ に連結 [ˈlɛɾ mi] レラミー
  • rest a → 語末 /t/ がフラップ化して /ə/ に連結 [ˈɹɛɾ ə] レラ
  • the は機能語で弱形 /ðə/
レラミー レラ ビット ビフォア ザ ネクスト ファイト
