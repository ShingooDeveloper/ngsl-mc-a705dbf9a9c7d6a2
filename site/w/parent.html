parent

名詞・動詞/ˈpɛɹənt/CEFR A2頻度#296
  1. 【名詞】 親（父親・母親）
  2. 【名詞】 親ブロック、元となるもの
  3. 【名詞】 源となるもの
  4. 【動詞】 親である、子どもを育てる

⛏️ マインクラフト例文

⛏The parent block must be solid, or the door won't work.

親ブロックが固くないと、ドアが動作しません。

[ðə ˈpɛɹənt blɑk məst̚ bi ˈsɑləd ɔr ðə dɔr woʊnt̚ wɜrk]
  • must の /t/ が未開放で be に続く
  • won't の /t/ が未開放で work に続く
ザ ペアレント ブロック マス ビ サラッド オア ザ ドア ウォウント ワーク

⛏The parent of this redstone circuit is unstable.

このレッドストーン回路の親が不安定です。

[ðə ˈpɛɹəɾəv ðɪs ˈɹɛdˌstoʊn ˈsɜrkət ɪz ʌnˈsteɪbəl]
