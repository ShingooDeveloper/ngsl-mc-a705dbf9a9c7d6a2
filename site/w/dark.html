dark

形容詞・名詞/dɑrk/CEFR A2頻度#921
  1. 【形容詞】暗い、明りが無い（光量が少ない）
  2. 【形容詞】（色が）濃い、深い色
  3. 【名詞】暗闇、暗い場所
  4. 【形容詞】邪悪な、不正な

⛏️ マインクラフト例文

⛏The cave is very dark, so I need torches.

洞窟はとても暗いので、たいまつが必要です。

/ðə keɪv ɪz ˈvɛri dɑrk soʊ aɪ ˈniːd ˈtɔrtʃəz/

  • cave の語末子音 /v/ が is の /ɪ/ に連結 [keɪvɪz] ケイヴイズ
