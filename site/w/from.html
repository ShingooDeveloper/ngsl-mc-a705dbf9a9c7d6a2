from

前置詞/frəm/CEFR A1頻度#25
  1. 【前置詞】出発地点や起源（…から来ている、…から始まる）
  2. 原因・理由や根拠（…が原因で、…に基づいて）
  3. 材料や構成要素（…から作られた、…で構成されている）
  4. 時間的・空間的範囲（…から...までの）

⛏️ マインクラフト例文

⛏I made a furnace from iron blocks and cobblestone.

鉄ブロックと丸石から炉を作りました。

[aɪ ˈmeɪdə ˈfɜrnəs frəmˈaɪrən ˈblɑks ænd ˈkɑbəlˌstoʊn]
  • made の語末 /d/ が a の弱形 /ə/ に連結し、[meɪdə] メイダ
  • from の語末 /m/ が iron の /aɪ/ に連結し、[frəmˈaɪrən] フラマイアン
アイ メイダ ファーネス フラマイアン ブラックス アンド コッブルストーン

⛏I learned a lot from an old miner.

昔のマイナーからたくさん学びました。

[aɪ ˈlɜrndə lɑt frəmən ˈoʊld ˈmaɪnər]
  • learned の語末 /d/ が a の弱形 /ə/ に連結し、[ˈlɜrndə] ラーンダ
  • from の語末 /m/ が an の弱形 /ən/ に連結し、[frəmən] フラマン
  • an の語末 /n/ が old の /oʊld/ に連結し、[ənˈoʊld] エノールド
アイ ラーンダ ラット フラマン オールド マイナー

⛏I can load a map from a file.

ファイルからマップを読み込むことができます。

[aɪ kæn loʊdə mæp frəmə ˈfaɪl]
  • load の語末 /d/ が a の弱形 /ə/ に連結し、[loʊdə] ロウダ
  • from の語末 /m/ が a の弱形 /ə/ に連結し、[frəmə] フラマ
アイ キャン ロウダ マップ フラマ ファイル

💬 よく使う熟語（マイクラ例文）

from now onこれからずっと、今後

⛏From now on, I need to keep armor in a safe place.

これからは防具を安全な場所に保管する必要があります。

[frəm naʊ ɑn aɪ ˈniːd tə kiːpˈɑrmər ɪnə ˈseɪf pleɪs]
  • keep の語末 /p/ が armor の /ɑ/ に連結し、[kiːpˈɑrmər] キープアーマー
  • in の語末 /n/ が a の弱形 /ə/ に連結し、[ɪnə] イナ
フラム ナウ オン アイ ニード トゥ キープアーマー イナ セイフ プレイス
from scratch最初から、ゼロから

⛏I want to build a house from scratch.

家をゼロから建てたいです。

[aɪ ˈwɑnt tə bɪldə ˈhaʊs frəm skrætʃ]
  • build の語末 /d/ が a の弱形 /ə/ に連結し、[bɪldə] ビルダ
アイ ウォント トゥ ビルダ ハウス フラム スクラッチ

[frəm taɪm tə taɪmə ˈkripər wɪl əˈtæk]

フラム タイム トゥ タイマ クリーパー ウィル アタック
