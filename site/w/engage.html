engage

動詞/ɪnˈɡeɪdʒ/CEFR B1頻度#1296
  1. 【動詞】（話題・活動に）従事する、取り組む
  2. 【動詞】（敵と）戦闘を開始する、交戦する
  3. 【動詞】（歯車などが）かみ合う、機械が起動する
  4. 【動詞】（人の注意・興味を）引く、引きつける

⛏️ マインクラフト例文

⛏I'm engaged in building a massive castle project.

巨大な城プロジェクトの建設に取り組んでいます。

[aɪm ɪnˈɡeɪdʒd ɪn ˈbɪldɪŋ ə ˈmæsɪv ˈkæsəl ˈprɑdʒɛkt]
  • I'm → I /aɪ/ と am /əm/ がつながる。am は弱形 /m/ [aɪm]
  • in a → in の語末 /n/ が a の /ə/ に連結 [ɪnə]
  • a massive → a は弱形。massive の /m/ に吸収 [əˈmæsɪv]
アイム インゲイジド イン ビルディング ア マッシヴ キャッスル プロジェクト

⛏Redstone pistons engage when the lever is activated.

レバーが起動するとレッドストーンピストンが動作します。

[ˈrɛdˌstoʊn ˈpɪstənz ɪnˈɡeɪdʒ wɛn ðə ˈlɛvər ɪz ˌæktɪˈveɪtəd]
  • pistons engage → pistons の /z/ が engage の /ɪ/ に連結 [ˈpɪstənzɪnˈɡeɪdʒ]
  • the lever → ðə が lever に連結。the は弱形 /ðə/ [ðəˈlɛvər]
  • is activated → is は弱形 /əz/、activated に続く
レッドストーン ピスターンズ インゲイジ ウェン ザ レヴァー イズ アクティベイティド

⛏We should engage the Wither boss carefully.

ウィザーボスと慎重に戦闘すべきです。

[wi ʃʊd ɪnˈɡeɪdʒ ðə ˈwɪðər bɔs ˈkɛrfəli]
  • should engage → should の /d/ が engage の /ɪ/ に連結 [ʃʊdɪnˈɡeɪdʒ]
  • the Wither → ðə が Wither に連結。the は弱形 /ðə/ [ðəˈwɪðər]
  • Wither boss → 語末 /r/ と /b/ が続く

💬 よく使う熟語（マイクラ例文）

engage in～に従事する、～に取り組む

⛏We're engaged in a survival challenge with friends.

友達とサバイバルチャレンジに取り組んでいます。

[wɪr ɪnˈɡeɪdʒd ɪn ə sərˈvaɪvəl ˈtʃælɪndʒ wɪð frɛndz]
  • we're → we と are が連結。are は弱形 /r/ [wɪr]
  • in a → in の /n/ が a の /ə/ に連結 [ɪnə]
  • with friends → with の /ð/ が friends に続く [wɪðˈfrɛndz]
ウィア インゲイジド イン ア サーバイヴァル チャレンジ ウィズ フレンズ
engage with～と相互作用する、～と関わる

⛏How do you engage with the redstone system in creative mode?

クリエイティブモードでレッドストーンシステムとどう相互作用しますか？

[haʊ du ju ɪnˈɡeɪdʒ wɪð ðə ˈrɛdˌstoʊn ˈsɪstəm ɪn kriˈeɪtɪv moʊd]
  • do you → do は弱形 /də/ になり you に連結 [dəˈju]
  • with the → ðə の弱形が redstone に続く [wɪðəˈrɛd...]
  • creative mode → /v/ と /m/ が続く
