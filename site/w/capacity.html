capacity

名詞/kəˈpæsɪti/CEFR B2頻度#1469
  1. 【名詞】容積；容器などが保有できる最大量（この樽の容積は64スタック）
  2. 【名詞】処理能力；システムやメカニズムが対応できる最大のスピード・規模
  3. 【名詞】能力；（…する）ことができる能力や可能性
  4. 【名詞】職務・立場；（…の立場で）個人が持つ公式な役職

⛏️ マインクラフト例文

⛏I need a bigger chest because I've reached my storage capacity.

もっと大きいチェストが必要です。ストレージ容量に達しました。

/aɪ ˈnid ə ˈbɪɡɚ tʃɛst bɪˈkɔz aɪv ritʃt maɪ ˈstɔrɪdʒ kəˈpæsɪti/
  • need の語末子音 /d/ が a の /ə/ に連結。[ˈnidə] ニーダ
  • reached の語末子音 /t/ が my の /m/（子音）の前で未開放。[ritʃt̚ maɪ] リーチッ マイ
アイ ニーダ ビガー チェスト ビコーズ アイヴ リーチッ マイ ストレッジ カパシティ
