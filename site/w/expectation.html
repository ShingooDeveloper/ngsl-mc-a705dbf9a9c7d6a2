expectation

名詞/ˌɛkspɛkˈteɪʃən/CEFR B2頻度#1603
  1. 【名詞】期待、予期（何かが起こることへの信念や期待）
  2. 【名詞】期待される状態・水準（多くの場合は複数形で、〜することが期待される）
  3. 【名詞】見通し、見込み（成功や特定の結果への見通し）

⛏️ マインクラフト例文

⛏My expectation about this build was proven wrong.

この建築についての私の期待は間違っていることが判明した。

/maɪ ˌɛkspɛkˈteɪʃən əˈbaʊt ðɪs ˈbɪld wəz ˈpruvən rɔŋ/
  • expectation about → 語末子音 /n/ が母音 /ə/ に連結
  • this build → 語末子音 /s/ が子音 /b/ に未開放で接続
  • build was → 語末子音 /d/ が子音 /w/ に未開放で接続
