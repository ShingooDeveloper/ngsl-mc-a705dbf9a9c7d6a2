if

接続詞/ɪf/CEFR A1頻度#33
  1. 【接続詞】～であれば、～ならば、もし～（条件を導入）
  2. 【接続詞】～かどうか、～であるか否か（間接疑問）

⛏️ マインクラフト例文

⛏If you see a creeper, run!

クリーパーを見かけたら、逃げろ！

[ɪfjuː siːə ˈkriːpər rʌn]
  • If you: 子音クラスタ /fj/ で続く [ɪfjuː] イフユ
イフユ シア クリーパー ラン

⛏If I collect enough obsidian, I'll build a Nether portal.

黒曜石をたくさん集めたら、ネザーゲートを作るつもり。

[ɪfaɪ kəˈlɛkt ɪˈnʌf ɑbˈsɪdiən aɪl ˈbɪld ə ˈnɛðɚ ˈpɔːrtəl]
  • If I: /f/ が /aɪ/ に連結して [ɪfaɪ] イファイ
  • collect enough: /t/ が /ɪ/ に続き [kəˈlɛkt ɪˈnʌf] コレクト イナフ
イファイ コレクト イナフ アプシディアン アイル ビルド ア ネザー ポータル

⛏I'm checking if this biome is safe for a base.

このバイオームがベースに安全かどうか、確認中。

[aɪm ˈtʃɛkɪŋ ɪf ðɪs ˈbaɪoʊm ɪz seɪf fər ə ˈbeɪs]

💬 よく使う熟語（マイクラ例文）

as ifあたかも～のように、まるで～のような

⛏The ghast moved as if it detected us.

ガストは、まるで私たちを検出したかのように動いた。

[ðə ɡɑːst ˈmuːvd æzɪf ɪt̚ dɪˈtɛktəd ʌs]
  • as if: /z/ が /ɪ/ に連結して [æzɪf] エズイフ
  • it detected: /t/ が未開放で [ɪt̚ dɪˈtɛktəd] イット ディテクテッド
ザ ガースト ムーヴド エズイフ イット ディテクテッド アス
what ifもし～だったら、～だったらどうだろう（仮定的質問）

⛏What if there's a cave under here?

ここの下に洞窟があったらどうしよう？

[wʌtɪf ðɛrz ə ˈkeɪv ˈʌndər hɪr]
  • what if: /t/ が /ɪ/ に連結して [wʌtɪf] ワティフ
  • there's a: /z/ が /ə/ に連結して [ðɛrzə] ゼアーザ
ワティフ ゼアーザ ケイヴ アンダー ヒア
