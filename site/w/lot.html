lot

名詞・副詞/lɑt/CEFR A1頻度#122
  1. 【名詞】大量、多数（a lot of で「～がたくさん」）
  2. 【名詞】敷地、区画（a building lot で「建築用地」）
  3. 【名詞】運命、立場（cast one's lot で「運命を共にする」）
  4. 【副詞】非常に、大いに（I like it a lot で「大好きだ」）

⛏️ マインクラフト例文

⛏I've gathered a lot of stone blocks for our base.

我たちのベースのために石ブロックをたくさん集めました。
