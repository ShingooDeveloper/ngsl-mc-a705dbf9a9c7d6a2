type

名詞・動詞/taɪp/CEFR A1頻度#388
  1. 【名詞】種類、タイプ（...different types of stone blocks...）
  2. 【動詞】タイプする、入力する（...type commands into the console...）

⛏️ マインクラフト例文

⛏This type of pickaxe mines stone faster than others.

この種類のつるはしは他のものより速く石を採掘します。

[ðɪs ˈtaɪpəv ˈpɪkæks ˈmaɪnz stoʊn ˈfæstɚ ðæn ˈʌðɚz]
  • type of → 語末子音 /p/ が弱形 /əv/ に連結 [ˈtaɪpəv]「タイパヴ」
  • than others → 語末 /n/ が母音 /ʌ/ に連結 [ðæn ˈʌðɚz]「ザンアザーズ」
ディス タイパヴ ピッカックス マインズ ストウン ファースター ザン アザーズ

⛏I quickly type out the command to the server.

サーバーにコマンドを素早くタイプします。

[aɪ ˈkwɪkli ˈtaɪp aʊt ðə ˈkɑmænd tə ðə ˈsɝːvɚ]
  • out the → 語末 /t/ が the の弱形 /ðə/ に連結 [aʊt ðə]「アウザ」
  • to the → 機能語 to の弱形 /tə/ が the の弱形 /ðə/ に連結 [tə ðə]「タザ」
アイ クウィックリー タイプアウ ザ コマンド タザ サーヴァー

⛏There are different types of stone blocks available.

利用可能な様々な種類の石のブロックがあります。

[ðɛɹ ɑɹ ˈdɪfərənt ˈtaɪps əv stoʊn ˈblɑks əˈveɪləbəl]
  • types of → 語末子音 /z/ が弱形 /əv/ に連結 [ˈtaɪps əv]「タイプザヴ」
  • blocks available → 語末子音 /s/ が母音 /ə/ に連結 [ˈblɑks əˈveɪləbəl]「ブロックスアベイラブル」
ゼア アー ディフレント タイプザヴ ストウン ブロックスアベイラブル
