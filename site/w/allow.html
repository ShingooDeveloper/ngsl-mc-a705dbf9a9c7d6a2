allow

動詞/əˈlaʊ/CEFR A2頻度#323
  1. 【動詞】1. （人に）許可する、認める。2. （物事を）可能にする、実現させる。3. （時間・余裕などを）設ける、確保する。

⛏️ マインクラフト例文

⛏The server admin allows players to build in creative mode.

サーバー管理者はクリエイティブモードでプレイヤーのビルドを許可しています。

/ðə ˈsɜrvɚ ˈædmɪn əˈlaʊz ˈpleɪɚz tə ˈbɪld ɪn kriˈeɪɾɪv ˈmoʊd/
  • admin の末尾子音 /n/ が allows の /ə/ に連結 [ˈædmɪnəˈlaʊz] アドミナロウズ
  • build の末尾子音 /d/ が in の /ɪ/ に連結 [ˈbɪldɪn] ビルディン
ザ サーバー アドミナロウズ プレイヤーズ タ ビルディン クリエイリヴ モード

⛏This redstone contraption allows the door to open.

このレッドストーン装置はドアが開くことを可能にします。

/ðɪs ˈrɛdstoʊn kənˈtræpʃən əˈlaʊz ðə ˈdɔɹ tə ˈoʊpən/
  • contraption の末尾子音 /n/ が allows の /ə/ に連結 [kənˈtræpʃənəˈlaʊz] コントラプションアラウズ
  • door の末尾子音 /r/ が to の /ə/ に連結 [ˈdɔɹtə] ドータ
ディス レッドストーン コントラプションアラウズ ザ ドータ オープン

/juː ˈmʌst əˈlaʊ jɔɹ ˈænɪməlz tə rɪˈmeɪn ɪn ðɛɹ ˈpɛn/
