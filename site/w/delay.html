delay

動詞・名詞/dɪˈleɪ/CEFR B1頻度#1307
  1. 【動詞】延期する、遅くする（活動を後の時点まで延ばす）
  2. 【名詞】遅延、遅れ（期待時点と実際の時点の間隔）

⛏️ マインクラフト例文

⛏Get it done without any delay.

遅延なくそれを完了させる

/ˈɡɛt ɪt ˈdʌn wɪðˈaʊt ˈɛni dɪˈleɪ/
  • get の /t/ が it の /i/ に連結してフラップ化 [ˈɡɛɾɪt]
  • without の /t/ が any の /ɛ/ に連結してフラップ化 [wɪðˈaʊɾ ˈɛni]
ゲリット ダン ウィザウラ エニー ディレイ

⛏Don't delay building—mobs arrive at night!

建築を遅延させるな—モブが夜に到着する

/ˈdoʊnt dɪˈleɪ ˈbɪldɪŋ ˈmoʊbz əˈraɪv æt ˈnaɪt/
  • don't の /t/ が未開放で delay の /d/ へ [ˈoʊn̚ dɪˈleɪ]
  • attack が弱形 /ə/ で、spawn の /n/ に連結
  • you /juː/ と will /wɪl/ の縮約で /juːl/
ドント ディレイ ビルディング モーブズ アライブ アット ナイト
