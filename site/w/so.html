so

副詞・接続詞/soʊ/CEFR A1頻度#31
  1. 【副詞】程度や量が大きい、非常に（…very, to such a degree …）
  2. 【副詞】このような方法で；その通りに（…in this way, like that …）
  3. 【接続詞】その結果として；だから（…as a result, therefore …）
  4. 【代名詞】前述の内容や事実（…that is true, that is the case …）

⛏️ マインクラフト例文

⛏This diamond ore is so rare that I found only three blocks.

このダイヤモンド鉱石は本当にレアで、3ブロックしか見つからない。

  • ore is → ore の末尾 /r/ が is の初頭母音 /ɪ/ に連結
  • that I → /t/ が未開放で次の /aɪ/ に [ðæt̚ aɪ] ザッ アイ
  • found only → found の末尾 /d/ が only の初頭 /oʊ/ に連結
