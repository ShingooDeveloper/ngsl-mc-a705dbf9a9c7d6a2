report

動詞・名詞/rɪˈpɔrt/CEFR B1頻度#165
  1. 【動詞】（情報・問題などを）報告する（…に）
  2. 【動詞】（ニュース・情報などを）報道する、知らせる
  3. 【名詞】報告書、レポート（…についての）
  4. 【名詞】評判、風評（どのようなか）
  5. 【動詞】（…の場所に）報告に行く、出向く（in/at）

⛏️ マインクラフト例文

⛏You need to report the creeper damage to your server admin.

クリーパーの被害をサーバー管理者に報告する必要があります。

[juː niːdə rɪˈpɔt̚ðə ˈkriːpɚ ˈdæmɪdʒ tə jɚ ˈsɝvɚ ˈædmɪn]
  • need to → 語末子音 /d/ が弱形 to の /ə/ に連結 [niːdə] ニーダ
  • report the → 語末子音 /t/ が未開放で詰まり、the の弱形に続く [ˈpɔt̚ðə] ラポッザ
  • to your → 弱形 to /tə/ が your /jɚ/ に先行 [tə jɚ] タ ユア
ユー ニーダ ラポッザ クリーパー ダメージ タ ユア サーバー アドミン

⛏I reported that I saw a Wither in the nether.

ネザーでウィザーを見たと報告しました。

[aɪ rɪˈpɔɾɪd̚ ðæt aɪ sɔ əˈwɪðɚ ɪn̚ðə ˈnɛðɚ]
  • a Wither → 弱形 a の /ə/ が Wither の /w/ に連結 [əˈwɪðɚ] ウィザー
  • reported that → 語末子音 /d/ が未開放で詰まり、that の /ðæt/ に続く [ɪd̚ ðæt] イッド ザット
  • in the → 語末子音 /n/ が未開放で詰まり、the の弱形に続く [ɪn̚ðə] イン ザ
アイ ラ イ ポー ラ イッド ザット アイ ソー ウィザー イン ザ ネザー

⛏The monthly mining report shows excellent progress in resource gathering.

毎月の採掘レポートは資源収集で優れた進捗を示しています。

[ðəˈmʌnθli ˈmaɪnɪŋ rɪˈpɔt̚ʃoʊz ɪkˈsɛlənt ˈprɑɡrɛs ɪn̚ rɪˈsɔrs ˈɡæðərɪŋ]
  • The monthly → 弱形 The の /ə/ が monthly の /m/ に連結 [ðəˈmʌnθli] ザマンスリー
  • report shows → 語末子音 /t/ が未開放で詰まり、shows の /ʃ/ に続く [ˈpɔt̚ʃoʊz] ラポー ショーズ
  • in resource → 語末子音 /n/ が未開放で resource に続く [ɪn̚ rɪˈsɔrs] イン リソース
ザ マンスリー マイニング ラポー ショーズ イクセラント プログレス イン リソース ガザリング
