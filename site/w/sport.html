sport

名詞・動詞/spɔrt/CEFR B1頻度#602
  1. 【名詞】スポーツ、競技活動、運動（play a sport, enjoy sports）
  2. 【名詞】楽しみ、娯楽、遊び（in sport で「ふざけて」「遊び感覚で」）
  3. 【動詞】（衣服・装身具などを）身につける、装備する（sport armor, sport a weapon）
  4. 【動詞】楽しむ、遊ぶ、楽しみにする（sport a trident, sport with friends）

⛏️ マインクラフト例文

⛏I sport full diamond armor when exploring the Nether.

ネザーを探索するときにダイヤのフルアーマーを装備しています。

/aɪ spɔɹt ˈfʊl ˈdaɪmənd ˈɑɹmɚ wɛn ɪkˈsplɔɹɪŋ ðə ˈnɛðɚ/
  • sport の /t/ は未開放で詰まる [spɔɹt̚]
  • the は機能語の弱形 /ðə/ で発音される
  • when exploring で when の末尾 /n/ が exploring の /ɪ/ に連結する
アイ スポッ フル ダイマンド アーマー ウェン イクスプローリング ザ ネザー

⛏Playing Minecraft is a fun sport for me.

マインクラフトをすることは私にとって楽しいスポーツです。

/ˈpleɪɪŋ ˈmaɪnˌkræft ɪz ə ˈfʌn spɔɹt fɚ mi/
  • Minecraft is で craft の /t/ がフラップ化して /ɾ/ に変わり、is の /ɪ/ に連結 [kræɾɪz]
  • is a で /z/ が母音 /ə/ に連結 [ɪzə]
  • sport の /t/ は未開放で詰まる [spɔɹt̚]
プレイング マインクラフリザ ファン スポッ フォー ミー

⛏They sport enchanted tools during raids.

彼らはレイド中にエンチャントされた道具を装備しています。

/ðeɪ spɔɹt ɪnˈtʃæntɪd ˈtulz ˈdʊɹɪŋ ˈreɪdz/
  • sport enchanted で /t/ がフラップ化して /ɾ/ に変わり、enchanted の /ɪ/ に連結 [spɔɹɾɪn]
  • tools during で tools の /z/ が未開放で詰まり、during の /d/ が続く
  • during raids で during の末尾 /ŋ/ と raids の /ɹ/ が自然に連なる
