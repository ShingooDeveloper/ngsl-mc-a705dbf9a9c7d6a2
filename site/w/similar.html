similar

形容詞/ˈsɪmɪlər/CEFR A2頻度#593
  1. 【形容詞】（通常 to で使用）…に似ている、類似している（alike）
  2. 【形容詞】外観や特性が共通する、同じような（resembling）
  3. 【形容詞】相互に比較して似ている、見分けがつきにくい

⛏️ マインクラフト例文

⛏These two blocks look similar.

これら2つのブロックは似ています

/ðiːz tuː blɑks lʊk ˈsɪmɪlər/
    ディーズ トゥー ブロックス ルック シミラー

    ⛏I built a similar structure to yours.

    私はあなたのような似た建造物を作りました

    /aɪ bɪlt ə ˈsɪmɪlər ˈstrʌkʃər tə jɔɹz/
    • built a → /t/ が次の母音に連結
    • to → 弱形 [tə] タ
    アイ ビルタ シミラー ストラクチャー タ ジョーズ

    ⛏You need similar tools for mining this ore.

    この鉱石を採掘するには似たようなツールが必要です

    /juː niːd ə ˈsɪmɪlər ˈtuːlz fɔɹ ˈmaɪnɪŋ ðɪs ɔɹ/
    • need a → /d/ が次の母音に連結
    • for → 弱形 [fɔɹ] フォー
    ユー ニーダ シミラー トゥールズ フォー マイニング ディス オー

    💬 よく使う熟語（マイクラ例文）

    similar to…に似ている

    ⛏These blocks are similar to diamond ore.

    このブロックはダイヤモンド鉱石に似ています

    /ðiːz blɑks ɑɹ ˈsɪmɪlər tə ˈdaɪmənd ɔɹ/
    • to → 弱形 [tə] タ
    ディーズ ブロックス アー シミラー タ ダイマンド オー
    in a similar way同じような方法で

    ⛏Build the next section in a similar way.

    次のセクションも同じような方法で作ります

    /bɪld ðə ˈnɛkst ˈsɛkʃən ɪn ə ˈsɪmɪlər weɪ/
    • the → 弱形 [ðə] ザ
    • a → 弱形 [ə] ア
    ビルド ザ ネクス セクション イン ア シミラー ウェイ
