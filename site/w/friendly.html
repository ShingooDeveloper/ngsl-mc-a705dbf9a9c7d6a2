friendly

形容詞/ˈfrendli/CEFR A2頻度#1535
  1. 【形容詞】(人が) 親切で温かい態度を示す、友好的である（showing kindness and warmth）
  2. 【形容詞】(関係や態度が) 敵対的でない、平和的である（not hostile or aggressive）
  3. 【形容詞】(環境や状況が) 歓迎的・便利である、～に適している（welcoming, favorable, or suitable）
  4. 【形容詞】(ゲーム用語) 味方に対する、チームメイトに対する（relating to teammates or allies）

⛏️ マインクラフト例文

⛏I need a friendly mob spawner here.

ここに親切なモブスポーナーが必要だ。

/aɪ nid ə ˈfrendli mɑb ˈspɑnər hɪr/
  • need a → 語末子音 /d/ が母音 /ə/（弱形）に連結 [ˈniːdə] ニーダ
  • a friendly → 弱形 a /ə/ が consonant /f/ に先行、通常の配列
  • spawner here → /r/ が終わり /h/ で始まる場合、/r/ は母音に挟まれず連結なし
アイ ニーダ フレンドリー モブ スポナー ヒア

⛏Friendly fire is off on this server.

このサーバーではフレンドリーファイアがオフです。

/ˈfrendli ˈfaɪər ɪz ɔf ɑn ðɪs ˈsɝvər/
  • fire is → 語末子音 /r/ が母音 /ɪ/ に連結 [ˈfaɪərɪz] ファイアリズ
  • on this → /n/ と /ð/ の子音同士、連結なし
  • this server → 語末 /s/ と語頭 /s/ の重なり、明確な連結なし
フレンドリー ファイアリズ オフ オン ディス サーバー

⛏The server is really friendly to new players.

そのサーバーは本当に新しいプレイヤーに親切です。

/ðə ˈsɝvər ɪz ˈrili ˈfrendli tə nu ˈpleɪərz/
  • server is → 語末子音 /r/ が母音 /ɪ/ に連結 [ˈsɝvərɪz] サーバーリズ
  • friendly to → 機能語 to は弱形 /tə/ に弱化、語末 /i/ + 弱形 /tə/
  • to new → 弱形 to /tə/ が母音 /u/ に先行 [tə ˈnu] タ ニュー
