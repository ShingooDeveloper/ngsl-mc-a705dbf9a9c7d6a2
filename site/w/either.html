either

限定詞・代名詞・接続詞・副詞/ˈiːðər/CEFR A2頻度#370
  1. 【限定詞・代名詞】2つのもの・ことのうちのどちらか（の）。「A or B」の形で両方の選択肢を示す（例：You can use either method.）
  2. 【限定詞・代名詞】（否定文で）2つのもの・ことのどちらも〜ない。「I don't like either option.」で「どちらのオプションも好きではない」という意
  3. 【接続詞】～か～か、～するか～するか（「either...or」で「AかBか」を示す二者択一）。例：Choose either red or blue.
  4. 【副詞】「同じく、～も」。「I don't like it either.」で「私も」の意

⛏️ マインクラフト例文

⛏You need either diamonds or emeralds.

ダイアモンドかエメラルドのどちらかが必要だ。

/ju ˈniːd ˈiːðər ˈdaɪmənz ɔr ˈɛmərəldz/
  • need either: need の語末子音 [d] が either の先頭母音 [ˈiː] に連結 [ˈniːd iːðər] → ニーダイザー
  • diamonds or: diamonds の語末子音 [z] が or の先頭母音 [ɔ] に連結 [ˈdaɪmənz ɔr] → ダイマンゾー
  • or emeralds: or の語末子音 [r] が emeralds の先頭母音 [ˈɛ] に連結 [ɔr ˈɛmərəldz] → オーエメラルズ

⛏This redstone needs either input or output.

このレッドストーンはインプットかアウトプットのどちらかが必要だ。

/ðɪs ˈrɛdˌstoʊn ˈniːdz ˈiːðər ˈɪnpʊt ɔr ˈaʊtpʊt/
  • needs either: needs の語末子音 [z] が either の先頭母音 [ˈiː] に連結 [ˈniːdz iːðər] → ニーズディザー
  • input or: input [t] は母音に挟まれてフラップ [ɾ] に変化 [ˈɪnpʊ ɾɔr] → インプラオー
  • or output: or の語末子音 [r] が output の先頭母音 [ˈaʊ] に連結 [ɔr ˈaʊtpʊt] → オーアウトプット

⛏Build with either wood or stone blocks.

木材か石ブロックのどちらかで建築する。

/ˈbɪld wɪð ˈiːðər ˈwʊd ɔr ˈstoʊn blɑks/
  • with either: with の語末子音 [ð] が either の先頭母音 [ˈiː] に連結 [wɪð iːðər] → ウィザイザー
  • wood or: wood の語末子音 [d] が or の先頭母音 [ɔ] に連結 [ˈwʊd ɔr] → ウッドオー
  • or stone: or [r] と stone [s] は子音+子音（連結なし）

💬 よく使う熟語（マイクラ例文）

either...or～か～か、～するか～するか（二者択一を示す定型表現）

⛏Either mine stone or search for gravel in that biome.

その生物群系で石を採掘するか砂利を探すかのどちらかだ。

/ˈiːðər ˈmaɪn stoʊn ɔr ˈsɜrtʃ fɚ ˈɡrævəl ɪn ðæt ˈbaɪoʊm/
  • Either mine: either [r] と mine [m] は子音+子音（連結なし）
  • stone or: stone の語末子音 [n] が or の先頭母音 [ɔ] に連結 [stoʊn ɔr] → ストーノー
  • or search: or [r] と search [s] は子音+子音（連結なし）
  • for: 助詞的用法で弱形化 [fɚ] → ファー
