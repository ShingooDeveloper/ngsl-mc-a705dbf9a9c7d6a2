name

名詞・動詞/neɪm/CEFR A1頻度#205
  1. 【名詞】人・物・場所などの呼び名、名前（What is your server's name?）
  2. 【名詞】評判、名声（Players respect builders with a good name.）
  3. 【動詞】〜と名付ける、指名する（You can name pets using name tags.）

⛏️ マインクラフト例文

⛏What's your name on this server?

このサーバーでのあなたの名前は何？

[wʌts jɔr ˈneɪmɑn ðɪs ˈsɜrvər]
  • name on で、語末の子音 /m/ が母音 /ɑ/ に連結 [ˈneɪmɑn]→「ネイマン」
ワッツ ヨア ネイマン ディス サーヴァー

⛏I need to name my horse before the night falls.

夜が明ける前に馬に名前を付けないといけない。

[aɪ niːd̚ tə ˈneɪm maɪ hɔrs bɪˈfɔr ðə naɪt fɔlz]
  • need の語末子音 /d/ が未開放のまま to に続く [niːd̚ tə]→「ニード タ」
  • to は弱形 /tə/→「タ」
  • the は弱形 /ðə/→「ザ」
アイ ニード タ ネイム マイ ホース ビフォア ザ ナイト フォーズ

⛏Name tags are rare drops from mobs.

名札はモブからのレアドロップだ。

[ˈneɪm̚ tægz ə ˈrer drɑps frəm mɑbz]
  • Name tags で、name の語末子音 /m/ が未開放のまま tags に続く [ˈneɪm̚ tægz]→「ネイムッ タグズ」
  • are は弱形 /ə/→「ア」
  • from は弱形 /frəm/→「フラム」
ネイムッ タグズ ア レア ドラップス フラム マブズ

💬 よく使う熟語（マイクラ例文）

good name良い評判、世評が良い

⛏That builder has a good name for creative structures.

あのビルダーはクリエイティブな建築で良い評判がある。

[ðæt ˈbɪldər hæzə ɡʊd̚ ˈneɪm fər kriˈeɪtɪv ˈstrʌkʧərz]
  • has a で、語末の子音 /z/ が母音 /ə/ に連結 [hæzə]→「ハザ」
  • good の語末子音 /d/ が未開放のまま name に続く [ɡʊd̚ neɪm]→「グッド ネイム」
  • for は弱形 /fər/→「ファー」
ザット ビルダー ハザ グッド ネイム ファー クリエイティヴ ストラクチャーズ
name something after〜の名前で命名する、〜にちなんで名付ける

⛏I'll name this village after my friend.

この村を友達の名前で命名しよう。

[aɪl neɪm̚ ðɪs ˈvɪlɪdʒ ˈæftər maɪ frend]
  • name の語末子音 /m/ が未開放のまま this に続く [neɪm̚ ðɪs]→「ネイム ディス」
アイル ネイム ディス ヴィリッジ アフター マイ フレンド
make a name for oneself名をなす、成功を収める

⛏You can make a name for yourself by winning the tournament.

トーナメントに勝つことで名をなすことができる。

[juː kən ˈmeɪkə ˈneɪm fər jɔrˈself baɪ ˈwɪnɪŋ ðə ˈtɜrnəmənt]
  • can は弱形 /kən/→「カン」
  • make a で、語末の子音 /k/ が母音 /ə/ に連結 [ˈmeɪkə]→「メイカ」
  • for は弱形 /fər/→「ファー」
  • the は弱形 /ðə/→「ザ」
ユー カン メイカ ネイム ファー ヨアセルフ バイ ウィニング ザ トーナメント
