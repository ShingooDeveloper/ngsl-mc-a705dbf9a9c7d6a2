remote

形容詞・名詞/rɪˈmoʊt/CEFR B1頻度#2494
  1. 【形容詞】遠く離れた、隔絶した（遠い場所にある、アクセスが困難な）
  2. 【形容詞】可能性が極めて低い、ほぼあり得ない（確率が非常に小さい）
  3. 【名詞】遠隔制御装置、リモートコントロール（a remote control device）

⛏️ マインクラフト例文

⛏I built a base on a remote island far from spawn.

スポーンから遠く離れた孤島に基地を作りました。

/aɪ ˈbɪlt ə ˈbeɪs ɑn ə rɪˈmoʊt ˈaɪlənd ˈfɑr frəm ˈspɔn/
  • built a base → 語末子音 /t/ が母音 /ə/ に連結 [ˈbɪltə ˈbeɪs] ビルタ ベイス
  • on a remote → on の語末子音 /n/ が母音 /ə/ に連結 [ɑn ə rɪˈmoʊt] オン ア リモウト
アイ ビルタ ベイス オン ア リモウト アイレンド ファー フラム スポーン

⛏There is a remote chance of winning this challenge.

このチャレンジに勝つ可能性は非常に低いです。

/ðɛr ɪz ə rɪˈmoʊt ˈtʃæns əv ˈwɪnɪŋ ðɪs ˈtʃælɪndʒ/
  • is a → is の語末子音 /z/ が a の /ə/ に連結、a は弱形 [ɪz ə]
  • of winning → 前置詞 of は弱形 /əv/ [əv ˈwɪnɪŋ] アヴ ウィニング
ゼア イズ ア リモウト チャンス アヴ ウィニング ディス チャレンジ
