assign

動詞/əˈsaɪn/CEFR B1頻度#2490
  1. 【動詞】（仕事・役割・責任を）割り当てる、配置する（assign a task to someone）
  2. 【動詞】（権利・所有権を）譲渡する、譲る
  3. 【動詞】（原因・理由を）帰属させる、考える

⛏️ マインクラフト例文

⛏I will assign you to the building team.

私があなたを建築チームに割り当てます。

/aɪ wɪl əˈsaɪn juː tə ðə ˈbɪldɪŋ tiːm/
  • will assign → [wɪləˈsaɪn] ウィラサイン / will の語末 l が assign の語頭 /ə/ に連結
  • you to the → [juː tə ðə] ユータザ / you の語末母音 /uː/ が to の語頭子音 /t/ に連結、to と the は両方弱形 /tə ðə/
アイ ウィラサイン ユータザ ビルディング ティーム

⛏The admin assigned resources to each player.

管理者は各プレイヤーにリソースを割り当てました。

/ðə ˈædmɪn əˈsaɪnd ˈrɛsɝsɪz tə iːtʃ ˈpleɪər/
  • the admin → [ðə ˈædmɪn] ザ エドミン / the が弱形 /ðə/
  • assigned resources → [əˈsaɪn̚d ˈrɛsɝsɪz] アサインド レサーシーズ / assigned の語末 d が次の r の直前で未開放
  • to each → [tə iːtʃ] タ イーチ / to が弱形 /tə/
ザ エドミン アサインド レサーシーズ タ イーチ プレイアー

⛏Can you assign this mining task to someone?

この採掘タスクを誰かに割り当てることができますか？

/kən juː əˈsaɪn ðɪs ˈmaɪnɪŋ tæsk tə ˈsʌmwʌn/
  • Can you → [kən juː] クン ユー / can が弱形 /kən/、語末 n が未開放
  • you assign this → [juː əˈsaɪn ðɪs] ユー アサイン ディス / you の語末母音 /uː/ が assign の語頭 /ə/ に連結
  • to someone → [tə ˈsʌmwʌn] タ サムワン / to が弱形 /tə/
クン ユー アサイン ディス マイニング タスク タ サムワン

💬 よく使う熟語（マイクラ例文）

assign to〜に割り当てる

⛏They assigned roles to all team members.

彼らはすべてのチームメンバーに役割を割り当てました。

/ðeɪ əˈsaɪnd roʊlz tə ɔːl tiːm ˈmɛmbərz/

  • assigned roles → [əˈsaɪn̚d roʊlz] アサインド ロールズ / d が r の直前で未開放
  • to all → [tə ɔːl] タ オール / to が弱形 /tə/
ゼイ アサインド ロールズ タ オール ティーム メンバーズ
