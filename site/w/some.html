some

限定詞・代名詞/sʌm/CEFR A1頻度#52
  1. 【限定詞】いくつかの、ある（不特定だが存在する数量）
  2. 【限定詞】かなりの、相応の（顕著あるいは相当な量）
  3. 【限定詞】ある、…のいずれか（疑問文・条件文で）
  4. 【代名詞】その一部、それら（先行する何かの部分集合を指す）

⛏️ マインクラフト例文

⛏Some blocks need a pickaxe to break.

ブロックの一部はつるはしが必要です。

/səm blɑks niːd ə ˈpɪkæks tə ˈbreɪk/
  • 語末子音 /d/ が母音 /ə/ に連結、[ˈniːdə] ニーダ（need a）
  • some は不強勢・弱形 /səm/
サム ブロック ニーダ ピッカックス タ ブレイク

⛏Some of our crops are fully grown.

私たちの農作物の一部が完全に成長しました。

/səm əv aʊr krɑps ɑr ˈfʊli ˈɡroʊn/
  • 語末子音 /m/ が母音 /ə/ に連結、of は弱形 /əv/ [səm əv] サムアヴ
  • of の /v/ が our の /aʊ/ に連結
サムアヴ アワー クロップス アー フーリー グロウン

⛏We found some amazing things underground.

地下で素晴らしいアイテムをいくつか見つけました。

/wi faʊnd səm əˈmeɪzɪŋ ˈθɪŋz ˌʌndərˈɡraʊnd/
  • 語末子音 /m/ が母音 /ə/ に連結、[səm əˈmeɪzɪŋ] サムアメイジング（some amazing）
  • some は不強勢・弱形 /səm/
ウィ ファウンド サムアメイジング シングズ アンダーグラウンド

💬 よく使う熟語（マイクラ例文）

some kind ofある種の、何らかの

⛏We need some kind of shelter before night falls.

夜が来る前に何らかの避難所が必要です。

/wi niːd səm ˈkaɪnd əv ˈʃɛltər bɪˈfɔr ˈnaɪt ˈfɔlz/
  • some は弱形 /səm/、of は弱形 /əv/
  • need と some の間は子音群のため明確な連結なし
ウィ ニード サムカインド アヴ シェルター ビフォア ナイト フォルズ
some...others一部は…、他は…

⛏Some ore blocks are common, others are very rare.

一部の鉱石ブロックは一般的ですが、他は非常にレアです。

/səm ˈɔr blɑks ɑr ˈkɑmən / ˈʌðərz ɑr ˈvɛri ˈrɛr/
  • some は弱形 /səm/
サム オア ブロック アー コモン / アザーズ アー ヴェリー レア
