error

名詞/ˈɛɹɚ/CEFR B1頻度#1494
  1. 【名詞】間違い、誤り（不正確な行為や判断）
  2. 【名詞】エラー、不具合（コンピュータやシステムが正常に機能していない状態）
  3. 【名詞】誤認、勘違い（正しくない理解や判断）
  4. （make an error：誤りを犯す、間違いを起こす）
  5. （in error：誤りによって、誤解して）

⛏️ マインクラフト例文

⛏I found an error in my command block.

コマンドブロックにエラーを見つけました。

/aɪ ˈfaʊnd ən ˈɛɹɚ ɪn maɪ kəˈmænd blɑk/
  • found の語末子音 /d/ が an の /ə/（弱形）に連結 [ˈfaʊndən] ファウンダン
  • command の語末子音 /d/ が block の /b/ に続く場合、/d/ は未開放 [kəˈmænd̚ blɑk]
アイ ファウンダン エラー イン マイ コマンド ブロック
