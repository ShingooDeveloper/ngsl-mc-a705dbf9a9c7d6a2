stir

動詞・名詞/stɝ/CEFR B2頻度#2736
  1. 【動詞】液体や混合物をかき混ぜる、攪拌する
  2. 【動詞】（生き物が）動く、活動する、移動する
  3. 【動詞】（感情や興奮を）起こさせる、刺激する、鼓舞する
  4. 【名詞】かき混ぜる行為、騒動

⛏️ マインクラフト例文

⛏I stirred up a lot of dust while mining stone.

石を採掘している時に、大量の粉塵を巻き上げた。

[aɪ ˈstɝd ʌp ə ˈlɑɾəv ˈdʌst waɪl ˈmaɪnɪŋ ˈstoʊn]
  • stirred の /d/ が up の /ʌ/ に連結し、語末子音が次の母音に繋がる [ˈstɝdʌp]。
  • lot の /t/ が of の /ə/ に挟まれてフラップ化し、of の弱形 /əv/ と連結して [lɑɾəv]。機能語 of は弱形で発音される。
アイ スターダップ ア ラーラヴ ダスト ワイル マイニング ストウン

⛏You stir the potion in the brewing stand.

醸造台でポーション液をかき混ぜている。

[ju ˈstɝ ðə ˈpoʊʃən ɪn ðə ˈbruːɪŋ stænd]
  • the は常に弱形 /ðə/ で発音され、複数回出現する。
  • the の /ə/ と potion の /p/ が連結して [ðəˈpoʊʃən]「ザポーション」となり、母音が子音に自然に繋がる。
  • in の /n/ と the の /ð/ は子音+子音なので特に変化なし、the は弱形 /ðə/ で発音される。
ユー スター ザ ポーション イン ザ ブルーイング スタンド

⛏Give the mixture a good stir before drinking it.

飲む前に、混合物をよくかき混ぜてください。

[ɡɪv ðə ˈmɪkstʃɚ ə ˈɡʊd ˈstɝ bɪˈfɔr ˈdrɪŋkɪŋ ɪt]
  • the の弱形 /ðə/ が mixture の /m/ に連結して [ðəˈmɪkstʃɚ]「ザミクスチャー」。
  • a の弱形 /ə/ が good の /ɡ/ に連結して [əˈɡʊd]「ァグッド」。機能詞 a は弱化して母音のみになり、次の子音に繋がる。
  • want の /t/ は unreleased になり、to の /t/ に繋がる。

💬 よく使う熟語（マイクラ例文）

stir up巻き起こす、立ち上らせる、扇動する、混乱させる

⛏Stir up the creepers when you want to practice fighting.

戦闘の練習をしたい時は、クリーパーを刺激してけしかけるといい。

[ˈstɝ ʌp ðə ˈkripərz wɛn ju wɑnt tu ˈpræktɪs ˈfaɪtɪŋ]
  • stir の /r/ が up の /ʌ/ に連結。rhotic r が母音の前で明確に聞こえる。
  • up の /p/ と the の /ð/ は子音+子音なので特に変化なし、the は弱形 /ðə/ で発音される。
  • want の /t/ は unreleased になり、to の /t/ に繋がる。
