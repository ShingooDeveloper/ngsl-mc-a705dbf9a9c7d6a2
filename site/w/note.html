note

名詞・動詞/noʊt/CEFR A2頻度#383
  1. 【名詞】紙に書いた短い記録。メモ帳。ノート。
  2. 【名詞】銀行券。紙幣。
  3. 【名詞】音符。
  4. 【動詞】注意を払う。気づく。（…に）
  5. 【動詞】記録する。書き留める。

⛏️ マインクラフト例文

⛏I keep a note about my base's coordinates.

基地の座標についてメモを保持する。

[aɪ ˈkiːpə ˈnoʊtəˈbaʊt maɪ ˈbeɪsɪz ˈkoʊɚdɪneɪts]
  • keep a → p が母音 /ə/ に連結 [kiːpə]
  • note about → t が弱形母音 /ə/ に連結 [noʊtə]
アイ キーパ ノゥタバウト マイ ベイシズ コウアディネイツ

⛏Note that zombies spawn faster in dark caves.

ゾンビは暗い洞窟でより速く出現することに注意してください。

[noʊt̚ ðæt ˈzɑmbiz ˈspɔn ˈfæstərɪn dɑrk keɪvz]
  • Note の末子音 /t/ が未開放 [noʊt̚]
  • faster in → r が母音 /ɪ/ に連結 [fæstərɪn]
ノゥッ ザット ザンビーズ スポーン ファスタリン ダーク ケイヴズ

⛏Don't forget to note down where you found diamonds.

ダイヤモンドを見つけた場所をメモするのを忘れずに。

[doʊnt̚ fərˈɡɛt tə ˈnoʊt daʊn wɛr juː faʊnd ˈdaɪmənz]
  • Don't → t が未開放 [doʊnt̚]
  • to → 機能語は弱形 /tə/ に弱化
ドゥント ファゲット ター ノゥト ダウン ウェア ユー ファウンド ダイマンズ
