announce

動詞/əˈnaʊns/CEFR B1頻度#1001
  1. 【動詞】（公式に、広く）～を発表する、告知する。（イベント、ルール変更、ニュースなど重要な情報を人々に知らせる）
  2. 【動詞】（話し始めるなど）～を明白にする、宣言する。（新しい計画や決定をサーバー内で公表する）
  3. 【動詞】～の到着または存在を示す、告げる。（プレイヤーの参加や拠点の完成などを知らせる）

⛏️ マインクラフト例文

⛏I announce a new cave system to explore.

探検する新しい洞窟システムを発表します。

/aɪ əˈnaʊns ə ˈnjuː keɪv ˈsɪstəm tə ɪkˈsplɔr/
  • announce の /s/ が a の /ə/ に連結 [əˈnaʊnsə]→アナウンサ
  • to は機能語で弱形 /tə/→タ
アイ アナウンサ ニュー ケイヴ システム タ イクスプロア

⛏The admin announces all updates in chat.

管理者がすべてのアップデートをチャットで告知します。

/ðə ˈædmɪn əˈnaʊnsɪz ɔl ˈʌpˌdeɪts ɪn tʃæt/
  • the は弱形 /ðə/→ザ
  • announces の /z/ が all の /ɔ/ に連結 [əˈnaʊnsɪzɔl]→アナウンシズオル
ザ アドミン アナウンシズオル アップデイツ イン チャット

⛏We announce all our new discoveries.

私たちは全ての新しい発見を発表します。

/wi əˈnaʊns ɔl aʊr ˈnjuː dɪˈskʌvəriz/
  • announce の /s/ が all の /ɔ/ に連結 [əˈnaʊnsɔl]→アナウンスオル
  • our は /aʊr/→アワ（弱形なし）
ウィ アナウンスオル アワ ニュー ディスカバリーズ
