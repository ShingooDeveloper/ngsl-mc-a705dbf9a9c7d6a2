release

動詞・名詞/rɪˈliːs/CEFR B1頻度#785
  1. 【動詞】（矢・ブロック・アイテムなど）を手から放つ、発射する、解放する
  2. 【動詞】（液体・ガス・レッドストーン信号など）を放出する、流す
  3. 【動詞】（新バージョン・更新・アップデートなど）を公開する、リリースする
  4. 【名詞】解放、放出、公開、発売

⛏️ マインクラフト例文

⛏Release the arrow when you aim at the skeleton.

スケルトンに狙いを定めたら矢を放ってください。

/rɪˈliːs ðə ˈæroʊ wɛn ju eɪm æt ðə ˈskɛlɪtən/
  • Release the → Release の語末 /s/ が the の初頭 /ð/ に先行。子音+子音のため /s/ は未開放 [rɪˈliːs̚ðə]「リリースザ」
  • aim at the → aim の語末 /m/ が at の初頭 /æ/ に連結。at の語末 /t/ は the の /ð/ に未開放で先行 [eɪm ætˤðə]「エイム アッザ」
