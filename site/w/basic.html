basic

形容詞・名詞/ˈbeɪsɪk/CEFR A1頻度#909
  1. 【形容詞】基本的な・基礎的な（基本となる、最も必要な）
  2. 【形容詞】単純な・簡素な（複雑でない）
  3. 【形容詞】最低限の（…に必要な最小限のもの）
  4. 【名詞】基本・基礎（…の基盤となるもの）

⛏️ マインクラフト例文

⛏Mining for iron ore is a basic skill in survival mode.

マインクラフトのサバイバルモードでは、鉄鉱石を採掘することは基本的なスキルです。

/ˈmaɪnɪŋ fɔɹ ˈaɪɹən ɔɹ ɪz ə ˈbeɪsɪk skɪl ɪn sɚˈvaɪvəl moʊd/
  • for の語末 /ɹ/ が iron の /ˈaɪ/ に連結 [fɔɹ ˈaɪ] フォーアイ
  • is の語末 /z/ が a の /ə/ に連結、a は弱形 [ɪz ə] イザ
マイニング フォーアイラン オー イザ ベイシック スキル イン サーバイバル モード

⛏Wood is a basic material for building your house.

木材は家を建築するための基本的な材料です。

/wʊd ɪz ə ˈbeɪsɪk məˈtɪɹiəl fɔɹ ˈbɪldɪŋ jɔɹ haʊs/
  • Wood の語末 /d/ が is の /ɪ/ に連結 [wʊd ɪz] ウッディズ
  • is の語末 /z/ が a の /ə/ に連結、a は弱形 [ɪz ə] イザ
  • for の語末 /ɹ/ が building の /ˈbɪ/ に連結 [fɔɹ ˈbɪ] フォービ
ウッド イザ ベイシック マテリアル フォービルディング ユア ハウス

⛏Knowing the basics helps you build anything in the game.

基本を知ることで、ゲーム内で何でも建築できるようになります。

/ˈnoʊɪŋ ðə ˈbeɪsɪks hɛlps ju bɪld ˈɛniθɪŋ ɪn ðə ɡeɪm/
  • in の /n/ が未開放で the の /ðə/ に続く [ɪn̚ ðə] イン・ザ
ノーイング ザ ベイシックス ヘルプス ユー ビルド エニスィング イン ザ ゲイム
