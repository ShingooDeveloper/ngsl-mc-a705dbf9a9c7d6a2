suggest

動詞・名詞/səˈdʒɛst/CEFR B1頻度#328
  1. 【動詞】提案する、勧める
  2. 【動詞】ほのめかす、示唆する
  3. 【動詞】~があることを示す、証拠として示す
  4. 【名詞】提案、示唆

⛏️ マインクラフト例文

⛏I suggest we go mining for diamonds.

私たちがダイヤモンドを採掘しに行くことを提案します。

/aɪ səˈdʒɛst wi ɡoʊ ˈmaɪnɪŋ fɚ ˈdaɪmənz/
  • 「for」は弱形 /fɚ/ に弱化
アイ サジェスト ウィ ゴー マイニング フラ ダイマンズ

⛏The village suggests a good place to build.

その村は建築に良い場所を示唆しています。

/ðə ˈvɪlɪdʒ səˈdʒɛsts ə ɡʊd pleɪs tə ˈbɪld/
  • 「The」は弱形 /ðə/、「a」は弱形 /ə/、「to」は弱形 /tə/
ザ ヴィリッジ サジェスツ ア グッド プレイス タ ビルド

⛏Your creeper damage suggests we need better armor.

クリーパーのダメージがあなたにはより良い防具が必要なことを示唆しています。

/jɚ ˈkriːpɚ ˈdæmɪdʒ səˈdʒɛsts wi nid ˈbɛɾɚ ˈɑɹmɚ/
  • 「Your」は弱形 /jɚ/
ヤ クリーパラ ダメッジ サジェスツ ウィ ニード ベララ アーマラ

💬 よく使う熟語（マイクラ例文）

suggest + ing～することを提案する

⛏I suggest using better tools for mining.

採掘に更に良いツールを使うことを提案します。

/aɪ səˈdʒɛst ˈjuzɪŋ ˈbɛɾɚ tulz fɚ ˈmaɪnɪŋ/
  • 「for」は弱形 /fɚ/ に弱化
