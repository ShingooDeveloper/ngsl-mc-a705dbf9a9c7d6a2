multiple

形容詞・名詞/ˈmʌltɪpəl/CEFR B1頻度#1729
  1. 【形容詞】複数の、多くの、たくさんの（more than one; many）
  2. 【形容詞】複数の部分・要素からなる、多面的な（having many components or aspects）
  3. 【名詞】倍数、倍（a number that contains another number a certain number of times）

⛏️ マインクラフト例文

⛏I need multiple tools for building.

建築に複数の道具が必要です。

/aɪ ˈniːd ˈmʌltɪpəl ˈtulz fɚ ˈbɪldɪŋ/
  • need の語末子音 /d/ が multiple の /m/（子音）に直続、/d/ は未開放 [ˈniːd̚ ˈmʌltɪpəl]
  • for は弱形 /fɚ/
アイ ニード マルティパル トゥルズ ファー ビルディング
