reference

名詞・動詞/ˈɹɛfəɹəns/CEFR B1頻度#876
  1. 【名詞】参考資料、参考文献（…reference material）
  2. 【名詞】言及、参照（…make a reference to something）
  3. 【名詞】基準、基準点（…reference point）
  4. 【動詞】言及する、参照する、参照に挙げる（…reference a guide）

⛏️ マインクラフト例文

⛏I need a reference for building this castle.

この城を建てるための参考資料が必要だ。

/aɪ ˈniːd ə ˈɹɛfəɹəns fɔɹ ˈbɪldɪŋ ðɪs ˈkæsəl/
  • need a：語末の /d/ が次の弱形 /ə/ に連結 [ˈniːdə]
アイ ニーダ レファレンス フォー ビルディング ディス キャスル

⛏Can you reference the redstone recipe?

レッドストーンのレシピを参照してくれませんか？

/ˈkæn jə ˈɹɛfəɹəns ðə ˈɹɛdstoʊn ˈɹɛsɪpi/
  • Can you：機能語 you /jə/ は弱形に弱化
キャン ヤ レファレンス ザ レッドストーン レシピ

⛏The map is my reference point for building.

地図は建築のための私の参照ポイントです。

/ðə ˈmæp ɪz maɪ ˈɹɛfəɹəns ˈpɔɪnt fɔɹ ˈbɪldɪŋ/
  • The map：定冠詞 the /ðə/ は弱形
ザ マップ イズ マイ レファレンス ポイント フォー ビルディング
