model

名詞・動詞/ˈmɑːdəl/CEFR B1頻度#484
  1. 【名詞】見本、模型、モデル（…a scale model of a building）
  2. 【名詞】ファッションモデル、モデル業界の人（…work as a model）
  3. 【名詞】理論的枠組み、システム（…the data model for this game）
  4. 【動詞】模型を作る、形を成形する（…model a structure from clay）
  5. 【動詞】～に倣う、手本にする（…model yourself after someone）

⛏️ マインクラフト例文

⛏I built a model of the Nether portal using obsidian blocks.

黒曜石ブロックを使ってネザーゲートの模型を作った。

/aɪ ˈbɪltə ˈmɑːdəl əv ðə ˈnɛðər ˈpɔːɹtəl ˈjuːzɪŋ əbˈsɪdiən ˈblɑːks/
  • built + a: 語末子音 /t/ が母音 /ə/ に連結 [ˈbɪltə] ビルタ
  • model の /l/ が of の /əv/ に連結。of は弱形 [ˈmɑːdəl əv] マデル アヴ
  • the は弱形 /ðə/ に弱化。母音前なので独立 [ðə] ザ
アイ ビルタ マデル アヴ ザ ネザー ポータル ユージング アブシディアン ブラックス

⛏You should model your base design after famous builders on YouTube.

有名なビルダーたちのように、あなたの基地デザインをまねるべきだ。

/juː ʃəd ˈmɑːdəl jɔːɹ ˈbeɪs dɪˈzaɪn ˈæftər ˈfeɪməs ˈbɪldəɹz ɑːn ˈjuːˌtjuːb/
  • should は弱形 /ʃəd/。語末 /d/ が未開放で model の /m/ に連接 [ʃəd̚ ˈmɑːdəl] シャッ マデル
  • model + your: 語末液音 /l/ が半母音 /j/ に連結 [ˈmɑːdəl jɔːɹ] マデルユア
  • on + YouTube: /n/ と /j/ が連結 [ɑn ˈjuːˌtjuːb] オン ユーチューブ
ユー シャッ マデルユア ベース ディザイン アフター フェイマス ビルダーズ オン ユーチューブ

/ðə ˈkʌstəm ˈɑːɹmər ˈmɑːdəl ˈmeɪks jɔːɹ ˈkɛɹɪktər ˈlʊk ˈmʌtʃ ˈstɹɑːŋɡər/

  • makes + your: 語末 /z/ が半母音 /j/ に連結 [ˈmeɪkz jɔːɹ] メイクスユア
