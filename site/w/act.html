act

動詞・名詞/ækt/CEFR A2頻度#514
  1. 【動詞】行動する、対応する（take action）
  2. 【動詞】機能する、作用する（function, work as）
  3. 【動詞】演じる、役を演じる（perform, play a role）
  4. 【名詞】行動、動作（action, deed）
  5. 【名詞】劇の幕、舞台演技（scene in a play, performance）

⛏️ マインクラフト例文

⛏We should act quickly before the creeper explodes.

クリーパーが爆発する前に素早く行動すべきだ。

/wi ˈʃəd æk̚ ˈkwɪkli bɪˈfɔr ðə ˈkriːpər ɪkˈsploʊdz/
  • should act → should の /d/ が act の /æ/ に連結 [ʃəd ækt]
  • act quickly → act の /t/ は同子音 /k/ の前で未開放（口を開かずに止まる）
  • before the → before の /r/ が the の /ð/ に連結、the は弱形 /ðə/
ウィ シュド アック クウィックリ ビフォー ザ クリーパー イクスプロウズ

/ðə ˈrɛdstəʊn ˈæktsəz ə ˈpaʊər ˈsɔrs fər aʊr ˈdɔrz/
  • acts as → acts の /s/ が as の /ə/ に連結して [ˈæktsəz]
  • for our doors → for は弱形 /fər/、our の /r/ が doors の /d/ に連結
ザ レッストーン アクツェズ ア パウアー ソース ファー アワー ドーズ

  • acted out the → acted の /d/ が out の /aʊ/ に連結、out の /t/ が the の /ð/ に連結 [ˈæktəd aʊt ðə]
