aim

動詞・名詞/eɪm/CEFR A2頻度#793
  1. 【動詞】狙う（銃やボウを使って敵をターゲットに定める）
  2. 【動詞】目指す（ゲームの目標に向かって進む）
  3. 【動詞】目的とする（特定の結果を意図する）
  4. 【名詞】狙い、目標（達成したい最終的な目的）
  5. 【名詞】照準、方向（狙いを定めた向き）

⛏️ マインクラフト例文

⛏I aim to find diamonds in this cave.

この洞窟でダイアモンドを見つけることを目指しています。

/aɪ eɪm tə ˈfaɪnd ˈdaɪmənz ɪn ðɪs keɪv/
  • aim to → aim の語末 /m/ の後に to の弱形 /tə/ が続く（連結なし、子音+子音）
アイ エイム タ ファインド ダイアモンズ イン ディス キェイヴ

⛏Aim for the enderman's head to defeat it quickly.

エンダーマンの頭を狙って素早く倒してください。

/eɪm fɔrðə ˈɛndərmænz hɛd tə dɪˈfiːɾɪt ˈkwɪkli/
  • defeat it → defeat の /t/ が it の /ɪ/ の前でフラップ [ɾ]。/dɪˈfiːɾɪt/「ディフィーリット」
エイム フォーザ エンダーマンズ ヘッド タ ディフィーリット クィックリー

⛏Our aim is survival on this difficult mode.

このモードでの生き残りが我々の目標です。

/ˈaʊr eɪˈmɪz sərˈvaɪvəl ɑn ðɪs ˈdɪfɪkəlt moʊd/
  • aim is → aim の語末 /m/ が is の /ɪ/ に連結。/eɪˈmɪz/「エイミズ」
アワー エイミズ サーバイバル オン ディス ディフィカルト モード
