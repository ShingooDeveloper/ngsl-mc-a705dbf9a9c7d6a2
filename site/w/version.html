version

名詞/ˈvɝːʒən/CEFR B1頻度#978
  1. 【名詞】バージョン；ゲームやソフトウェアの特定のリリースまたは形態（…the latest version of Minecraft…）
  2. 【名詞】（事柄の）版、変種；ある時点での形態または説明（…a different version of the design…）
  3. 【名詞】説明、アカウント；出来事の描写（…give your version of what happened…）

⛏️ マインクラフト例文

⛏I need a new version of the game to play online.

オンラインでプレイするために新しいバージョンのゲームが必要です。

/aɪ ˈniːd ə njuː ˈvɝːʒən əv ðə ɡeɪm tə pleɪ ˈɑnlaɪn/

  • to は弱形 /tə/

⛏The latest version has better graphics and performance.

/ðə ˈleɪtɪst ˈvɝːʒən hæz ˈbɛɾər ˈɡræfɪks ənd pərˈfɔrməns/
