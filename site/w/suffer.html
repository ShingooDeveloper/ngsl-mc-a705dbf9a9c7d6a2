suffer

動詞・名詞/ˈsʌfɚ/CEFR B2頻度#836
  1. 【動詞】痛みや不快感を経験する。耐える（...を受ける）
  2. 【動詞】悪い結果や損害を被る。被害を蒙る（...を被る）
  3. 【動詞】支障をきたす。品質が低下する（...が損なわれる）
  4. 【名詞】苦難、被害（...を経験する）

⛏️ マインクラフト例文

⛏My world suffers from lag when I render too many chunks at once.

一度にあまりにも多くのチャンクをレンダリングするとき、私のワールドはラグに苦しむ。

/maɪ ˈwɝld ˈsʌfɚz frəm ˈlæɡ wɛn aɪ ˈrɛndɚ ˈtuː ˈmɛni ˈʧʌŋks ət ˈwʌns/
  • suffers from → 連結なし（子音+子音）。suffers の /z/ と from の /f/ が別々。from は弱形 /frəm/
  • from lag → 連結なし（子音+子音）
  • lag when → 連結なし。lag の /g/ と when の /w/ が別々に発音
  • render too → 連結なし（子音+子音）
