this

限定詞・代名詞・副詞/ðɪs/CEFR A1頻度#18
  1. 【限定詞】この、この…（ものを特定する）（this ore, this block）
  2. 【代名詞】これ、このもの（単独で指す）（This is a treasure）
  3. 【副詞】このほど、これほど（距離や量）（dig this deep）

⛏️ マインクラフト例文

⛏This ore is obsidian.

この鉱石は黒曜石だ

/ðɪs ɔr ɪz ɑbˈsɪdiən/
  • this ore → 語末 /s/ が ore の /ɔ/ に連結し [ðɪsɔɹ]「ディソー」
  • ore is → 語末 /r/ が is の /ɪ/ に連結し [ɔɹɪz]「オーイズ」
ディソーイズ アブシディアン

⛏This is a treasure!

これは宝物だ！

/ðɪs ɪz ə ˈtrɛʒər/
  • is a → 語末 /z/ が a の /ə/ に連結し [ɪzə]「イザ」
ディス イザ トレジャー

⛏I dug this deep and found diamonds.

私はこんなに深く掘って、ダイヤモンドを見つけた

/aɪ dʌg ðɪs diːp ænd faʊnd ˈdaɪməndz/
  • deep and → 語末 /p/ が and の弱形 /ən/ に連結し [diːpən]「ディーペン」
アイ ダグ ディス ディーペン ファウンド ダイマンズ
