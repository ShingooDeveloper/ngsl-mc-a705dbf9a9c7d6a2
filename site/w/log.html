log

名詞・動詞/lɑːɡ/CEFR A2頻度#2223
  1. 【名詞】丸太；木製ブロック（I need five oak logs to build a cabin.）
  2. 【動詞】記録する；ログを取る（The server logs all player activity.）
  3. 【名詞】航海日誌；日記；記録簿
  4. 【動詞】伐採する；〜を丸太に切る

⛏️ マインクラフト例文

⛏I need to chop this oak log.

このオークの丸太をチョップする必要があります。

/aɪ ˈniːd tə ˈtʃɑp ðɪs ˈoʊk ˌlɑːɡ/
  • to は機能語で弱形 /tə/ に弱化。
アイ ニーダ チャップ ディス オークログ

⛏One log makes four wooden planks.

1つの丸太が4枚の木の板を作ります。

/ˈwʌn ˌlɑːɡ ˈmeɪks ˈfɔr ˈwʊdn̩ ˈplæŋks/
  • One の末尾子音 /n/ が log の初頭子音 /l/ に接近 [ˈwʌn ˌlɑːɡ]。
ワン ログ メイクス フォー ウッドゥン プランクス

⛏The server logs all player activity.

サーバーがすべてのプレイヤーの活動を記録します。

/ðə ˈsɝvɚ ˌlɑːɡz ɔl ˈpleɪɚ ækˈtɪvədi/
  • The は冠詞で弱形 /ðə/ に弱化。
  • logs の末尾子音 /z/ が all の初頭母音に連結 [ˌlɑːɡz ˈɔːl]。
