space

名詞・動詞/speɪs/CEFR A1頻度#551
  1. 【名詞】物が入る空の領域（…build in space、take up space）
  2. 【名詞】2つのもの間の距離（…leave space、space between）
  3. 【動詞】均等に配置する、間隔を開ける（…space out）

⛏️ マインクラフト例文

⛏You need space to build.

家を建てるにはスペースが必要です。

[ju ˈniːd speɪs tə ˈbɪld]
    ユ ニード スペイス タ ビルド

    ⛏Let me have some space for mining.

    採掘用にスペースをください。

    [lɛt mə hæv səm speɪs fɚ ˈmaɪnɪŋ]
      レッ ム ハヴ サム スペイス ファー マイニング

      ⛏Space the blocks out evenly when building.

      建築するときブロックの間隔を均等に開けてください。

      [speɪs ðə ˈblɑks aʊt ˈiːvənli wɛn ˈbɪldɪŋ]
      • blocks の s が out の aʊ に連結
      • out の t が evenly の iː に連結（フラップT）
      スペイス ザ ブロックサウリーヴンリー ウェン ビルディング

      💬 よく使う熟語（マイクラ例文）

      take up spaceスペース・容量を使う、場所を取る

      ⛏Your big storage room takes up too much space in the base.

      大きな保管室は基地の場所を取りすぎます。

      [jɔr bɪɡ ˈstɔrɪdʒ ˈrum ˈteɪks əp ˈtu ˈmʌtʃ speɪs ɪn ðə ˈbeɪs]
      • takes の s が up の ə に連結
      • space の s が in の ɪ に連結
      ユア ビッ ストーリジ ルーム テイクサップ トゥー マッチ スペイシン ザ ベイス
      make spaceスペースを作る、場所を空ける

      ⛏I need to make space in my inventory for new tools.

      新しいツール用にインベントリにスペースを作らなければなりません。

      [aɪ ˈniːd tə ˈmeɪk speɪs ɪn maɪ ɪnˈvɛntəri fɔr ˈnu ˈtulz]
      • space の s が in の ɪ に連結
      アイ ニード タ メイク スペイシン マイ インヴェントリー フォー ニュー トゥールズ
