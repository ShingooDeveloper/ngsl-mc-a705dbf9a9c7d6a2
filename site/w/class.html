class

名詞・動詞/klæs/CEFR A1頻度#361
  1. 【名詞】グループまたは種類（同じ性質や等級でまとめた集まり…例：warrior class）
  2. 【名詞】授業、講義、レッスン（教育の場）
  3. 【名詞】級、等級、クラス（品質や段階を表すレベル…例：first class）
  4. 【動詞】分類する、クラス分けする
