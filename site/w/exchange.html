exchange

動詞・名詞/ɪkˈstʃeɪndʒ/CEFR B1頻度#1012
  1. 【動詞】…と…を交換する（物やアイテムをやり取りする）
  2. 【動詞】…を…に両替する（リソースを変換する）
  3. 【動詞】意見・情報・言葉を交わす
  4. 【名詞】交換、取り替え（…の交換）
  5. 【名詞】両替、為替（…の両替）

⛏️ マインクラフト例文

⛏I exchanged my old diamond pickaxe for a new one at the village.

村でボロボロのダイヤモンドツルハシを新しいものと交換した。

/aɪ ɪkˈstʃeɪndʒd maɪ oʊld ˈdaɪmənd ˈpɪkæks fɔɹə ˈnjuː wʌnət ðə ˈvɪlɪdʒ/
  • for a → 語末子音 /r/ が母音 /ə/ に連結 [fɔɹə]「フォーア」
  • one at → 語末子音 /n/ が弱形 /ə/（at の弱形）に連結 [wʌnət]「ワナット」
  • at the → /t/ が未開放で次の /ð/ に続く [ət̚ðə]「アッ ザ」
アイ イクスチェインジド マイ オウルド ダイマンド ピックアックス フォーア ニュー ワナット ザ ビリッジ

⛏We exchanged emeralds for better tools from the merchant.

私たちは商人からより良い道具と引き換えにエメラルドを与えた。

/wiː ɪkˈstʃeɪndʒd ˈɛmərəldz fɔɹ ˈbɛtɚ ˈtulz frəm ðə ˈmɝtʃənt/
  • from the → from は弱形 /frəm/、the は弱形 /ðə/ で [frəm ðə]「フラム ザ」
ウィー イクスチェインジド エメラルズ フォー ベラー トゥールズ フラム ザ マーチャント

⛏They exchanged information about the secret base location.

彼らは秘密基地の場所に関する情報を交わした。

/ðeɪ ɪkˈstʃeɪndʒd ˌɪnfɚˈmeɪʃən əˈbaʊt ðə ˈsikrət ˈbeɪs loʊˈkeɪʃən/
  • information about → information の語末子音 /n/ が about の母音 /ə/ に連結 [ʃən əˈbaʊt]「インファーメイション アバウト」
  • about the → about の /t/ が未開放で the の /ð/ に続く [əˈbaʊt̚ðə]「アバウット ザ」
  • base location → base の語末子音 /s/ が location の母音 /oʊ/ に連結 [ˈbeɪs loʊˈkeɪʃən]「ベイス ロウケイション」
ザイ イクスチェインジド インファーメイション アバウット ザ シークレット ベイス ロウケイション

💬 よく使う熟語（マイクラ例文）

exchange for～と引き換えに、～に変える

⛏I can exchange my gold for emeralds with the villager.

私は村人とゴールドをエメラルドと交換できる。

/aɪ kən ɪkˈstʃeɪndʒ maɪ ˈɡoʊld fɔɹ ˈɛmərəldz wɪð ðə ˈvɪlədʒɚ/
  • can exchange → can は弱形 /kən/ で exchange の /ɪ/ に連結 [kən ɪkˈstʃeɪndʒ]「カン イクスチェインジ」
  • for emeralds → for は弱形 /fɔɹ/ で emeralds の /ɛ/ に連結 [fɔɹ ˈɛm]「フォー エメラルズ」
  • with the → with の語末子音 /ð/ が the の母音 /ə/ に連結 [wɪðə ðə]「ウィザ ザ」
アイ カン イクスチェインジ マイ ゴールド フォー エメラルズ ウィザ ザ ビラジャー
exchange with～と交換する、～とやり取りする

⛏Can you exchange this stone with me for your pickaxe?

この石をあなたのツルハシと交換してくれませんか？

/kæn ju ɪkˈstʃeɪndʒ ðɪs ˈstoʊn wɪð mi fɔɹ jɔɹ ˈpɪkæks/
  • Can you → can の語末子音 /n/ が you の /j/ に連結 [kænjuː]「キャンユー」
  • with me → with は弱形 /wɪðə/ で [wɪðə mi]「ウィザ ミー」
  • for your → for の語末子音 /ɹ/ が your の /j/ に連結 [fɔɹjɔɹ]「フォーユア」
キャンユー イクスチェインジ ディス ストウン ウィザ ミー フォーユア ピックアックス
