comment

動詞・名詞/ˈkɑmənt/CEFR B1頻度#715
  1. 【名詞】意見、考え（個人の見方や感想）
  2. 【名詞】コメント、論評（対象についての簡潔な意見的説明）
  3. 【動詞】意見を述べる、コメントする（…について考えや意見を述べる）

⛏️ マインクラフト例文

⛏The server admin commented on the lag issue affecting players in the new world.

サーバー管理者は新しいワールドのプレイヤーに影響するラグの問題についてコメントした。

[ðə ˈsɜːrvər ˈædmɪn ˈkɑmənɾɪd ɑn ðə læg ˈɪʃuː əˈfɛktɪŋ ˈpleɪərz ɪn ðə ˈnjuː ˈwɜːrld]
  • commented on - /d/ が /ɑ/ に連結
  • lag issue - /g/ が /ɪ/ に連結
  • in the - /n/ が未開放で /ð/ に接続、後に弱形 /ðə/
  • new world - /uː/ が /w/ に連結
ザ サーヴァー アドミン カマンラッド アン ザ ラグ イッシュー アフェクティング プレイアーズ イン ザ ニューワールド

⛏I can't see your comment about the new redstone design in the chat.

チャットで新しいレッドストーン設計についてのあなたのコメントが見えません。

[aɪ ˈkæn̚ siː jər ˈkɑmənɾ əˈbaʊt̚ ðə ˈnjuː ˈrɛdˌstoʊn dɪˈzaɪn ɪn ðə tʃæt]
  • can't see - /t/ が未開放で /s/ に先行
  • see your - /iː/ が /j/ に連結
  • comment about - /t/ が /ə/ に連結（フラップ化）
  • the chat - 弱形 /ðə/ が /tʃ/ に連結
アイ キャント シー ユア カメンラ アバウト ザ ニュー レッドストーン ディザイン イン ザ チャット

⛏Some players left comments about difficulty with the Ender Dragon fight.

何人かのプレイヤーがエンダードラゴンとの戦闘の難しさについてコメントを残しました。

[ˈsʌm̚ ˈpleɪərz ˈlɛft̚ ˈkɑmənts əˈbaʊt̚ ˈdɪfɪkəlti wɪð ðə ˈɛndər ˈdræɡən faɪt]
  • Some players - /m/ が未開放で /p/ に先行
  • left comments - /t/ が未開放で /k/ に先行
  • about difficulty - /t/ が未開放、/d/ が /ɪ/ に連結
  • the Ender - 弱形 /ðə/ が /ɛ/ に連結
サム プレイアーズ レフト コメンツ アバウト ディフィカルティ ウィズ ザ エンダー ドラゴン ファイト

💬 よく使う熟語（マイクラ例文）

make a comment意見を述べる、コメントを言う

⛏The builder made a comment about the unusual stone brick placement in the house.

ビルダーは家の珍しい石レンガの配置についてコメントを述べた。

[ðə ˈbɪldər ˈmeɪd ə ˈkɑmənɾ əˈbaʊt ðə ənˈjuːʒuəl ˈstoʊn ˈbrɪk ˈpleɪsmənt ɪn ðə ˈhaʊs]
  • made a - /d/ が /ə/ に連結
  • a comment - /ə/ が /k/ に連結
  • comment about - /t/ が /ə/ に連結（フラップ化）
ザ ビルダー メイド ア カメンラ アバウト ザ アンユージュアル ストーン ブリック プレイスメント イン ザ ハウス
leave a commentコメントを残す、意見を記録に残す

⛏Please leave a comment on the server board if you need help with mining.

採掘で助けが必要な場合は、サーバーボードにコメントを残してください。

[pliːz ˈliːv ə ˈkɑmənɾ ɑn ðə ˈsɜːrvər ˈbɔːrd ɪf ju ˈniːd ˈhɛlp wɪð ˈmaɪnɪŋ]
  • leave a - /v/ が /ə/ に連結
  • a comment - /ə/ が /k/ に連結
  • comment on - /t/ が /ɑ/ に連結（フラップ化）
プリーズ リーヴ ア カメンラ アン ザ サーヴァー ボード イフ ユー ニード ヘルプ ウィズ マイニング
