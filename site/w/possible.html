possible

形容詞/ˈpɑsəbəl/CEFR A2頻度#289
  1. 【形容詞】（事柄が）可能な、起こり得る（実現する可能性がある）
  2. 【形容詞】（人が）できる、遂行能力がある
  3. 【形容詞】あり得る、存在し得る（現実的である、もっともらしい）

⛏️ マインクラフト例文

⛏It's possible to build a house in Minecraft.

マインクラフトで家を建てることは可能だ。

/ɪts ˈpɑsəbəl tə ˈbɪld ə ˈhaʊs ɪn ˈmaɪnˌkræft/
  • build の語末 /d/ が a /ə/ に連結
イッツ パーサブル タ ビルダ ハウス イン マインクラフト

⛏Is it possible to find diamonds at this level?

このレベルでダイヤを見つけることは可能ですか？

/ɪz ɪt ˈpɑsəbəl tə ˈfaɪnd ˈdaɪməndz æt ðɪs ˈlɛvəl/
  • is の語末 /z/ が it の /ɪ/ に連結
イズイ パーサブル タ ファインド ダイマンズ アッ ディス レヴル

⛏That's a possible location for the base.

それはベースの場所として考えられる位置だ。

/ðæts ə ˈpɑsəbəl ˈloʊkeɪʃən fɔr ðə ˈbeɪs/
  • That's の /s/ が a /ə/ に連結
ザッツア パーサブル ロケイション フォー ザ ベース

💬 よく使う熟語（マイクラ例文）

as ~ as possibleできるだけ～な、～の程度まで可能な限り

⛏Break blocks as fast as possible in the deep cave.

深い洞窟でできるだけ速くブロックを壊す。

/ˈbreɪk ˈblɑks æz ˈfæst əz ˈpɑsəbəl ɪn ðə ˈdiːp ˈkeɪv/
  • blocks の語末 /z/ が as の /ə/ に連結
ブレイク ブラッザ ファース アズ パーサブル イン ザ ディープ ケイブ
if possibleもしできれば、可能ならば

⛏If possible, bring water to survive the lava.

できれば、溶岩から生き残るために水を持ってくる。

/ɪf ˈpɑsəbəl ˈbrɪŋ ˈwɔtər tə sərˈvaɪv ðə ˈlɑvə/
  • if と possible の間は子音＋子音のため連結なし
