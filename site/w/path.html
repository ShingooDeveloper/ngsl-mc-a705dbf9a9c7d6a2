path

名詞・動詞/pæθ/CEFR A1頻度#1292
  1. 【名詞】小道、歩道、進路（…a narrow path through the woods…）
  2. 【名詞】移動ルート、軌跡、進行方向（…follow the mob's path…）
  3. 【名詞】レッドストーン信号やエネルギーの伝播経路（…the redstone signal path…）
  4. 【動詞】経路を設定する、ルートを計画する（…path the minecart to the farm…）

⛏️ マインクラフト例文

⛏I cleared a path through the forest to reach the village.

森を切り開いて村に到達する小道を作った。

/aɪ ˈklɪrd ə ˈpæθ θruː ðə ˈfɔrəst tə ˈritʃ ðə ˈvɪlɪdʒ/
  • cleared a: 語末 d が次の ə に連結
  • forest to: 語末 t がフラップ化 [ɾ] になる可能性あり（[ɾ] = ラ）
  • a、to、the は弱形（/ə/, /tə/, /ðə/ = ア、タ、ザ）
アイ クリアード ア パース スルー ザ フォレスト ラ リーチ ザ ヴィリジ

⛏Watch the path of that arrow—it's heading toward the mob farm.

その矢の軌跡を見て。モブファームに向かっている。

/wɑtʃ ðə ˈpæθ əv ðæt ˈæroʊ ɪts ˈhɛdɪŋ təˈwɔrd ðə ˈmɑb ˈfɑrm/
  • that arrow: 語末 t がフラップ化 [ɾ] になる（[ɾ] = ラ）
  • path of: θ が次の ə に連結（特に変化なし）
  • toward the: 語末 d が次の ə に連結
  • the（複数）、of、toward は弱形
ウォッチ ザ パース アヴ ザット ラロー イッツ ヘディング タワード ザ マブ ファーム

⛏We need to path the redstone signal from the farm to the storage.

農場からストレージまでレッドストーン信号の経路を設定する必要がある。

/wi ˈnid tə ˈpæθ ðə ˈrɛdˌstoʊn ˈsɪɡnəl frəm ðə ˈfɑrm tə ðə ˈstɔrɪdʒ/
  • to（2回）、the（2回）、from は弱形（/tə/, /ðə/, /frəm/ = タ、ザ、フラム）
ウィ ニード タ パース ザ レッドストーン シグナル フラム ザ ファーム タ ザ ストレッジ

💬 よく使う熟語（マイクラ例文）

off the path道から外れる、迷う

⛏Don't go off the path—mobs spawn in the dark forest.

道から外れないで。敵mobは暗い森に出現する。

/doʊnt ɡoʊ ɔf ðə ˈpæθ moʊbz spɔn ɪn ðə dɑrk ˈfɔrəst/
  • spawn in: 語末 n が次の ɪ に連結（鼻音の自然な連結）
  • the（複数回）は弱形 /ðə/（[ðə] = ザ）
ドント ゴー オフ ザ パース モウズ スポーン イン ザ ダーク フォレスト
path to〜への道、手段、方法

⛏Mining obsidian is a path to the Nether.

黒曜石を採掘することはネザーへの道である。

/ˈmaɪnɪŋ əbˈsɪdiən ɪz ə ˈpæθ tə ðə ˈnɛðər/
  • Mining obsidian: 語末 ŋ が次の ə に連結（鼻音の自然な連結）
  • is a: 語末 z が次の ə に連結
  • a、to、the は弱形（/ə/, /tə/, /ðə/ = ア、タ、ザ）
マイニング アブシジアン イズ ア パース タ ザ ネザー
