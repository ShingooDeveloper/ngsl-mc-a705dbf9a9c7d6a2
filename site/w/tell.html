tell

動詞/tɛl/CEFR A1頻度#102
  1. 【動詞】（人に）〜を言う、述べる（…tell someone something）
  2. 【動詞】〜に教える、指示する（…tell someone to do）
  3. 【動詞】〜の区別をつける、見分ける（…tell the difference、tell A from B）
  4. 【動詞】〜を話す、物語を語る（…tell a story）

⛏️ マインクラフト例文

⛏Tell me how to build a house.

家の建て方を教えてください。

[tɛl mi haʊ tə ˈbɪldə haʊs]
  • to が弱形 /tə/ に弱化
  • build a → build の語末 /d/ が a の /ə/ に連結 [bɪldə]
「テル ミー ハウ タ ビルダ ハウス」

⛏This note tells you where the diamonds are.

このメモはダイヤの場所を教えてくれます。

[ðɪs noʊt̚ tɛlz jə wɛr ðə ˈdaɪmændz ɑr]
  • note tells → note の語末 /t/ が未開放で、tells の /t/ に接続
  • you が弱形 /jə/ に弱化
  • the diamonds → the の弱形 /ə/ が diamonds の /d/ に連結 [ðəˈdaɪmændz]
  • diamonds are → diamonds の語末 /z/ が are の /ɑ/ に連結 [ˈdaɪmændzɑr]
「ディス ノッ テルズ ヤ ウェア ザダイマンズアー」

⛏Can you tell the difference between oak and birch wood?

オークとシラカバの違いが分かりますか？

[kən jə ˈtɛl ðə ˈdɪfərəns bɪˈtwɪnoʊkənd bɝtʃ wʊd]
  • Can you → can と you が弱形に弱化 (/kən/, /jə/)
  • the difference → the の弱形 /ə/ が difference の /d/ に連結 [ðə ˈdɪfərəns]
  • between oak and → between の語末 /n/ が oak の /oʊ/ に連結、oak の語末 /k/ が and の /ə/ に連結 [bɪˈtwɪnoʊkənd]
「カン ヤテル ザディファレンス ビトゥウィンノークアンド バーチ ウッド」

💬 よく使う熟語（マイクラ例文）

tell apart（異なるもの）を見分ける、区別する

⛏I can tell these ores apart.

これらの鉱石を見分けられます。

[aɪ kən ˈtɛl ðizoʊrz əˈpɑrt]
  • can が弱形 /kən/ に弱化
  • these ores → these の語末 /z/ が ores の /oʊ/ に連結 [ðizoʊrz]
「アイ カン テル ディズオーズ ア パート」
