cycle

名詞・動詞/ˈsaɪkəl/CEFR B1頻度#1391
  1. 【名詞】周期、循環（例: the day-night cycle）
  2. 【名詞】一連のプロセス、サイクル（例: the growth cycle）
  3. 【動詞】循環する、周期を繰り返す（例: resources cycle）
  4. 【名詞】自転車（主に英国英語）

⛏️ マインクラフト例文

⛏The day-night cycle in Minecraft repeats every 20 minutes of real time.

マインクラフトの昼夜サイクルは実時間で20分ごとに繰り返される。

/ðə ˈdeɪ naɪt̚ ˈsaɪkəlɪn maɪnˈkræft̚ rɪˈpits ˈɛvəri ˈtwɛnti ˈmɪnɪts əvˈril taɪm/
  • cycle in で cycle の /l/ が in の /ɪ/ に連結 [ˈsaɪkəlɪn]
  • night, Minecraft の語末 /t/ が未開放で詰まる
  • of が弱形 /əv/ となる
ザ デイ ナイッ サイカリン マインクラフッ リピーツ エヴァリ トゥエンティ ミニッツ アヴリル タイム

⛏I planted crops and waited for the growth cycle to complete.

作物を植えて、成長サイクルが終わるのを待った。

/aɪ ˈplæntɪd̚ krɑpz ənd̚ ˈweɪtɪd̚ fɚðə ˈɡroʊθ̚ ˈsaɪkəltə kəmˈplit/
  • crops の /s/ が有声化して /z/ となる（voicing in context）
  • for の /r/ が the の /ð/ に連結 [fɚðə]
  • cycle の /l/ が to の /tə/ に連結（to は弱形）
アイ プランティッ クラップス アン ウェイティッ フォーザ グロウス サイカルタ カムプリート

⛏Redstone circuits can cycle through multiple states automatically.

レッドストーン回路は複数の状態を自動的に循環できる。

/ˈrɛdˌstoʊn ˈsɜrkɪt̚s kən ˈsaɪkəl̚ θru ˈmʌltəpəl̚ ˈsteɪt̚s ˌɑtəmætɪkli/
  • circuits, multiple, states の語末子音が未開放で詰まる
  • can が助動詞として弱形 /kən/ になる

💬 よく使う熟語（マイクラ例文）

cycle through〜を順番に回す、次々と切り替える

⛏I cycle through my armor sets to find the best protection against mobs.

モブから最高の保護を得るために、防具セットを順番に切り替えている。

/aɪ ˈsaɪkəl̚ θru maɪ ˈɑrmər sɛt̚s tə ˈfaɪnd̚ ðə ˈbɛst̚ prəˈtɛkʃən əˈɡɛnst ˈmɑbz/

  • find, best の語末子音が未開放
