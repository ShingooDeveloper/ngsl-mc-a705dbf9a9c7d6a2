think

動詞・名詞/θɪŋk/CEFR A1頻度#46
  1. 【動詞】思考する、考える、心に思い浮かべる
  2. 【動詞】～だと思う、考える、判断する（…it's too dangerous…）
  3. 【動詞】計画する、意図する、～するつもりだ（…to build a base…）
  4. 【名詞】考え、思想

⛏️ マインクラフト例文

⛏I think this seed will make an amazing world.

このシードは素晴らしい世界を作ると思う。

/aɪ ˈθɪŋk ðɪs ˈsiːd wəl ˈmeɪkən əˈmeɪzɪŋ ˈwɜrld/
  • make の /k/ が an の /ə/ に連結 [ˈmeɪkən] メイカン
  • an の /n/ が amazing の /ə/ に連結 [ənəˈmeɪzɪŋ] アンアメイジング
  • will は弱形 /wəl/ に弱化
アイ シンク ディス シード ウル メイカン アメイジング ワールド

⛏She thinks mining at night is too dangerous.

彼女は夜間採掘は危険すぎると思っている。

/ʃi ˈθɪŋks ˈmaɪnɪŋ æt ˈnaɪt ɪz ˈtuː ˈdeɪndʒərəs/
  • at の /t/ が night に未開放で連接 [æt̚ ˈnaɪt] アッ ナイッ
  • night の /t/ が is に未開放で接する [ˈnaɪt̚ ɪz]
  • is は弱形 /əz/
シー シングス マイニング アッ ナイッ イズ トゥー デインジャラス

⛏Think before you place redstone.

レッドストーンを設置する前に考えなさい。

/ˈθɪŋk bɪˈfɔrju ˈpleɪs ˈrɛdˌstoʊn/
  • think の /k/ は未開放で before に連接 [ˈθɪŋk̚ bɪˈfɔr]
  • before の /r/ が you に流れ込む [bɪˈfɔrju] ビフォーユ
シンク ビフォーユ プレイス レッストーン

💬 よく使う熟語（マイクラ例文）

think outside the box型にはまらない考え方をする、創造的に考える

⛏You need to think outside the box to create this contraption.

このコントラプションを作るには創造的な考え方が必要だ。

/ju ˈniːd tə ˈθɪŋk ˌaʊtˈsaɪd ðə ˈbɑks tə kriˈeɪt ðɪs kənˈtræpʃən/
  • need の /d/ が to の /t/ に未開放で連接 [ˈniːd̚ tə]
  • to は弱形 /tə/
  • think の /k/ が outside の /aʊ/ に連結 [ˈθɪŋk ˌaʊtˈsaɪd]
