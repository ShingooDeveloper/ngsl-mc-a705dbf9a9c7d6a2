think

動詞・名詞/θɪŋk/CEFR A1頻度#46
  1. 【動詞】思考する、考える、心に思い浮かべる
  2. 【動詞】～だと思う、考える、判断する（…it's too dangerous…）
  3. 【動詞】計画する、意図する、～するつもりだ（…to build a base…）
  4. 【名詞】考え、思想

⛏️ マインクラフト例文

⛏I think this seed will make an amazing world.

このシードは素晴らしい世界を作ると思う。

/aɪ ˈθɪŋk ðɪs ˈsiːd wəl ˈmeɪkən əˈmeɪzɪŋ ˈwɜrld/
  • make の /k/ が an の /ə/ に連結 [ˈmeɪkən] メイカン
  • an の /n/ が amazing の /ə/ に連結 [ənəˈmeɪzɪŋ] アンアメイジング
  • will は弱形 /wəl/ に弱化
アイ シンク ディス シード ウル メイカン アメイジング ワールド

  • night の /t/ が is に未開放で接する [ˈnaɪt̚ ɪz]
  • is は弱形 /əz/
