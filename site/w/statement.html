statement

名詞/ˈsteɪtmənt/CEFR B1頻度#768
  1. 【名詞】陳述、声明。何かについての公式な発言（official statement）
  2. 【名詞】メッセージ、個性の表現。芸術や建築で思想やスタイルを示すこと（make a statement）
  3. 【名詞】証言、供述（legal statement）

⛏️ マインクラフト例文

⛏The new statement from the admin set the server rules.

管理者の新しい声明がサーバーのルールを設定した。

/ðə ˈnuː ˈsteɪtmənt frəm ðə ˈædmɪn sɛt ðə ˈsɜrvər ˈrulz/
