skill

名詞/skɪl/CEFR B1頻度#684
  1. 【名詞】技能、スキル（例：（a particular ability or expertise in an activity））
  2. 【名詞】腕前、手腕（例：（proficiency or competence in performing a task））

⛏️ マインクラフト例文

⛏I need more mining skill to dig through this obsidian.

この黒曜石を採掘するには、採掘スキルをもっと高める必要がある。

[aɪ ˈniːd ˈmɔr ˈmaɪnɪŋ ˈskɪl tə ˈdɪɡ θruː ðɪsəbˈsɪdiən]
  • this obsidian → this の /s/ が obsidian の /ə/ に連結 [ðɪsəbˈsɪdiən]
  • to → 機能語 to は弱形 /tə/ に弱化
アイ・ニード・モア・マイニング・スキル・タ・ディグ・スルー・ディスオブシディアン

⛏My redstone skill has improved a lot this season.

このシーズン、レッドストーン技術が大幅に向上した。

[maɪ ˈredstoʊn ˈskɪl əzɪmˈpruːvd əˈlɑt ðɪs ˈsiːzən]
  • a lot → a の弱形 /ə/ が lot の /l/ に連結 [əˈlɑt]
  • has improved → has は弱形 /əz/、/z/ が improved の /ɪ/ に連結 [əzɪm]
マイ・レッドストウン・スキル・アズイムプルーヴド・アロット・ジス・シーズン

⛏Building skill is what separates experienced players from beginners.

建築スキルが、経験豊かなプレイヤーと初心者を分ける要因である。

[ˈbɪldɪŋ ˈskɪlɪz wʌt ˈsɛpəreɪts ɪkˈspɪriənst ˈpleɪərz frɑm bɪˈɡɪnərz]
  • skill is → skill の /l/ が is の /ɪ/ に連結 [ˈskɪlɪz]
ビルディング・スキリズ・ワット・セパレイツ・イクスピアリエンスト・プレイアーズ・フラムビギナーズ
