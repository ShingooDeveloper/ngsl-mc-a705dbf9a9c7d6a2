buy

動詞/baɪ/CEFR A1頻度#241
  1. 【動詞】金銭を払って物を得る・購入する（I need to buy diamonds from the villager.）
  2. 【動詞】商人との交易で物を得る（You can buy emeralds at the village.）
  3. 【動詞】（比喩的に）時間・利益などを得る・確保する（We bought enough time before the Wither appeared.）

⛏️ マインクラフト例文

⛏I'll buy some iron ingots from the trader.

商人から鉄インゴットをいくつか買おう。

/aɪl ˈbaɪ səm ˈaɪrən ˈɪŋɡəts frəm ðə ˈtreɪdər/
  • some iron → some の /m/ が iron の母音 /aɪ/ に連結 [səm ˈaɪrən] サムアイロン
アイル バイ サムアイロン イングッツ フラム ザ トレイダー

⛏We bought a lot of wood for building.

建築用に大量の木をまとめて買った。

/wiː ˈbɔt ə ˈlɑt əv ˈwʊd fər ˈbɪldɪŋ/
  • bought a → bought の /t/ が /ə/ に挟まれ、フラップ化して連結 [bɔɾə] ボラ
  • lot of → lot の /t/ が /ə/ に挟まれフラップ化、of は弱形 /əv/ [lɑɾəv] ラーラヴ
  • for building → for は弱形 /fər/ で [fər ˈbɪldɪŋ] ファー ビルディング
ウィー ボラ ラーラヴ ウッド ファー ビルディング

⛏Can you buy me some stone blocks?

石ブロックをいくつか買ってくれない？

/kæn ju ˈbaɪ mi səm ˈstoʊn ˈblɑks/
  • Can you → can は質問形で、weak form /kən/ になることもある [kən ju]
  • some stone → some の /m/ と stone の /s/ は子音が連続するため、リンキングは起こらない
キャン ユー バイ ミー サム ストーン ブロックス

💬 よく使う熟語（マイクラ例文）

buy time時間を稼ぐ・遅延させる・準備時間を得る

⛏We need to buy time before the Wither arrives.

ウィザーが来る前に時間を稼ぐ必要があるね。

/wiː ˈniːd tə ˈbaɪ ˈtaɪm bɪˈfɔr ðə ˈwɪðər əˈraɪvz/
  • need to → need の /d/ が to の /t/ に続き、未開放 /d̚/ で接続 [niːd̚ tə]
  • the Wither → the は弱形 /ðə/ で [ðə ˈwɪðər] ザウィザー
ウィー ニード タ バイ タイム ビフォー ザ ウィザー アライズ
