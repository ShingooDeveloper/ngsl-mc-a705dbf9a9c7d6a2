anywhere

副詞/ˈɛnɪwɛɹ/CEFR A2頻度#1645
  1. 【副詞】「どこか」「どこかに」（不特定の場所を示す、…in some place）
  2. 【副詞】「どこへ」「どこへでも」（方向を示す、…to any place）
  3. 【副詞】「どこでも」（全ての場所で成立する、…in all places）

⛏️ マインクラフト例文

⛏I can build anywhere in this world.

このワールドではどこでも建築できます。

/aɪ kən ˈbɪld ˈɛnɪwɛɹ ɪn ðɪs wɜːld/
  • build の語末 /d/ が anywhere の /ɛ/ に連結 [bɪldˈɛnɪwɛɹ] ビルデニウェア
アイ カン ビルデニウェア イン ディス ワールド

⛏You can get anywhere with enough supplies.

十分な物資があればどこへでも進められます。

/juː kən ˈɡɛt ˈɛnɪwɛɹ wɪð ɪˈnʌf səˈplaɪz/
  • can の弱形 /kən/
  • get の語末 /t/ が anywhere の /ɛ/ 前でフラップ化 [ɡɛɾɛnɪwɛɹ] ゲレニウェア
ユー カン ゲレニウェア ウィズ イナフ サプライズ

⛏Mobs don't spawn anywhere if you're on peaceful mode.

ピースフルモードにしているとモブはどこにも出現しません。

/moʊbz doʊnt spɔːn ˈɛnɪwɛɹ ɪf juːr ɑn ˈpiːsfəl moʊd/
  • spawn の語末 /n/ が anywhere の /ɛ/ に連結 [spɔːnˈɛnɪwɛɹ] スポーネニウェア
モウズ ドウント スポーネニウェア イフ ユアー オン ピースフル モウド
