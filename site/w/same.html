same

形容詞・代名詞/seɪm/CEFR A1頻度#121
  1. 【形容詞】同じ、一致した（他のものと変わらない）
  2. 【代名詞】同じもの、同じこと（…the same）
  3. 【形容詞】相変わらずの、いつもの（…the same old）
  4. 【副詞】同様に、同じく（稀な用法）

⛏️ マインクラフト例文

⛏I built the same house as you.

私はあなたと同じ家を建てました。

/aɪ ˈbɪlt ðə ˈseɪm ˈhaʊs æz ju/
  • built の /t/ が the の /ə/ に未開放で続く [bɪl̚ðə]。the は弱形 /ðə/
  • as は弱形 /əz/。you に続く
アイ ビルザ セーム ハウス エズ ユー

⛏These two swords are the same.

この2本の剣は同じです。

/ðiz tu sɔrdz ɑr ðə ˈseɪm/
  • are の /r/ が the の /ə/ に連結 [ɑrðə]。「アーザ」
  • the は弱形。same に先行する
ディーズ トゥー ソーズ アーザ セーム

⛏My biome has the same weather.

私のバイオームは同じ天気です。

/maɪ ˈbaɪoʊm hæz ðə ˈseɪm ˈwɛðər/
  • has の /z/ が the の /ə/ に続く [ˈhæzðə]。「ハズザ」
  • the は弱形。same に先行する
マイ バイオーム ハズザ セーム ウェザー

💬 よく使う熟語（マイクラ例文）

the same同じもの、同じこと

⛏We want to build the same structure.

同じ構造を建てたいんだ。

/wi ˈwɑnt tə ˈbɪld ðə ˈseɪm ˈstrʌktʃər/
  • want の /t/ + to の弱形 /tə/。want の /t/ が未開放で to の /ə/ に続く [ˈwɑn̚tə]
  • to は弱形のまま build に続く

at the same time同時に、一方で

⛏I can mine and craft at the same time.

同時に採掘してクラフトできます。

/aɪ kən ˈmaɪn ænd kræft æt ðə ˈseɪm taɪm/
  • can は弱形 /kən/。mine に続く
  • and の /d/ が craft の /k/ に未開放で続く [ən̚ˈkræft]
  • at の /t/ が the の /ə/ に未開放で続く [æt̚ðə]
