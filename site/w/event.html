event

名詞/ɪˈvɛnt/CEFR B1頻度#441
  1. 【名詞】起こること、出来事、事象（テニスの試合はスポーツイベントの一例です）
  2. 【名詞】計画された公式の行事、催し物（オリンピックは世界最大級のイベントです）
  3. 【名詞】特に注目される重要な出来事（戦闘はマインクラフトにおける主要なイベント）

⛏️ マインクラフト例文

⛏I missed the village event last night.

昨夜の村のイベントを逃してしまった。

/aɪ ˈmɪst ðə ˈvɪlɪdʒ ɪˈvɛnt læst̚ naɪt/
  • 「the」は弱形 /ðə/
  • 「village event」で village 語末の子音 /dʒ/ が event 初頭の母音 /ɪ/ に連結
  • 「last」の語末 /t/ は未開放 [læst̚]
アイ ミスト ザ ヴィリジ イヴェント ラスト ナイト

⛏The server hosts a PvP event every weekend.

そのサーバーは毎週末にPvPイベントを開催しています。

/ðə ˈsɝvər ˈhoʊsts ə ˈpi ˈvi ˈpi ɪˈvɛnt ˈɛvɚi ˈwikɛnd/
  • 「the」は弱形 /ðə/
  • 「a」は弱形 /ə/
  • 「every」は /ˈɛvɚi/ で r-colored
ザ サーヴァー ホースツ ア ピー ヴィー ピー イヴェント エヴリー ウィークエンド

⛏This event will give us free diamonds.

このイベントは私たちに無料のダイヤを与えるだろう。

/ðɪs ɪˈvɛnt wəl ˈɡɪv əs ˈfri ˈdaɪmənz/
  • 「will」は助動詞で弱化 /wəl/
  • 「us」は弱形 /əs/

We host special building events every weekend on our creative server.

私たちのクリエイティブサーバーでは毎週末特別な建築イベントを開催しています。

/wi ˈhoʊst ˈspɛʃəl ˈbɪldɪŋ ɪˈvɛnts ˈɛvri ˈwiːkend ɑn ˈaʊɚ kriˈeɪɾɪv ˈsɜːrvɚ/

⛏In the event of a creeper attack, hide in the bunker.

クリーパーの攻撃が起きた場合、バンカーに隠れてください。

/ɪn ðə ɪˈvɛnt əv ə ˈkripər əˈtæk ˈhaɪd ɪn ðə ˈbʌŋkər/
