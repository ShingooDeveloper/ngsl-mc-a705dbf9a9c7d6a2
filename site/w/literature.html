literature

名詞/ˈlɪt.ə.rə.tʃɚ/CEFR B1頻度#1516
  1. 【名詞】文学。（小説・詩・劇作などの文学作品の総体。...literature as an art form...）
  2. 【名詞】文献・著作物集。（ある主題に関する記述や研究・知識。...relevant literature on a subject...）
  3. 【名詞】読み物・テキスト。（情報提供や教育的な出版物・記録。...practical or instructional literature...）

⛏️ マインクラフト例文

⛏I found some literature about building more efficient farms.

より効率的な農場建築についての文献を見つけた。

/aɪ ˈfaʊnd səm ˈlɪt.ə.rə.tʃɚ əˈbaʊt ˈbɪl.dɪŋ moʊr ɪˈfɪ.ʃənt ˈfɑrmz/
  • found の語末 /d/ が some の /s/ に先行する [ˈfaʊnd səm] ファウンド・サム
  • about の語末 /t/ は未開放で building に先行する [ˈbaʊt̚ ˈbɪl] アバウ・ビル
アイ・ファウンド・サム・リテラチャー・アバウ・ビルディング・モア・イフィシェント・ファームズ

⛏Need some literature about building better defenses?

より良い防御建築についての文献が必要ですか？

/niːd səm ˈlɪt.ə.rə.tʃɚ əˈbaʊt ˈbɪl.dɪŋ ˈbɛt.ɚ dɪˈfɛn.sɪz/
  • need の語末 /d/ が未開放で some に先行する [ˈniːd̚ səm] ニード・サム
  • about の語末 /t/ が未開放で building の頭 /b/ に先行する [ˈbaʊt̚ ˈbɪl] アバウ・ビル
ニード・サム・リテラチャー・アバウ・ビルディング・ベター・ディフェンシーズ

⛏Take advantage of this literature to learn about ore mining techniques.

この文献を活かして鉱石採掘技術について学んでください。

/ˈteɪk ədˈvæn.tɪj əv ðɪs ˈlɪt.ə.rə.tʃɚ tə ˈlɝn əˈbaʊt ˈɔr ˈmaɪ.nɪŋ tɛkˈniks/
  • Take の語末子音 /k/ が advantage の頭の弱形母音 /ə/ に連結し、[ˈteɪ.kə] テイカ
  • about の語末 /t/ が未開放で ore の頭 /ɔ/ に先行する [ˈbaʊt̚ ˈɔr] アバウ・オア
テイク・アドヴァンティジ・アヴ・ディス・リテラチャー・タ・ラーン・アバウ・オア・マイニング・テクニークス

💬 よく使う熟語（マイクラ例文）

literature on...についての文献（または著作物・記録）

⛏All the literature on enchanting methods is stored in the main library.

エンチャント方法についてのすべての文献は大図書館に保管されている。

/ɔl ðə ˈlɪt.ə.rə.tʃɚ ɑn ɪnˈtʃænt.ɪŋ ˈmɛθ.ədz ɪz ˈstɔrd ɪn ðə ˈmeɪn ˈlaɪ.brer.i/
  • literature の語末 /ɚ/ と on の /ɑn/ (弱形) の間では連結は起きない
  • stored in の /d/ が /ɪ/ に連結し、[ˈstɔrd ɪn] ストアード・イン
オール・ザ・リテラチャー・オン・イントシャント・メサデス・イズ・ストアード・イン・ザ・メイン・ライブラリー
