repeat

動詞・名詞/rɪˈpiːt/CEFR A2頻度#1250
  1. 【動詞】何度も何度もやる、繰り返す（do something again）
  2. 【動詞】誰かが言ったことをもう一度言う、復唱する
  3. 【名詞】繰り返される事柄、パターン
  4. 【名詞】（ゲーム・音楽）リピート機能、繰り返し再生

⛏️ マインクラフト例文

⛏I repeat it every day.

毎日それを繰り返します。

/aɪ rɪˈpiːɾɪt ˈɛvɚi deɪ/
  • repeat it → /t/ が母音に挟まれてフラップ化し連結 [rɪˈpiːɾɪt] リピーリット
アイ リピーリット エブリ デイ

⛏We need to repeat the mining run.

採掘ツアーをもう一度繰り返す必要があります。

/wi niːd̚ tə rɪˈpiːt̚ ðə ˈmaɪnɪŋ rʌn/
  • need to → /d/ が未開放で次の /t/ に接続 [niːd̚ tə] ニードゥ
  • to → 機能語は弱形 /tə/
  • repeat the → /t/ 未開放、/ðə/ に接続 [rɪˈpiːt̚ ðə] リピートザ
  • the → 弱形 /ðə/
ウィ ニード タ リピート ザ マイニング ラン

⛏This command repeats the signal.

このコマンドが信号を繰り返します。

/ðɪs ˈkɑmænd rɪˈpiːts ðə ˈsɪɡnəl/
  • command repeats → /d/ が /r/ の前で未開放 [ˈkɑmænd rɪˈpiːts] コマンド リピーツ
  • repeats the → /s/（-s 語尾）が /ð/ に連結 [rɪˈpiːts ðə] リピーツ ザ
ディス コマンド リピーツ ザ シグナル

💬 よく使う熟語（マイクラ例文）

repeat after～の後に繰り返す、～に続いて言う

⛏Repeat after me: that's how you place blocks correctly.

僕の後に言ってください。それが正しいブロック配置の方法です。

/rɪˈpiːt ˈæftɚ mi ðæts haʊ ju pleɪs blɑks ˈkɛrɪkɾli/
  • repeat after → /t/ が /æ/ に連結 [rɪˈpiːt ˈæftɚ]
  • that's → /z/ が母音に連結 [ðæts]
  • correctly → /t/ が母音に挟まれてフラップ化 /ˈkɛrɪkɾli/ コレクリー

⛏This music disc is on repeat in my base.
