require

動詞/rɪˈkwaɪr/CEFR B1頻度#401
  1. 【動詞】～を必要とする、～がなくてはならない（…to be essential）
  2. ～を要求する、～することを要求する
  3. （ルール・規則・仕様が）～を義務付ける、～を要する

⛏️ マインクラフト例文

⛏This recipe requires diamonds.

このレシピはダイヤモンドが必要だ。

/ðɪs ˈrɛsəpi rɪˈkwaɪrz ˈdaɪmənz/
  • requires の語末 /z/ と diamonds の語頭 /d/ は連結しない（子音同士）

⛏Building in Survival mode requires planning.

サバイバルモードで建築するには計画が必要だ。

/ˈbɪldɪŋ ɪn sərˈvaɪvəl moʊd rɪˈkwaɪrz ˈplænɪŋ/
  • Building の語末 /ŋ/ が in の語頭 /ɪ/ に連結
  • mode の語末 /d/ は未開放で requires に続く
  • requires の語末 /z/ と planning の語頭 /p/ は連結しない（子音同士）

  • obsidian の語末 /n/ と blocks の語頭 /b/ は連結しない（子音同士）
