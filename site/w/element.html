element

名詞/ˈɛləmənt/CEFR B1頻度#1027
  1. 【名詞】構成する基本的な部分。素材や成分（...を構成する）要素
  2. 【名詞】必須の要因。（...に欠かせない）成分、要素
  3. 【名詞】自然界の根本的な構成要素（earth, water など）

⛏️ マインクラフト例文

⛏Stone is a key element for all structures.

石はすべての建造物の重要な要素だ。

/stoʊn ɪz ə ˈkiː ˈɛləmənt fɔr ɔl ˈstrʌktʃɚz/
  • is /z/ が a /ə/ に連結 [zə]
  • for は機能語で弱形 /fɚ/ に弱化
ストウン イズ ア キー エレメント ファー オール ストラクチャーズ

⛏Water is an essential element in survival mode.

水はサバイバルモードで本質的な要素である。

/ˈwɔtɚ ɪz ən əˈsɛnʃəl ˈɛləmənt ɪn sɚˈvaɪvəl ˈmoʊd/
  • water の /t/ が母音に挟まれてフラップT化 [ˈwɔɾɚ]→「ウォーラー」
  • is /z/ が an /ən/ に連結 [zən]
  • in /n/ と survival /s/ は子音+子音で gap
ウォーラー イズ アン エッセンシャル エレメント イン サヴァイヴァル モウド

⛏Each element of the redstone circuit works perfectly.

レッドストーン回路の各要素は完璧に動作する。

/itʃ ˈɛləmənt əv ðə ˈrɛdˌstoʊn ˈsɝkət ˈwɝks ˈpɝfɪktli/
  • element /t/ が of /ə/ に連結 [tə]
  • the は機能語で弱形 /ðə/
  • circuit /t/ と works /w/ は子音+子音で gap
イッチ エレメント アヴ ザ レッドストウン サーキット ワークス パーフィクトリー
