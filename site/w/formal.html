formal

形容詞/ˈfɔrml̩/CEFR B2頻度#1697
  1. 【形容詞】公式の、正式な（…official or authorized…）
  2. 【形容詞】礼装用の、フォーマルな（…suited for formal occasions…）
  3. 【形容詞】かしこまった、よそよしい（態度や言葉が）
  4. 【形容詞】正規の、公式に認められた（…officially recognized…）

⛏️ マインクラフト例文

⛏You need a formal set of enchanted tools for deep mining.

深い採掘には正式に整えたエンチャント道具が必要だ。

/ju ˈniːdə ˈfɔrml̩ sɛt əv ɪnˈtʃæntɪd tulz fɔr dip ˈmaɪnɪŋ/

  • set の語末子音 /t/ が of の /ə/（弱形）に連結 [ˈsɛtəv] セッタヴ
