send

動詞/sɛnd/CEFR A1頻度#322
  1. 【動詞】～を送る、メールやメッセージで送信する（…に）
  2. 【動詞】～を派遣する、送り出す（…へ）
  3. 【動詞】（信号・メッセージなどを）出す、発信する

⛏️ マインクラフト例文

⛏I sent my friend the location of my base.

友人に基地の場所を送りました。

/aɪ ˈsɛnt maɪ frɛnd ðə loʊˈkeɪʃən əv maɪ ˈbeɪs/
  • sent my → sent の /t/ が未開放で my の /m/ に接続[t̚ m]「セント マイ」
  • location の /ən/ が of の弱形 /əv/ に続く[ʃən əv] シャン アヴ
  • of my → /v/ が /m/ に続く[əv maɪ] アヴ マイ
アイ セント マイ フレンド ザ ロウケイション アヴ マイ ベイス

⛏Can you send me some redstone?

レッドストーンを送ってもらえますか？

/kæn juː ˈsɛnd mi səm ˈrɛdstoʊn/
  • send me → send の /d/ が me の /i/ に連結[d i] → [di] センディー
  • some redstone → some の /m/ が redstone の /r/ に続く[əm ˈrɛdstoʊn] サム レッドストウン
キャン ユー センディー サム レッドストウン

⛏The Ender Dragon sent us flying back to spawn.

エンダードラゴンは私たちを飛ばしてスポーン地点に送り返しました。

/ðə ˈɛndər ˈdræɡən sɛnt əs ˈflaɪɪŋ bæk tə ˈspoʊn/
  • sent us → sent の /t/ が未開放で us の /ə/ に接続[t̚ ə] センタス
  • flying back → flying の /ŋ/ が back の /b/ に続く。特に変化なし[ŋ b] フライング バック
  • back to → to の弱形 /tə/[bæk tə] バック タ
ザ エンダー ドラゴン セント ス フライング バック タ スポウン

💬 よく使う熟語（マイクラ例文）

send out配信する、送り出す（情報やメッセージを広く配信する）

⛏We sent out invitations to all the players on the server.

サーバーのすべてのプレイヤーに招待を送り出しました。

/wiː sɛnt aʊt ɪnˈvɪteɪʃənz tə ɔl ðə ˈpleɪərz ɑn ðə ˈsɜrvər/
  • sent out → sent の /t/ が未開放で out の /aʊ/ に接続[t̚ aʊ] センタウト
  • to all → to の弱形 /tə/、all の /ɔ/ に続く[tə ɔl] タ オール
  • on the → on の /n/ が the の /ð/ に、the は弱形 /ðə/[ɑn ðə] オン ザ
ウィー センタウト インヴィテイションズ タ オール ザ プレイヤーズ オン ザ サーヴァー
send back返す、送り返す（借りたものを返すなど）

⛏I sent back the pickaxes you lent me.

貸してくれたツルハシを送り返しました。

/aɪ sɛnt bæk ðə ˈpɪkˌæksɪz juː lɛnt mi/
  • sent back → sent の /t/ が未開放で back の /b/ に接続[t̚ b] セント バック
  • back の /k/ が the の弱形 /ðə/ に続く[bæk ðə] バック ザ
  • lent me → lent の /t/ が未開放で me の /m/ に接続[t̚ m] レント ミー
アイ セント バック ザ ピックアックシズ ユー レント ミー
send away追い出す、去らせる（不要な人や物を遠ざける）

⛏The ghasts send us away from the Nether portal.

ガストはネザーポータルから私たちを追い出します。

/ðə ɡæsts sɛnd əs əˈweɪ frɑm ðə ˈnɛðər ˈpoʊrtəl/
  • send us → send の /d/ が us の /ə/ に連結[d ə] センダス
  • away の /eɪ/ に from の /f/ が続く。特に変化なし[əˈweɪ frɑm] ァウェイ フラム
  • from the → from の /m/ が the の /ðə/ に。the は弱形[frəm ðə] フラム ザ
ザ ギャスツ センダス ァウェイ フラム ザ ネザー ポータル
