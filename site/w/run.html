run

動詞・名詞/rʌn/CEFR A1頻度#191
  1. 【動詞】速く移動する、走る（足を使って素早く動く）
  2. 【動詞】（機械やプログラムを）動かす、実行する、作動させる
  3. 【動詞】流れる、漏れ出す（液体などが流れ落ちる）
  4. 【動詞】伸びる、続く（距離や時間にわたって）
  5. 【名詞】走行、走ること；一連の連続した出来事

⛏️ マインクラフト例文

⛏I need to run and find wood.

走って木を見つけなければなりません。

[aɪ ˈniːdə ˈrʌn ənd ˈfaɪnd ˈwʊd]
  • need to: need の /d/ が to の弱形 /tə/ に連結 [ˈniːdə]『ニーダ』
  • and: 弱形 /ənd/ or /ən/『アン』
アイ ニーダ ラン アン ファインド ウッド

⛏Water runs through the cave.

水が洞窟を通って流れています。

[ˈwɔɾər ˈrʌnz θruː ðə ˈkeɪv]
  • water: 米式でフラップT [ɾ]『ウォーラー』
  • the: 弱形 /ðə/『ザ』

⛏We all ran out of stone.

[wi ˈɔl ˈræn ˈaʊɾəv ˈstoʊn]
  • out of: out の /t/ がフラップ化 [ɾ]、of は弱形 /əv/、連結 [ˈaʊɾəv]『アウラヴ』
  • of: 弱形 /əv/『アヴ』
