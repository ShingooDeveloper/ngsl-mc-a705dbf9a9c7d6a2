association

名詞/ˌæsoʊʃiˈeɪʃən/CEFR B1頻度#1592
  1. 【名詞】結社、団体、協会（共通の目的を持つ人々の組織）
  2. 【名詞】つながり、関係（2つ以上のもの間の結びつき）
  3. 【名詞】連想、連想作用（別の事柄を思い出させる心理的なつながり）

⛏️ マインクラフト例文

⛏The Builders' Association has strict safety rules for construction projects.

ビルダー協会は建築プロジェクトに厳しい安全規則を持っている。

/ðə ˈbɪldərz əˌsoʊsiˈeɪʃən hæz strɪkt ˈseɪfti ˈrulz fɔr kənˈstrʌkʃən ˈprɑdʒɛkts/
  • Builders' Association : 語末子音 /z/ が /ə/ に連結
  • for construction : for は弱形 /fər/、語末 r と語頭 k は clean contact
  • has strict : /z/ と /s/ は close contact で連結なし
ザ ビルダーズ アソーシエーション ハズ ストリクト セイフティ ルーズ ファ コンストラクション プロジェクツ

⛏There's a strong association between mining diamonds and finding rare treasures.

ダイヤ採掘と稀なお宝探しの間には強い結びつきがある。

/ðɛrz ə strɔŋ əˌsoʊsiˈeɪʃən bɪˈtwin ˈmaɪnɪŋ ˈdaɪməndz ənd ˈfaɪndɪŋ rɛr ˈtrɛʒərz/
  • There's a : 語末 /z/ が /ə/ に連結
  • mining diamonds : /ŋ/ と /d/ は clean contact で連結なし
  • rare treasures : /r/ と /t/ は clean contact で連結なし
ゼアーズ ア ストロング アソーシエーション ビットウィーン マイニング ダイマンズ ン ファインディング レア トレジャーズ

⛏Players made an association between red wool and the Nether dimension.

プレイヤーたちは赤いウールとネザー次元の間に関連性を作った。

/ˈpleɪərz meɪd ən əˌsoʊsiˈeɪʃən bɪˈtwin rɛd wʊl ənd ðə ˈnɛðər dɪˈmɛnʃən/
  • made an : 語末 /d/ が /ə/ に連結 [ˈmeɪd ən] →「メイダン」
  • between red : /n/ と /r/ は clean contact で連結なし
  • the Nether : the は弱形 /ðə/、/ə/ と /n/ は連結なし
プレイヤーズ メイダン アソーシエーション ビットウィーン レッド ウール ン ザ ネザー ディメンション

💬 よく使う熟語（マイクラ例文）

⛏By association with the village, I can trade with the locals.

村との関係で、私はローカルと取引できる。

/baɪ əˌsoʊsiˈeɪʃən wɪð ðə ˈvɪlɪdʒ aɪ kən ˈtreɪd wɪð ðə ˈloʊkəlz/
  • by association : /n/ と /w/ は clean contact で連結なし
  • with the : /ð/ と /ð/ は clean contact、the は弱形 /ðə/
  • can trade : can は弱形 /kən/、/n/ と /t/ は clean contact
