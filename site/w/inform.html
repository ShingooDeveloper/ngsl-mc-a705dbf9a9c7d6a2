inform

動詞/ɪnˈfɔːrm/CEFR B1頻度#1067
  1. 【動詞】（人に）知らせる、通知する
  2. 【動詞】（…について）情報を与える、伝える
  3. 【動詞】～のことを通達する

⛏️ マインクラフト例文

⛏I need to inform the other players about the hidden diamonds.

他のプレイヤーに隠されたダイヤモンドについて知らせる必要がある

/aɪ ˈniːd tə ɪnˈfɔːrm ðə ˈʌðə ˈpleɪərz əˈbaʊt ðə ˈhɪdn̩ ˈdaɪməndz/
  • players の /z/ が about の /ə/ に連結 [ˈpleɪərzəˈbaʊt]
アイ ニード タ インフォーム ザ アザー プレイヤーズ アバウト ザ ヒドゥン ダイモンズ

⛏I informed my friend about the ender dragon threat.

エンダードラゴンの脅威について友人に知らせた

/aɪ ɪnˈfɔːrmd maɪ ˈfrɛnd əˈbaʊt ðə ˈɛndə ˈdræɡən θrɛt/
  • friend の /d/ が about の /ə/ に連結 [frɛndəˈbaʊt]
アイ インフォームド マイ フレンダバウト ザ エンダー ドラゴン スレット

⛏The server is informing everyone about the new Nether portal location.

サーバーがすべての人に新しいネザーポータルの位置について知らせている

/ðə ˈsɜːrvər ɪz ɪnˈfɔːrmɪŋ ˈɛvriwʌn əˈbaʊt ðə ˈnjuː ˈnɛðə ˈpoʊɾəl ˌloʊˈkeɪʃən/
  • server の /r/ が is の /ɪ/ に連結 [ˈsɜːrvərɪz]
  • is の /z/ が informing の /ɪ/ に連結 [ɪzɪnˈfɔːrmɪŋ]
ザ サーバー イズ インフォーミング エブリワン アバウト ザ ニュー ネザー ポーラル ロウケイション

💬 よく使う熟語（マイクラ例文）

be informed of～について知らせられる、通知される

⛏All players must be informed of the server rules.

すべてのプレイヤーはサーバールールについて知らせられる必要がある

/ɔːl ˈpleɪərz mʌst bi ɪnˈfɔːrmd əv ðə ˈsɜːrvər ˈrulz/
  • informed の /d/ が of の /ə/ に連結 [ɪnˈfɔːrmdəv]
オール プレイヤーズ マスト ビー インフォームダヴ ザ サーバー ルールズ
