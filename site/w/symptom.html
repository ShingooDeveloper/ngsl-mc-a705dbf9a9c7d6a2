symptom

名詞/ˈsɪmptəm/CEFR B2頻度#2068
  1. 【名詞】病気やけが時に見られる身体的な印、徴候（shows flu symptoms）
  2. 【名詞】何か悪いことが起きている、または起きようとしていることの兆候（a symptom of something wrong）
  3. 【名詞】社会や状況の悪化や問題を示す表れ（symptoms of decline）

⛏️ マインクラフト例文

⛏The wither effect is a symptom that a wither mob is nearby.

ウィザーのダメージエフェクトは、ウィザーモブが近くにいる兆候だ。

[ðə ˈwɪðər ɪˈfɛkt ɪz ə ˈsɪmptəm ðæt ə ˈwɪðər mɑb ɪz ˈnɪrbaɪ]
  • is の語末 /z/ が a（弱形 /ə/）に連結 [ɪzə]「イザ」
  • that の語末 /t/ が a の /ə/ に未開放で続く [ðæt ə]「ザッア」
ザ ウィザー イフェクト イザ シンプトム ザッア ウィザー モブ イズ ニアバイ

⛏Lag spikes indicate that the server is overwhelmed.

ラグスパイクは、サーバーが過負荷状態にあることを示す兆候だ。

[læg spaɪks ˈɪndɪkeɪt ðæt ðə ˈsɜrvər ɪz ˌoʊvərˈwɛlmd]
  • that の語末 /t/ が the（弱形 /ðə/）に未開放で続く [ðæt ðə]「ザッザ」
  • is の語末 /z/ が overwhelmed の /oʊ/ に連結 [ɪzoʊvərˈwɛlmd]「イゾーバーウェルムド」
ラーグ スパイクス インディケイト ザッザ サーバー イゾーバーウェルムド

⛏Damaged stone blocks are a symptom of underground cave systems.

傷ついた石ブロックは、地下洞窟システムが存在することの兆候だ。

[ˈdæmɪdʒd stoʊn blɑks ɑr ə ˈsɪmptəm əv ˌʌndərˈɡraʊnd keɪv ˈsɪstəmz]
  • are の語末 /r/ が a（弱形 /ə/）に連結 [ɑrə]「アーラ」
  • of の語末 /v/ が underground の /ʌ/ に連結 [əv‿ʌndərˈɡraʊnd]「ァヴアンダーグラウンド」
ダーミッジド ストウン ブラックス アーラ シンプトム ァヴアンダーグラウンド ケイヴ シスタムズ

💬 よく使う熟語（マイクラ例文）

show symptoms of～の症状を示す、～の兆候を見せる

⛏The server shows symptoms of overload.

サーバーは過負荷の症状を示している。

[ðə ˈsɜrvər ʃoʊz ˈsɪmptəmz əv ˌoʊvərˈloʊd]
  • the の弱形 /ðə/ が server へ続く [ðə ˈsɜrvər]「ザ サーバー」
  • of の語末 /v/ が overload の /oʊ/ に連結 [əv‿oʊvərˈloʊd]「ァヴォーバーロード」
ザ サーバー ショウズ シンプトムズ ァヴォーバーロード
