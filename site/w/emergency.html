emergency

名詞/ɪˈmɝːdʒənsi/CEFR B1頻度#1888
  1. 【名詞】緊急事態、予期しない危機的状況。直ちに対応が必要な状況（unexpected serious situation requiring immediate action）
  2. 【名詞】（形容詞的用法）非常時用、緊急用。他の名詞を修飾して使う（used before a noun, as in 'emergency exit' or 'emergency room'）

⛏️ マインクラフト例文

⛏There is an emergency in the village—creepers are attacking the houses!

村で緊急事態が発生した—クリーパーが家を攻撃している！

/ðɛr ɪz ən ɪˈmɝːdʒənsi ɪn ðə ˈvɪlɪdʒ—ˈkripərz ɑr əˈtækɪŋ ðə ˈhaʊzɪz/
  • is an emergency → is の /z/ が an /ən/（弱形）に連結。an の /n/ が emergency の /ɪ/ に連結し [ɪzənɪˈmɝːdʒənsi]
  • in the village → in の /n/ が the の弱形 /ðə/ に連結 [ɪnðə]
  • are attacking → are の /r/ が attacking の /ə/ に連結 [ɑrə]
ザー イズアン イマージェンスィ イン ザ ヴィリッジ—クリーパーズ アラ タッキング ザ ハウジズ

⛏In case of emergency, make sure you have enough food and tools ready.

緊急時に備えて、十分な食料と道具を用意してください。

/ɪn ˈkeɪs əv ɪˈmɝːdʒənsi, meɪk ʃɔr ju hæv ɪˈnʌf fud ənd tulz ˈrɛdi/
  • of emergency → of の弱形 /əv/ が emergency の /ɪ/ に連結 [əvɪˈmɝː]
  • you have → you の /u/ に have が続き、have は弱形 /əv/ に弱化。[juəv]
  • food and tools → food の /d/ が and の弱形 /ən/ に連結し [fudən]
イン ケース アヴ イマージェンスィ、メイク シュアー ユーアヴ イナフ フード アン トゥールズ レディ

⛏If there is an emergency, use that nether portal I built last night.

もし緊急事態が起きたら、昨夜作ったそのネザーポータルを使ってください。

/ɪf ðɛr ɪz ən ɪˈmɝːdʒənsi, juz ðæt ˈnɛðər ˈpoʊrtəl aɪ bɪlt læst naɪt/
  • is an emergency → is の /z/ が an の弱形 /ən/ に連結。an の /n/ が emergency の /ɪ/ に連結
  • use that → use の /z/ に that の /ðæt/ が続く。通常 /ðæt/ だが、unstressed では /ðət/ に弱化する場合もある
  • built last → built の /t/ は未開放 [bɪl̚t]。built と last の間で閉塞の感覚 [bɪl̚t læst]

⛏I designed an emergency exit using water and vines on the side of the base.

基地の側面に水とつるを使った緊急脱出口を設計しました。

/aɪ dɪˈzaɪnd ən ɪˈmɝːdʒənsi ˈɛɡzɪt ˈjuːzɪŋ ˈwɔtər ənd vaɪnz ɑn ðə saɪd əv ðə beɪs/

  • using water → using の /ŋ/ に water の /w/ が続く。water の /t/ はフラップ化 [ˈwɔɾər]
  • on the side → on の /n/ が the の弱形 /ðə/ に連結。side の /d/ が of の弱形 /əv/ に連結 [saɪdəv]
