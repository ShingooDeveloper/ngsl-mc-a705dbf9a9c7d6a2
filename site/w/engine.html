engine

名詞・動詞/ˈɛndʒən/CEFR B1頻度#1614
  1. 【名詞】動力装置、機関。ピストンエンジンなど自動車やレッドストーンメカニズムを動かす装置。
  2. 【名詞】原動力、推進力。比喩的用法。（creativity is the engine of growth）
  3. 【名詞】（コンピュータ）エンジン、処理プログラム。（search engine）など。
  4. 【動詞】〜に動力を与える。（to engine a device）稀な用法。

⛏️ マインクラフト例文

⛏I need an engine for the elevator.

エレベーターのためにエンジンが必要だ。

/aɪ ˈnid ən ˈɛndʒən fɔr ðə ˈɛləˌveɪtɚ/
  • need の語末 /d/ が an の /ə/ に連結 → [ˈniːdən]「ニーダン」
  • an の /n/ が engine の /ɛ/ に連結
アイ ニーダン エンジン フォー ザ エレヴェイター

⛏The piston engine is very powerful.

ピストンエンジンはとても強力だ。
