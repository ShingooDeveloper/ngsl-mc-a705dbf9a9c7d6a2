since

接続詞・前置詞/sɪns/CEFR A1頻度#190
  1. 【接続詞】…以来。（…since I started のように、ある時点からの経過時間や継続を表す）
  2. 【接続詞】…ので、…だから。（原因・理由を示す）
  3. 【前置詞】…以来、…から。（時間軸上の起点を示す；…since the update など）
  4. 【副詞】あの時以来、ずっと。（…ever since のような継続を示す）

⛏️ マインクラフト例文

⛏I need a pickaxe since I'm going deep mining.

深掘りをするから、つるはしが必要だ。

/aɪ ˈniːd ə ˈpɪkˌæks sɪns aɪm ˈɡoʊɪŋ diːp ˈmaɪnɪŋ/
  • need a：need /niːd/ 語末 /d/ が a /ə/ に連結 [ˈniːdə]→ニーダ
  • a pickaxe：a /ə/ が pickaxe /ˈpɪkˌæks/ の /p/ に連結 [əˈpɪkˌæks]→アピックアックス
  • since I'm：since /sɪns/ 語末の /s/ が I'm の /aɪ/ に連結し /z/ に有声化 [sɪnzaɪm]→シンザイム
アイニーダピックアックスシンザイムゴーイングディープマイニング

⛏Ever since I found diamonds, I've been happy.

ダイヤモンドを見つけて以来、ずっと幸せだ。

/ˈɛvər sɪns aɪ ˈfaʊnd ˈdaɪəmənz aɪv bɪn ˈhæpi/
  • since I：since /sɪns/ 語末の /s/ が I /aɪ/ に連結し /z/ に有声化 [sɪnzaɪ]→シンザイ
  • I've been：I've /aɪv/ 語末 /v/ と been /bɪn/ 頭 /b/ は子音+子音で連結なし
エバシンザイファウンダイアモンズアイヴビンハッピー

⛏Since the server updated, we moved our base.

サーバーがアップデートされて以来、拠点を移動させた。

/sɪns ðə ˈsɜːvər əpˈdeɪtɪd wi ˈmuːvd ˈaʊər ˈbeɪs/
  • the server：the /ðə/ が server /ˈsɜːvər/ の /s/ に連結 [ðəˈsɜːvər]→ザサーバー
  • moved our：moved /muːvd/ 語末 /d/ が our /ˈaʊər/ の /aʊ/ に連結 [muːvˈdaʊər]→ムーヴダワー
シンザサーバーアップデイティドウィムーヴダワーベイス

💬 よく使う熟語（マイクラ例文）

ever sinceあの時以来、ずっと

⛏I found a cave, and I've been exploring ever since.

洞窟を見つけて以来、ずっと探検を続けている。

/aɪ ˈfaʊnd ə ˈkeɪv ənd aɪv bɪn ɪkˈsplɔːrɪŋ ˈɛvər sɪns/
  • found a：found /faʊnd/ 語末 /d/ が a /ə/ に連結 [ˈfaʊndə]→ファウンダ
  • a cave：a /ə/ が cave /ˈkeɪv/ の /k/ に連結 [əˈkeɪv]→アケイヴ
  • since I：since /sɪns/ 語末の /s/ が I /aɪ/ に連結し /z/ に有声化 [sɪnzaɪ]→シンザイ
アイファウンダケイヴアンダイヴビンイクスプローリングエバシンス
since thenその後、それ以来

⛏I had a base there, but I moved all my stuff since then.

昔あそこに拠点があったが、それ以来、すべての荷物を移動させた。

/aɪ hæd ə ˈbeɪs ðɛər bət aɪ ˈmuːvd ɔːl maɪ stʌf sɪns ðɛn/
  • had a：had /hæd/ 語末 /d/ が a /ə/ に連結 [ˈhædə]→ハダ
  • a base：a /ə/ が base /beɪs/ の /b/ に連結 [əˈbeɪs]→アベイス
  • since then：since /sɪns/ 語末の /s/ と then /ðɛn/ の /ð/ は子音+子音で連結なし
