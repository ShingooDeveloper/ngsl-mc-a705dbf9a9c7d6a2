correct

形容詞・動詞/kəˈrɛkt/CEFR B1頻度#1029
  1. 【形容詞】正しい、正確である（誤りのない、間違っていない）
  2. 【形容詞】適切な、ふさわしい（状況に合った、ふさわしい）
  3. 【動詞】(誤りを)訂正する、直す、修正する（～を正しくする、誤りを改める）
  4. 【動詞】(人を)正す、指摘する（誤りを指摘して直させる）

⛏️ マインクラフト例文

⛏You're using the correct tools for mining diamonds.

ダイヤモンド採掘に正しいツールを使っている。

[juːr ˈjuːzɪŋ ðə kəˈrɛkt tuːlz fɚ ˈmaɪnɪŋ ˈdaɪəməndz]
  • You're と using: 語末子音 /r/ が母音 /j/ に連結
ユアー ユーズィング ザ カレクト トゥールズ ファー マイニング ダイアモンズ

⛏My friend gave me the correct coordinates for the village.

友人が村の正確な座標を教えてくれた。

[maɪ ˈfrɛnd ɡeɪv mi ðə kəˈrɛkt koʊˈɔrdənɪts fɚ ðə ˈvɪlɪdʒ]
  • gave と me: 語末子音 /v/ が母音 /i/ に連結
  • for と the: 語末子音 /r/ が母音 /ə/ に連結
マイ フレンド ゲイヴ ミー ザ カレクト コオーディネイツ ファー ザ ヴィリッジ

⛏I had to correct the redstone circuit because it wasn't working properly.

レッドストーン回路が正しく機能していないので直さなければならなかった。

[aɪ hæd tə kəˈrɛkt ðə ˈrɛdˌstoʊn ˈsɜːrkɪt bɪˈkɔːz ɪt ˈwɑːzənt̚ ˈwɜːrkɪŋ ˈprɑːpərli]
  • to の弱形 /tə/
  • it wasn't: wasn't の /t/ は未開放で詰まる
アイ ハッド タ カレクト ザ レッドストーン サーキット ビコーズ イッ ウォズント ワーキング プローパーリー
