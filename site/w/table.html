table

名詞・動詞/ˈteɪbəl/CEFR A1頻度#449
  1. 【名詞】平らな天板と脚を持つ家具。食卓、机、テーブル（…wooden table、dining table など）
  2. 【名詞】ゲーム内の特定用途を持つ台や装置（…crafting table、enchanting table など）
  3. 【名詞】データを行と列に整理して示した表。一覧表（…table of recipes、loot table など）
  4. 【動詞】議論や提案を後回しにする、棚上げにする

⛏️ マインクラフト例文

⛏I placed a wooden table in my house.

私は家に木製のテーブルを置いた。
