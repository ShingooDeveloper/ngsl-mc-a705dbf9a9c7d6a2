export

動詞・名詞/ɪkˈspɔɚrt/CEFR B1頻度#1540
  1. 【動詞】輸出する、（商品を）国外に販売・送付する（export goods/resources）
  2. 【動詞】（データ・ファイルを）エクスポートする、外部形式に出力する
  3. 【名詞】輸出品、輸出される商品や資源
  4. 【名詞】輸出、商品などの国外への送付・販売
  5. 【動詞】（影響・技術などを）外部に広める、伝える

⛏️ マインクラフト例文

⛏I'll export this world to share it with my friend.

このワールドをエクスポートして友達とシェアしよう。

/aɪl ɪkˈspɔɚrt̚ ðɪs wɝːld tə ʃɛɚ ɪt wɪð maɪ frɛnd/
  • export の語末 /t/ が this の /ð/（子音）に続くため、/t/ は未開放 [ɪkˈspɔɚrt̚ ðɪs]
  • world の /d/ の後、to は弱形 /tə/ で続く
アイル イクスポート ディス ワールド トゥ シェア イット ウィズ マイ フレンド

⛏We need to export diamonds from this mine as soon as possible.

この鉱山からダイヤを輸出する必要があります。

/wi niːd tə ɪkˈspɔɚrt̚ ˈdaɪməndz frəm ðɪs maɪn æz suːn əz ˈpɑsəbəl/
  • need の /d/ の後、to は弱形 /tə/ で続く
  • export の語末 /t/ が diamonds の /d/（子音）に続くため、/t/ は未開放 [ɪkˈspɔɚrt̚ ˈdaɪmənds]
  • soon の /n/ の後、as は弱形 /əz/ で続く [suːn əz]
ウィー ニード タ イクスポート ダイマンズ フラム ディス マイン アズ スーン アズ パッサボル

⛏This export shipment contains rare materials from our base.

この輸出荷物は私たちの基地から採取した希少資源を含んでいます。

/ðɪs ˈɛkspɔɚrt̚ ˈʃɪpmənt̚ kənˈteɪnz rɛr məˈtɪrəlz frəm aʊɚ beɪs/
  • export（名詞）の語末 /t/ が shipment の /ʃ/（子音）に続くため、/t/ は未開放 [ˈɛkspɔɚrt̚ ˈʃɪpmənt]
  • shipment の語末 /t/ が contains の /k/（子音）に続くため、/t/ は未開放 [ˈʃɪpmənt̚ kənˈteɪnz]
  • from の /m/ の後、our の母音 /aʊ/ が続き自然に連結 [frəm aʊɚ]

💬 よく使う熟語（マイクラ例文）

export to～に輸出する、～にエクスポートする

⛏I want to export my world to the Java launcher.

Javaランチャーにワールドをエクスポートしたいです。

/aɪ ˈwɑnt tə ɪkˈspɔɚrt̚ maɪ wɝːld tə ðə ˈdʒɑːvə ˈlɔːntʃər/
  • want の /t/ は未開放、to は弱形 /tə/ で続く [ˈwɑnt̚ tə]
  • export の語末 /t/ が my の /m/（子音）に続くため、/t/ は未開放 [ɪkˈspɔɚrt̚ maɪ]
  • world の /d/ の後、to は弱形 /tə/ で続く
アイ ウォント タ イクスポート マイ ワールド タ ザ ジャーバ ローンチャー
export from～から輸出する、～から取り出す

⛏We export emeralds from the deep caverns beneath our mine.

私たちは鉱山下の深い洞窟からエメラルドを輸出しています。

/wi ɪkˈspɔɚrt̚ ˈɛmərəldz frəm ðə ˈdiːp ˈkævərnz bɪˈniːθ aʊɚ maɪn/
  • export の語末 /t/ が emeralds の /ɛ/（母音）に続き、通常に開放される
  • from は弱形 /frəm/ で、the は弱形 /ðə/ で続く
  • beneath の /θ/ の後、our の母音 /aʊ/ が続き自然に連結 [bɪˈniːθ aʊɚ]
