several

形容詞・代名詞/ˈsɛvɚəl/CEFR B1頻度#347
  1. 【形容詞・限定詞】複数の、いくつかの（several items で「いくつかの項目」）
  2. 【代名詞】いくつか、複数のもの（several of them で「それらのいくつか」）

⛏️ マインクラフト例文

⛏I found several diamonds in a cave.

洞窟でダイヤを何個か見つけたんだ。

/aɪ ˈfaʊnd ˈsɛvɚəl ˈdaɪməndz ɪn ə keɪv/
  • in a: 語末子音 /n/ が母音 /ə/ に連結 [ɪnə]
  • diamonds in: 語末子音 /z/ が母音 /ɪ/ に連結 [ˈdaɪməndz ɪn]
アイ ファウンド セヴァル ダイモンズ イナ ケイヴ

⛏We need several more oak logs to build the wall.

この壁を作るにはオークの丸太がもっと必要だ。

/wi ˈniːd ˈsɛvɚəl mɔr ˈoʊk lɔgz tə bɪld ðə wɔl/
  • logs to: logs の語末 /z/ が to の /ə/ に連結
  • build the: the は弱形 /ðə/ に弱化
ウィ ニード セヴァル モア オウク ログズ タ ビルド ザ ウォル

⛏Several creepers spawned near the base at night.

夜間、ベースの近くに複数のクリーパーが湧いた。

/ˈsɛvɚəl ˈkriːpərz spɔnd nɪr ðə ˈbeɪs æt naɪt/
  • near the: near の語末 /r/ が the の /ə/ に連結、the は弱形 /ðə/
  • base at: base の語末 /s/ が at の /æ/ に連結
セヴァル クリーパーズ スポーンド ニア ザ ベイス アッ ナイト
