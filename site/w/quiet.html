quiet

形容詞・動詞/ˈkwaɪ.ət/CEFR A2頻度#1310
  1. 【形容詞】（音が）静かな、物音がしない（…：a quiet room）
  2. 【形容詞】（人が）おしゃべりでない、話さずにいる
  3. 【形容詞】穏やかな、落ち着いた、平穏な（…：a quiet life）
  4. 【動詞】〜を静かにする、〜を落ち着かせる（…：to quiet the crowd）

⛏️ マインクラフト例文

⛏This is a quiet place to build my house.

ここは私の家を建てるのに静かな場所です。

[ðɪs ɪzə ˈkwaɪ.ət ˈpleɪs tə ˈbɪld maɪ ˈhaʊs]
  • is a → is の語末子音 /z/ が a の弱形 /ə/ に連結 [ɪzə] イザ
  • to → 機能語は弱形 /tə/ に弱化し自然に接続
ディス イザ クワイエッ プレイス タ ビルド マイ ハウス

⛏The cave went quiet when the creepers spawned.

クリーパーが出現したとき、洞窟は静寂に包まれました。

[ðə ˈkeɪv ˈwɛn̚t ˈkwaɪ.ət wɛn ðə ˈkri.pɚz ˈspɔnd]
  • went quiet → went の語末子音 /t/ が quiet の語頭 /k/ に続くため未開放 [t̚]
  • the creepers → the の弱形 /ðə/ に弱化 [ðə ˈkri.pɚz] ザ クリーパーズ
ザ ケイヴ ウェント クワイエッ ウェン ザ クリーパーズ スポーンド

⛏You need to stay quiet so the Enderman won't notice you.

エンダーマンに見つからないように、静かにしていなければなりません。

[ju ˈnid̚ tə ˈsteɪ ˈkwaɪ.ət soʊðə ˈɛndɚmæn ˈwoʊn̚t ˈnoʊtɪs ju]
  • need to → need の語末 /d/ が to の弱形 /tə/ に続くため未開放 [d̚]、[ˈnid̚ tə] ニッタ
  • so the → so の語末母音が the の子音 /ð/ に自然に接続 [soʊðə] ソウザ
  • won't notice → won't の /t/ は未開放で notice に続く
