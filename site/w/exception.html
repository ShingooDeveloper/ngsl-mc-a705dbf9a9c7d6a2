exception

名詞/ɪkˈsɛpʃən/CEFR B1頻度#1905
  1. 【名詞】規則や通常から外れた人・物・事象；例外（…an exception to the rule）
  2. 【名詞】除外される人や物；対象外のもの（…with the exception of）
  3. 【名詞】異議、反論；異議申し立て（…take exception to）

⛏️ マインクラフト例文

⛏I made an exception for you on this server.

このサーバーであなたのために例外を認めました。

/aɪ meɪdən ɪkˈsɛpʃən fɔɹ jə ɑn ðɪs ˈsɝvɚ/
  • made の語末子音 /d/ が a の弱形 /ə/ に連結: [meɪdən] メイダン
  • for の語末音 /ɹ/ が you の /j/ に連結、you は弱形化: [fɔɹjə] フォーユ
アイ メイダン イクセプション フォーユ オン ディス サーヴァー

⛏With one exception, all players found emeralds.

1人を除いて、全員がエメラルドを見つけました。

/wɪð wʌnɪkˈsɛpʃən ɔl ˈpleɪɚz faʊnd ɪˈmɛɹəldz/
  • one の語末子音 /n/ が exception の /ɪ/ に連結: [wʌnɪkˈsɛpʃən] ワンイクセプション
ウィズ ワンイクセプション オール プレイアーズ ファウンド イメラルズ

⛏That is an exception to the mining rules.

それは採掘ルールの例外です。

/ðæt ɪzən ɪkˈsɛpʃən tu ðə ˈmaɪnɪŋ ˈrulz/
  • is の語末子音 /z/ が an の弱形 /ə/ に連結: [ɪzən] イザン
  • to は機能語で弱形 /tə/ に弱化、the の /ð/ に連結: [tə ðə] タ ザ
ザット イザン イクセプション タ ザ マイニング ルールズ
