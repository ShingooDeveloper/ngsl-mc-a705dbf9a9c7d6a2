branch

名詞・動詞/brænʃ/CEFR B1頻度#1404
  1. 【名詞】木や灌木の枝（the woody part extending from a tree trunk or limb）
  2. 【名詞】河川や洞窟などの分岐・支流（a tributary or division extending from a main body）
  3. 【名詞】組織の支店・支部（a division of an organization in a different location）
  4. 【動詞】枝分かれする、分岐する（to divide into branches or extend in different directions）
  5. 【動詞】（…から）派生する、分かれ出る（to separate or extend from a main line or system）

⛏️ マインクラフト例文

⛏I made a crafting table from this oak branch.

このオークの枝からクラフト台を作りました。

/aɪ ˈmeɪd ə ˈkræftɪŋ ˈteɪbəl frəm ðɪs oʊk brænʃ/
  • 「made a」: d が a の ə に連結 [ˈmeɪdə] メイダ
  • 「a crafting」: a の ə が crafting の c に連結 [ə ˈkræftɪŋ]
アイ メイダ クラフティング テイブル フラム ディス オウク ブランチ

⛏The tunnel branches to the left toward a cave system.

トンネルは左に分岐して洞窟システムへ向かっています。

/ðə ˈtʌnəl ˈbrænʃɪz tu ðə lɛft təˈwɔrd ə ˈkeɪv ˈsɪstəm/
  • 「the tunnel」: the の ə が tunnel の t に連結 [ðə ˈtʌnəl]
  • 「to the」: to /tə/ と the /ðə/ は両方弱形で続く [tə ðə] タ ザ
  • 「a cave」: a の ə が cave の c に連結 [ə ˈkeɪv]
ザ タナル ブランチズ タ ザ レフト タワード ア ケイヴ システム

💬 よく使う熟語（マイクラ例文）

branch off…から分岐する、派生する

⛏The mineshaft branches off into smaller tunnels deeper underground.

鉱山は地下深くへ、より小さなトンネルに分岐しています。

/ðə ˈmaɪnʃæft ˈbrænʃɪz ɔf ˈɪntu ˈsmɔlər ˈtʌnəlz ˈdipər ˈʌndərgraʊnd/
  • 「branches off」: z が off の ɔ に連結
  • 「off into」: f が into の ɪ に連結
  • 「deeper underground」: r が underground の ʌ に連結
ザ マインシャフト ブランチズ オフ イントゥ スモーラー タナルズ ディーパー アンダーグラウンド

⛏We decided to branch out and explore the Nether dimension.

私たちはネザー次元を探検するために活動を広げることにしました。

/wi dɪˈsaɪdəd tu ˈbrænʃ aʊt ənd ɪkˈsplɔr ðə ˈnɛðər dɪˈmɛnʃən/
  • 「to branch」: to の弱形 /tə/ が branch に続く [tə ˈbrænʃ]
  • 「the Nether」: the の ə が Nether の n に連結 [ðə ˈnɛðər]
